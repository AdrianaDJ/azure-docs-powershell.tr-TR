# Get-AzureDeploymentEvent

## SYNOPSIS
Azure 'un başlattığı olaylar hakkında, bu etkiyi gerçekleştiren sanal makineleri ve bulut hizmetlerini getirir.

## INDEKI

### GetDeploymentEventBySlot (varsayılan)
```
Get-AzureDeploymentEvent [-ServiceName] <String> [-StartTime] <DateTime> [-EndTime] <DateTime>
 [[-Slot] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

### GetDeploymentEventByName
```
Get-AzureDeploymentEvent [-ServiceName] <String> [-StartTime] <DateTime> [-EndTime] <DateTime>
 [-DeploymentName] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Get-AzureDeploymentEvent** cmdlet 'ı, Azure 'un bu etkiyi gerçekleştiren olaylarla ilgili bilgileri alır.
Bu olaylar, planlanmış bakım olaylarını içerir.
Bu cmdlet, rolün etkinlik örneğini veya sanal makinesini tanımlayan olayların listesini, etkisinin nedenini ve olayın başlangıç saatini döndürür.

## ÖRNEKLERDEN

### 2
```
Get-AzureDeploymentEvent -DeploymentName "ConstosoDeployment" -ServiceName "ContosoService"
```
