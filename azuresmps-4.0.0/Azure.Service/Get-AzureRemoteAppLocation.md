# Get-AzureRemoteAppLocation

## SYNOPSIS
Azure RemoteApp 'in desteklediği konumları listeler.

## INDEKI

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Get-AzureRemoteAppLocation** cmdlet 'ı Azure RemoteApp 'in desteklediği konumları listeler.
