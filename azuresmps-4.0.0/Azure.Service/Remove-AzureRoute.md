# Remove-AzureRoute

## SYNOPSIS
Yol tablosundan bir yol kaldırır.

## INDEKI

```
Remove-AzureRoute -RouteName <String> [-Force] -RouteTable <IRouteTable> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Tanım
**Remove-AzureRoute** cmdlet 'i, yol tablosundan bir yol kaldırır.

## ÖRNEKLERDEN

### Örnek 1: rotayı kaldırma
```
PS C:\> Get-AzureRouteTable -Name "ApplianceRouteTable" | Remove-AzureRoute -RouteName "InternetRoute"
Confirm
Are you sure you want to remove the Route "InternetRoute" from Route Table "ApplianceRouteTable"?
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y

Routes                        Name                          Location                      Label
------                        ----                          --------                      -----
{approute}                    AppRT                         Central US                    Appliance Route Table
```

Bu komut, ApplianceRouteTable adlı bir yol tablosu alır.
Komut bu yol tablosunu geçerli cmdlet 'e geçirir.
Geçerli cmdlet, ınternetroute adlı bir yolu kaldırır.
