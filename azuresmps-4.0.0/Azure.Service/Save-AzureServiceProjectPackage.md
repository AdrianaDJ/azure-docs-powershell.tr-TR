# Save-AzureServiceProjectPackage

## SYNOPSIS
Hizmet projesini Azure bulut paketine paketler.

## INDEKI

```
Save-AzureServiceProjectPackage [-Local] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
Bu konuda, Microsoft Azure PowerShell modülünün 0.8.10 sürümündeki cmdlet açıklanmaktadır.
Kullandığınız modülün sürümünü edinmek için, Azure PowerShell konsolunda yazın `(Get-Module -Name Azure).Version` .

**Save-AzureServiceProjectPackage** cmdlet 'i, hizmet projesini bir Azure bulut paketine (*. cspkg) paketler.

## ÖRNEKLERDEN

### Örnek 1: hizmet projesi paketi oluşturma
```
PS C:\> Save-AzureServiceProjectPackage
```

Bu örnek MyAzureServiceProject adındaki bir hizmet projesi için *. cspgk oluşturur.
