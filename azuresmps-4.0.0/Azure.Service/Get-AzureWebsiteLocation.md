# Get-AzureWebsiteLocation

## SYNOPSIS
Kullanılabilir tüm Web sitesi konumlarını alır.

## INDEKI

```
Get-AzureWebsiteLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
Bu konuda, Microsoft Azure PowerShell modülünün 0.8.10 sürümündeki cmdlet açıklanmaktadır.
Kullandığınız modülün sürümünü edinmek için, Azure PowerShell konsolunda yazın `(Get-Module -Name Azure).Version` .

Geçerli aboneliğin kullanılabilir tüm Web sitesi konumlarını alır

## ÖRNEKLERDEN

### Örnek 1: tüm kullanılabilir konumları al
```
PS C:\> Get-AzureWebsiteLocations
```

Bu örnekte, geçerli aboneliğin kullanılabilir tüm Web sitesi konumları alınır.
