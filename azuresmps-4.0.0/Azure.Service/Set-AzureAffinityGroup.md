# Set-AzureAffinityGroup

## SYNOPSIS
Benzeşim grubunun özelliklerini değiştirir.

## INDEKI

```
Set-AzureAffinityGroup [-Name] <String> -Label <String> [-Description <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Set-AzureAffinityGroup** cmdlet 'ı bir Azure benzeşim grubunun özelliklerini değiştirir.
Etiketi ve açıklamayı değiştirebilirsiniz.

## ÖRNEKLERDEN

### Örnek 1: benzeşim grubunu değiştirme
```
PS C:\> Set-AzureAffinityGroup -Name "South01" -Label "SouthUSProduction" -Description "Production applications for Southern US locations"
```

Bu komut, South01 adlı benzeşim grubunun, Güneydoğu üretim olacak şekilde değiştirir komut ayrıca açıklamayı değiştirir.
