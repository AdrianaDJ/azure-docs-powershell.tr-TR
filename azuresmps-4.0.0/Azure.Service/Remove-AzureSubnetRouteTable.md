# Remove-AzureSubnetRouteTable

## SYNOPSIS
Alt ağdan yol tablosu ilişkisini kaldırır.

## INDEKI

```
Remove-AzureSubnetRouteTable -VirtualNetworkName <String> -SubnetName <String> [-Force] [-PassThru]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Remove-azuyeniden yönlendirme Netroutetable** cmdlet 'i, alt ağdan yol tablosu ilişkilendirmesini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: yol tablosuyla alt ağ arasındaki ilişkiyi kaldırma
```
PS C:\> Remove-AzureSubnetRouteTable -VirtualNetworkName "VNetUSCentral" -SubnetName "ContosoSubnet"
```

Bu komut, PublicRouteTable adlı yönlendirme tablosunun, ContosoSubnet adlı alt ağla ilişkisini kaldırır.
