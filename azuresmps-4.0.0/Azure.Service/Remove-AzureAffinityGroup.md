# Remove-AzureAffinityGroup

## SYNOPSIS
Bir abonelikteki benzeşim grubunu siler.

## INDEKI

```
Remove-AzureAffinityGroup [-Name] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Remove-AzureAffinityGroup** cmdlet 'i geçerli abonelikteki bir Azure benzeşim grubunu siler.
Üyesi olan bir benzeşim grubunu silemezsiniz.
Önce bir benzeşim grubunun tüm üyelerini silmelisiniz.

## ÖRNEKLERDEN

### Örnek 1: benzeşim grubunu kaldırma
```
PS C:\> Remove-AzureAffinityGroup -Name "South01"
```

Bu komut geçerli abonelikteki South01 benzeşim grubunu siler.
