# Get-AzureVMChefExtension

## SYNOPSIS
Sanal makineye uygulanan Chef uzantısını alır.

## INDEKI

```
Get-AzureVMChefExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Get-AzureVMChefExtension** cmdlet 'i sanal makineye uygulanan Chef uzantısını alır.

## ÖRNEKLERDEN

### Örnek 1: belirtilen sanal makineye, Chef uzantısının uygulanmasını sağlayın
```
PS C:\> Get-AzureVMChefExtension -VM $VM;
```

Bu komut belirtilen sanal makineye uygulanan Chef uzantısını alır.
