# Set-AzureVMPuppetExtension

## SYNOPSIS
Sanal makinenin Pupalı uzantısını ayarlar.

## INDEKI

```
Set-AzureVMPuppetExtension [-PuppetMasterServer] <String> [[-Version] <String>] [-Disable]
 [[-ReferenceName] <String>] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Set-AzureVMPuppetExtension** cmdlet 'i sanal makinenin Pupevcil hayvan uzantısını ayarlar.

## ÖRNEKLERDEN

### Örnek 1: sanal makine için Pupevcil hayvan uzantısını ayarlama
```
PS C:\> Set-AzureVMPuppetExtension -VM $VM
```

Bu örnekte, belirtilen sanal makinenin Pupevcil hayvan uzantısı $VM değişken olarak ayarlanır.
