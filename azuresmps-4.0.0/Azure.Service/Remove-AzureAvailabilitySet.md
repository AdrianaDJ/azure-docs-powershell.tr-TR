# Remove-AzureAvailabilitySet

## SYNOPSIS
Bir Azure sanal makinesinden kullanılabilirlik kümesini kaldırır.

## INDEKI

```
Remove-AzureAvailabilitySet -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Remove-AzureAvailabilitySet** cmdlet 'i, bir Azure sanal makinesinden kullanılabilirlik kümesini kaldırır.
