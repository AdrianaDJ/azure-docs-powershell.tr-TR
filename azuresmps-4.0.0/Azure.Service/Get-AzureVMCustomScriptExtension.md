# Get-AzureVMCustomScriptExtension

## SYNOPSIS
Azure sanal makinesi özel komut dosyası uzantısından bilgi alır.

## INDEKI

```
Get-AzureVMCustomScriptExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Get-AzureVMCustomScriptExtension** cmdlet 'ı bir Azure sanal makinesi özel komut dosyası uzantısından bilgi alır.

## ÖRNEKLERDEN

### Örnek 1: Azure sanal makine Betik uzantısı alma
```
PS C:\> Get-AzureVMCustomScriptExtension -VM $VM;
```

Bu komut, $VM değişkeninde depolanan bir Azure sanal makine komut dosyası uzantısını alır.
