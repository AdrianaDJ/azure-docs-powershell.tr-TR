# Set-AzureStoreAddOn

## SYNOPSIS
Var olan bir eklenti örneğini güncelleştirir.

## INDEKI

```
Set-AzureStoreAddOn -Name <String> -Plan <String> [-PromotionCode <String>] [-PassThru]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
Bu konuda, Microsoft Azure PowerShell modülünün 0.8.10 sürümündeki cmdlet açıklanmaktadır.
Kullandığınız modülün sürümünü edinmek için, Azure PowerShell konsolunda yazın `(Get-Module -Name Azure).Version` .

Bu cmdlet geçerli abonelikten var olan bir eklenti örneğini güncelleştirir.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Set-AzureStoreAddOn MyAddOn NewPlanId
```

Bu örnekte, yeni bir plan KIMLIĞIYLE birlikte eklenti güncelleştirilir.

### Örnek 2
```
PS C:\> Set-AzureStoreAddOn MyAddOn NewPlanId MyPromoCode
```

Bu örnek, yeni bir plan kodu ve Promosyon koduyla bir eklentiyi güncelleştirir.
