# Set-AzureInternalLoadBalancer

## SYNOPSIS
Bir Azure hizmetinde iç yük dengeleyici yapılandırmasını değiştirir.

## INDEKI

### ServiceAndSlot (varsayılan)
```
Set-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### Yalnızca subnetName
```
Set-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-SubnetName] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

### Subnetnameandıp
```
Set-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-SubnetName] <String> [-StaticVNetIPAddress] <IPAddress> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Set-AzureInternalLoadBalancer** cmdlet 'ı bir Azure hizmetinde iç yük dengeleyici yapılandırmasını değiştirir.
Sanal ağ için, iç yük dengeleyicinin bir alt ağını veya IP adresini belirtebilirsiniz.
