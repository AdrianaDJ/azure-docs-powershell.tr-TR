# Add-AzureDisk

## SYNOPSIS
Azure disk deposuna bir disk ekler.

## INDEKI

```
Add-AzureDisk [-DiskName] <String> [-MediaLocation] <String> [-Label <String>] [-OS <String>]
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## Tanım
**Add-AzureDisk** cmdlet 'i geçerli abonelikteki Azure disk deposuna bir disk ekler.
Bu cmdlet bir sistem diski veya veri diski ekleyebilir.
Sistem diski eklemek için *OS* parametresini kullanarak bir işletim sistemi türü belirtin.

## ÖRNEKLERDEN

### Örnek 1: Windows işletim sistemini kullanan bir başlangıç disketi ekleme
```
PS C:\> Add-AzureDisk -DiskName "MyWinDisk" -MediaLocation "http://contosostorage.blob.core.azure.com/vhds/winserver-system.vhd" -Label "StartupDisk" -OS "Windows"
```

Bu komut, disk deponuza bir sistem diski ekler.
Sistem diski Windows işletim sistemini kullanır.

### Örnek 2: veri disketi ekleme
```
PS C:\> Add-AzureDisk -DiskName "MyDataDisk" -MediaLocation "http://yourstorageaccount.blob.core.azure.com/vhds/winserver-data.vhd" -Label "DataDisk"
```

Bu komut, veri disketi ekler.

### Örnek 3: Linux sistem diski ekleme
```
PS C:\> Add-AzureDisk -DiskName "MyLinuxDisk" -MediaLocation "http://yourstorageaccount.blob.core.azure.com/vhds/linuxsys.vhd" -OS "Linux"
```

Bu komut, bir Linux sistem diski ekler.
