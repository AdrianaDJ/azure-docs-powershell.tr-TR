# Start-AzureWebsiteJob

## SYNOPSIS
Web sitesi için Web işi başlatır.

## INDEKI

```
Start-AzureWebsiteJob -JobName <String> -JobType <WebJobType> [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Start-AzureWebsiteJob** cmdlet 'i Web sitesi için Web işi başlatır.

## ÖRNEKLERDEN

### Örnek 1: Web sitesi için Web işi başlatma
```
PS C:\> Start-AzureWebsiteJob -Name MyWebsite -JobName MyWebJob -JobType Continuous
```

MyWebSite için MyWebJob adlı Web işini başlatır.
