# Switch-AzureWebsiteSlot

## SYNOPSIS
Başka bir yuvalı Web sitesinin üretim yuvasını değiştirir.

## INDEKI

```
Switch-AzureWebsiteSlot [-Name <String>] [-Slot1 <String>] [-Slot2 <String>] [-Force]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Switch-AzureWebsiteSlot** cmdlet 'i, başka bir yuvaya sahip bir Web sitesinin üretim yuvasını değiştirir.
Bu, yalnızca iki yuvalarla birlikte çalışır.

## ÖRNEKLERDEN

### Örnek 1: Web sitesi yuvasını değiştirme
```
PS C:\> Switch-AzureWebsiteSlot -Name MyWebsite
```

Azure Web sitesi MyWebsite yedekleme yuvasına üretim yuvası ile değiştirin.
