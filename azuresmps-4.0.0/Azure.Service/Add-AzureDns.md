# Add-AzureDns

## SYNOPSIS
Azure hizmetine DNS sunucusu ekler.

## INDEKI

```
Add-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Add-AzureDns** cmdlet 'ı bir Azure hizmetine DNS sunucusu ekler.

## ÖRNEKLERDEN

### Örnek 1: DNS sunucusu ekleme
```
PS C:\> Add-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.4 -Name "Dns07"
```

Bu komut, 10.1.2.4 adresine sahip olan DNS sunucusunu ContosoService 'e ekler.
