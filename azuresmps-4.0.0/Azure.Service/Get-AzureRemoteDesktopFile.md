# Get-AzureRemoteDesktopFile

## SYNOPSIS
Bir Azure sanal makinesi için RDP dosyası alır.

## INDEKI

### İndir (varsayılan)
```
Get-AzureRemoteDesktopFile [-Name] <String> [-LocalPath] <String> [-ServiceName] <String>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### Başlatılması
```
Get-AzureRemoteDesktopFile [-Name] <String> [[-LocalPath] <String>] [-Launch] [-ServiceName] <String>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## Tanım
**Get-AzureRemoteDesktopFile** cmdlet 'i, bir Azure sanal makinesi için bir uzak masaüstü BAĞLANTıSı (RDP) dosyası indirip kaydeder.
Cmdlet, belirtilen sanal makineye bir Uzak Masaüstü bağlantısı başlatabilir.

## ÖRNEKLERDEN

### Örnek 1: RDP dosyası alma
```
PS C:\> Get-AzureRemoteDesktopFile -ServiceName "ContosoService" -Name "VirtualMachine07" -LocalPath "C:\temp\VirtualMachine07.rdp"
```

Bu komut, ContosoService adlı hizmette çalışan VirtualMachine07 adındaki VirtualMachine07 sanal makinesi için bir RDP dosyası alır.
Komut bu dosyayı C:\temp\virtualmachine07.rdpadıyla depolar.

### Örnek 2: uzak oturum başlatma
```
PS C:\> Get-AzureRemoteDesktopFile -ServiceName "ContosoService" -Name "VirtualMachine07" -Launch
```

Bu komut, ContosoService adlı hizmette çalışan VirtualMachine07 adındaki VirtualMachine07 sanal makinesi için bir RDP dosyası alır.
Komut bir Uzak Masaüstü oturumu başlatır.
Komut, bağlantı kapalıyken RDP dosyasını siler.
