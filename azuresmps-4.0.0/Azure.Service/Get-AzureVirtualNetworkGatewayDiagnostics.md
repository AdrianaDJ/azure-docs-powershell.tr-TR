# Get-AzureVirtualNetworkGatewayDiagnostics

## SYNOPSIS
Azure sanal ağ geçidi Tanılamaların sonuçlarını alır.

## INDEKI

```
Get-AzureVirtualNetworkGatewayDiagnostics -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Get-AzureVirtualNetworkGatewayDiagnostics** cmdlet 'i Azure sanal ağ geçidi Tanılamaların sonuçlarını alır.
