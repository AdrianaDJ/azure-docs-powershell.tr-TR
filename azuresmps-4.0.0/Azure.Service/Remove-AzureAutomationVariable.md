# Remove-AzureAutomationVariable

## SYNOPSIS

Otomasyon değişkenini kaldırır.

## INDEKI

```
Remove-AzureAutomationVariable -Name <String> [-Force] -AutomationAccountName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

**Remove-AzureAutomationVariable** cmdlet 'ı, Microsoft Azure Otomasyonu 'ndan bir değişkeni kaldırır.

## ÖRNEKLERDEN

### Örnek 1: bir değişkeni kaldırma
```
PS C:\> Remove-AzureAutomationVariable -AutomationAccountName "Contoso17" -Name "MyStringVariable" -Force
```

Bu komut, kullanıcı doğrulaması istemeden Contoso17 adlı Otomasyon hesabındaki MyStringVariable adlı bir değişkeni kaldırır.
