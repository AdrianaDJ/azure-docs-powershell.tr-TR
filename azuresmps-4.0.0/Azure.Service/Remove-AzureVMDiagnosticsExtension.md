# Remove-AzureVMDiagnosticsExtension

## SYNOPSIS
Sanal makineden Azure tanılama uzantısını kaldırır.

## INDEKI

```
Remove-AzureVMDiagnosticsExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Remove-AzureVMDiagnosticsExtension** cmdlet 'i, bir sanal makineden Microsoft Azure tanılama uzantısını kaldırır.
Bu cmdlet 'in çıkışını **Update-AzureVM** cmdlet 'ine geçirmelisiniz.

## ÖRNEKLERDEN

### Örnek 1: sanal makineden Azure tanılama uzantısını kaldırma
```
PS C:\> Remove-AzureVMDiagnosticsExtension -VM $VM | Update-AzureVM
```

Bu komut, bir sanal makineden Azure tanılama uzantısını kaldırır.
