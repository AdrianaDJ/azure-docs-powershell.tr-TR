# Set-AzureSubnet

## SYNOPSIS
Bir Azure sanal makinesinin alt ağ listesini tanımlar.

## INDEKI

```
Set-AzureSubnet [-SubnetNames] <String[]> -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Set-azuyeniden PNet** cmdlet 'i sanal makine yapılandırması için alt ağ listesini ayarlar.
Bir sanal makinenin alt ağlarını ayarlamak için **New-AzureVM** ile kullanın.

## ÖRNEKLERDEN

### Örnek 1: sanal makine yapılandırmasına alt ağ ekleme
```
PS C:\> New-AzureVMConfig -Name "VirtualMachine04" -ImageName $image -InstanceSize "Small" | Add-AzureProvisioningConfig -Windows -Password "password" | Set-AzureSubnet "PubSubnet","PrivSubnet" | New-AzureVM -ServiceName "ContosoService03"
```

Bu komut sanal makine yapılandırmasına bir alt ağ ekler ve VirtualMachine04 adlı sanal makineyi oluşturur.
