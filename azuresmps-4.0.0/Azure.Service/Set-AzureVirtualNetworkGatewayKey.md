# Set-AzureVirtualNetworkGatewayKey

## SYNOPSIS
Azure sanal ağ geçidi için anahtarı ayarlar.

## INDEKI

```
Set-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> -SharedKey <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Set-AzureVirtualNetworkGatewayKey** cmdlet 'ı bir Azure sanal ağ geçidi için anahtarı ayarlar.
