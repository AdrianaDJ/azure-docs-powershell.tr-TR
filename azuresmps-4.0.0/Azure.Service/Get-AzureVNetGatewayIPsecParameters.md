# Get-AzureVNetGatewayIPsecParameters

## SYNOPSIS
Sanal ağ geçidi ile yerel ağ sitesi arasındaki bağlantının IPSec parametrelerini alır.

## INDEKI

```
Get-AzureVNetGatewayIPsecParameters -VNetName <String> -LocalNetworkSiteName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Get-AzureVNetGatewayIPsecParameters** cmdlet 'i sanal ağ geçidi ile yerel bir ağ sitesi arasındaki bağlantının geçerli Internet Protokolü güvenliğini (IPSec) ve Internet Anahtar DEĞIŞIMI (IKE) parametrelerini alır.
