# Remove-AzureRemoteAppVNet

## SYNOPSIS
Azure RemoteApp sanal ağını siler.

## INDEKI

```
Remove-AzureRemoteAppVNet -VNetName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Remove-AzureRemoteAppVNet** cmdlet 'ı bir Azure RemoteApp sanal ağını siler.

## ÖRNEKLERDEN

### Örnek 1: belirtilen sanal ağı silme
```
PS C:\> Remove-AzureRemoteAppVnet -VNetName "ContosoVNet"
```

Bu komut, ContosoVNet adlı sanal ağı siler.
