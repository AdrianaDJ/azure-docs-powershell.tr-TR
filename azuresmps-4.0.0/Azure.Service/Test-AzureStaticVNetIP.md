# Test-AzureStaticVNetIP

## SYNOPSIS
Statik sanal ağ IP adresinin kullanılabilirliğini sınar ve sorgulanan adres kullanılamıyorsa öneri listesini alır.

## INDEKI

```
Test-AzureStaticVNetIP [-VNetName] <String> [-IPAddress] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Test-AzureStaticVNetIP** cmdlet 'i statik sanal ağ IP adresinin kullanılabilirliğini sınar ve sorgulanan adres kullanılamıyorsa öneri listesini alır.
