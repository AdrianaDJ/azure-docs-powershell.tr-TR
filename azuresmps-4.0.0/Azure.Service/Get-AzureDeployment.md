# Get-AzureDeployment

## SYNOPSIS
Dağıtımın ayrıntılarını alır.

## INDEKI

```
Get-AzureDeployment [-ServiceName] <String> [[-Slot] <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Get-AzureDeployment** cmdlet 'ı bir Azure dağıtımının ayrıntılarını alır.
Azure hizmetinin adını ve dağıtımın yuvasını belirtin.

## ÖRNEKLERDEN

### Örnek 1: bir üretim dağıtımının ayrıntılarını alma
```
PS C:\> Get-AzureDeployment -ServiceName "ContosoService"
```

Bu komut, ContosoService adlı hizmetin dağıtımının ayrıntılarını döndürür.
Bu komut bir yuva belirtmez.
Bu nedenle, komut üretim varsayılan değerini kullanır.

### Örnek 2: düzeylendirme dağıtımının ayrıntılarını alma
```
PS C:\> Get-AzureDeployment -ServiceName "ContosoService" -Slot "Staging"
```

Bu komut, Contosohizmetinin hazırlama dağıtımının ayrıntılarını döndürür.
