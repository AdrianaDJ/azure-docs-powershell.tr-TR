# Remove-WAPackCloudService

## SYNOPSIS
Bulut hizmeti nesnelerini kaldırır.

## INDEKI

```
Remove-WAPackCloudService -CloudService <CloudService> [-PassThru] [-Force] [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Bu konular kullanım dışıdır ve gelecekte kaldırılacaktır.
Bu konuda, Microsoft Azure PowerShell modülünün 0.8.1 sürümündeki cmdlet açıklanmaktadır.
Kullandığınız modülün sürümünü bulmak için, Azure PowerShell konsolundan yazın `(Get-Module -Name Azure).Version` .

**Remove-WAPackCloudService** cmdlet 'i bulut hizmeti nesnelerini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: bulut hizmetini kaldırma
```
PS C:\> $CloudService = Get-WAPackCloudService -Name "ContosoCloudService01"
PS C:\> Remove-WAPackVM -VM $CloudService
```

İlk komut ContosoCloudService01 adındaki bulut hizmetini **Get-WAPackCloudService** cmdlet 'ini kullanarak alır ve bu nesneyi $CloudService değişkeninde depolar.

İkinci komut $CloudService depolanan CloudService öğesini kaldırır.
Komut sizden onaylamanızı ister.

### Örnek 2: onaysız bir bulut hizmetini kaldırma
```
PS C:\> $CloudService = Get-WAPackCloudService -Name "ContosoCloudService02"
PS C:\> Remove-WAPackCloudService -VM $CloudService -Force
```

İlk komut ContosoCloudService02 adındaki bulut hizmetini **Get-WAPackCloudService** cmdlet 'ini kullanarak alır ve bu nesneyi $CloudService değişkeninde depolar.

İkinci komut, $CloudService depolanan bulut hizmetini kaldırır.
Bu komut *Force* parametresini içerir.
Komut sizden onay istemez.
