# Get-AzureNetworkSecurityGroupConfig

## SYNOPSIS
Ağ güvenlik grubunun ayrıntılarını alır.

## INDEKI

```
Get-AzureNetworkSecurityGroupConfig [-Detailed] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Tanım
**Get-AzureNetworkSecurityGroupConfig** cmdlet 'i, bir ağ güvenlik grubunun ayrıntılarını döndürür.
Ağ güvenliği kurallarını görüntülemek için *ayrıntılı* parametreyi belirtin.
