# Get-AzureRemoteAppPlan

## SYNOPSIS
Azure RemoteApp planlarını listeler.

## INDEKI

```
Get-AzureRemoteAppPlan [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Get-AzureRemoteAppPlan** cmdlet 'ı Azure RemoteApp planlarını listeler.
