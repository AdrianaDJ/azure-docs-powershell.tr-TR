# Remove-AzureVirtualNetworkGatewayConnection

## SYNOPSIS
Sanal ağ geçidi bağlantısını kaldırır.

## INDEKI

```
Remove-AzureVirtualNetworkGatewayConnection -GatewayId <String> -ConnectedEntityId <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Remove-AzureVirtualNetworkGatewayConnection** cmdlet 'i sanal ağ geçidi bağlantısını kaldırır.
