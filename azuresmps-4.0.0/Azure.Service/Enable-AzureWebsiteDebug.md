# Enable-AzureWebsiteDebug

## SYNOPSIS
Web sitesinin hata ayıklamasını verir.

## INDEKI

```
Enable-AzureWebsiteDebug [-PassThru] -Version <String> [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
Visual Studio 'da Web sitesinin hata ayıklama özelliğini etkinleştirmiştir.

## ÖRNEKLERDEN

### Visual Studio 2013 hata ayıklamasını etkinleştirme
```
PS C:\> Enable-AzureWebsiteDebug -Name MyWebsite -Version VS2013
```

VS 2013 ' de hata ayıklamayı sağlar.
