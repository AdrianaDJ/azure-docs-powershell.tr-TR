# Remove-AzureStorageAccount

## SYNOPSIS
Belirtilen depolama hesabını abonelikten siler.

## INDEKI

```
Remove-AzureStorageAccount [-StorageAccountName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Remove-AzureStorageAccount** cmdlet 'ı bir Azure aboneliğindeki hesabı kaldırır.

## ÖRNEKLERDEN

### Örnek 1: depolama hesabını kaldırma
```
PS C:\> Remove-AzureStorageAccount -StorageAccountName "ContosoStore01"
```

Bu komut, ContosoStore01 depolama hesabını belirtilen abonelikten kaldırır.
