# Get-AzureCertificate

## SYNOPSIS
Bir Azure hizmetinden sertifika nesnesi alır.

## INDEKI

```
Get-AzureCertificate [-ServiceName] <String> [-ThumbprintAlgorithm <String>] [-Thumbprint <String>]
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## Tanım
**Get-Azurecercertificate** . cmdlet 'ı bir Azure hizmetinden sertifika nesnesi alır.

## ÖRNEKLERDEN

### Örnek 1: hizmetten sertifikaları alma
```
PS C:\> $AzureCert = Get-AzureCertificate -ServiceName "ContosoService"
```

Bu komut, ContosoService adlı hizmetten sertifika nesnelerini alır ve $AzureCert değişkeninde depolar.

### Örnek 2: hizmetten sertifika alma
```
PS C:\> $AzureCert = Get-AzureCertificate -ServiceName "ContosoService" -Thumbprint '5383CE0343CB6563281CA97C1D4D712209CFFA97'
```

Bu komut, ContosoService adlı hizmetten belirtilen parmak iziyle tanımlanan sertifika nesnesini alır ve $AzureCert değişkeninde depolar.
