# Update-AzureWebsiteRepository

## SYNOPSIS
Tüm yuvalar için yerel bir git deposunun uzaktaki depolarını güncelleştirir.

## INDEKI

```
Update-AzureWebsiteRepository [-Name <String>] -PublishingUsername <String> [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Update-AzureWebsiteRepository** cmdlet 'i, tüm yuvalar için yerel bir git deposunun uzaktaki depolarını güncelleştirir.

## ÖRNEKLERDEN

### Örnek 1: Web sitesi uzak depolarını güncelleştirme
```
PS C:\> Update-AzureWebsiteRepository -Name MyWebsite
```

Tüm Web sitesi için tüm yuvalar Web sitesi
