# New-AzureWebsiteJob

## SYNOPSIS
Web sitesi için Web işi oluşturur.

## INDEKI

```
New-AzureWebsiteJob -JobName <String> -JobType <WebJobType> -JobFile <String> [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**New-AzureWebsiteJob** cmdlet 'i Web sitesi için Web işi oluşturur.

## ÖRNEKLERDEN

### Örnek 1: Web sitesi için yeni Web işi oluşturma
```
PS C:\> New-AzureWebsiteJob -Name MyWebsite -JobName MyWebJob -JobType Continuous -JobFile job.bat
```

MyWebsite Web sitesinde job.bat aramak için sürekli bir iş oluşturur.
