# Update-AzureSiteRecoveryProtectionDirection

## SYNOPSIS
Site kurtarma nesnesinin korunması için kaynak ve hedef sunucuyu güncelleştirir.

## INDEKI

### ByRPObject (varsayılan)
```
Update-AzureSiteRecoveryProtectionDirection -RecoveryPlan <ASRRecoveryPlan> -Direction <String>
 [-WaitForCompletion] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### Byrpıd
```
Update-AzureSiteRecoveryProtectionDirection -RPId <String> -Direction <String> [-WaitForCompletion]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### Kod
```
Update-AzureSiteRecoveryProtectionDirection -ProtectionEntityId <String> -ProtectionContainerId <String>
 -Direction <String> [-WaitForCompletion] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### Btypeınfo
```
Update-AzureSiteRecoveryProtectionDirection -ProtectionEntity <ASRProtectionEntity> -Direction <String>
 [-WaitForCompletion] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Update-AzureSiteRecoveryProtectionDirection** cmdlet 'i, bir Azure Site Recovery nesnesinin yürütme işlemi tamamlandıktan sonra korunması için kaynak ve hedef sunucuyu güncelleştirir.

## ÖRNEKLERDEN

### Örnek 1: kapsayıcıda korumalı nesnenin yönünü değiştirme
```
PS C:\> $Container = Get-AzureSiteRecoveryProtectionContainer 
PS C:\> $Protected = Get-AzureSiteRecoveryProtectionEntity -ProtectionContainer $Container  
PS C:\> Update-AzureSiteRecoveryProtectionDirection -Direction RecoveryToPrimary -ProtectionEntity $Protected 
ID               : c38eecdc-731c-405b-a61c-08db99aae2fe

State            : NotStarted
StateDescription : NotStarted
StartTime        : 
EndTime          : 
AllowedActions   : {}
Name             : 
Tasks            : {}
Errors           : {}
```

İlk komut, **Get-AzureSiteRecoveryProtectionContainer** cmdlet 'ini kullanarak geçerli Azure Site Recovery kasasındaki korumalı kapsayıcıları alır ve ardından $Container değişkeninde depolar.

İkinci komut, **Get-AzureSiteRecoveryProtectionEntity** cmdlet 'ini kullanarak $Container depolanan kapsayıcıya ait sanal makineleri alır.
Komut sonuçları $Protected değişkeninde depolar.

Son komutu, $Protected depolanan nesneler için kurtarılan yönü ayarlar.
