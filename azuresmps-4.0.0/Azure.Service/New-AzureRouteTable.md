# New-AzureRouteTable

## SYNOPSIS
Yol tablosu oluşturur.

## INDEKI

```
New-AzureRouteTable -Name <String> -Location <String> [-Label <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Tanım
**New-AzureRouteTable** cmdlet 'i belirtilen konumda bir yol tablosu oluşturur.
Yönlendirme tablosuna Kullanıcı tanımlı yollar ekleyebilirsiniz.
Yol tablosunu sanal ağdaki bir alt ağla ilişkilendirebilirsiniz.
