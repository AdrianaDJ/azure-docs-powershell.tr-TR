# Add-AzureVirtualIP

## SYNOPSIS
Bulut hizmetine sanal IP ekler.

## INDEKI

```
Add-AzureVirtualIP [-ServiceName] <String> [-VirtualIPName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Add-AzureVirtualIP** cmdlet 'i Azure hizmetinize yeni BIR sanal IP (VIP) ekler.
Yeni sanal IP bir ada sahip ancak bir IP adresi ayrılmamış.

IP adresi yalnızca bir uç noktayı VIP ile ilişkilendirirseniz ayrılır.
Daha fazla bilgi için Add-AzureEndpoint bakın.

Aboneliğiniz ekstra VIP 'ler için ücretlendirilecek ve bu da son noktayla ilişkilendirilir.

## ÖRNEKLERDEN

### Örnek 1: hizmete sanal IP ekleme
```
PS C:\> Add-AzureVirtualIP -VirtualIPName "Vip01" -ServiceName "ContosoService03"
OperationDescription OperationId                          OperationStatus
-------------------- -----------                          ---------------
Add-AzureVirtualIP   4bd7b638-d2e7-216f-ba38-5221233d70ce Succeeded
```

Bu komut, bir hizmete sanal IP adresi ekler.
