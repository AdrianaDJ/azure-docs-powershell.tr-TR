# Remove-AzureLocalNetworkGateway

## SYNOPSIS
Azure yerel ağ ağ geçidini kaldırır.

## INDEKI

```
Remove-AzureLocalNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Tanım
**Remove-AzureLocalNetworkGateway** cmdlet 'ı bir Azure yerel ağ ağ geçidini kaldırır.
