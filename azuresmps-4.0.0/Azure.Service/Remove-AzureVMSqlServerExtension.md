# Remove-AzureVMSqlServerExtension

## SYNOPSIS
Bir sanal makine nesnesinden Azure sanal makine SQL Server uzantısını kaldırır.

## INDEKI

```
Remove-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Remove-AzureVMSqlServerExtension** cmdlet 'i, bir Azure sanal makine SQL Server uzantısını bir sanal makine nesnesinden kaldırır.
