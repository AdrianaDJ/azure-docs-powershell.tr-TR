# Set-AzureAvailabilitySet

## SYNOPSIS
Azure sanal makinesindeki kullanılabilirlik kümesinin adını ayarlar.

## INDEKI

```
Set-AzureAvailabilitySet [-AvailabilitySetName] <String> -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Tanım
**Set-AzureAvailabilitySet** cmdlet 'i, dağıtımdan sonra bir Azure sanal makinesindeki kullanılabilirlik kümesinin adını ayarlar.

## ÖRNEKLERDEN

### Örnek 1: kullanılabilirlik kümesinin adını değiştirme
```
PS C:\> Get-AzureVM -ServiceName "ContosoService" -Name "VirtualMachine24" | Set-AzureAvailabilitySet -AvailabilitySetName "AvailabilitySet14" | Update-AzureVM
```

İlk komut, VirtualMachine07 adındaki sanal makineyi, **Get-AzureVM** cmdlet 'Ini kullanarak contososervice adındaki hizmette alır.
Komut, ardışık düzen işlecini kullanarak nesneyi geçerli cmdlet 'e geçirir.
Bu cmdlet, bu sanal makinenin kullanılabilirlik kümesinin adını değiştirir.
Komut sanal makineyi güncelleştirir.
