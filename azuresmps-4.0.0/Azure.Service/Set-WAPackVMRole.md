# Set-WAPackVMRole

## SYNOPSIS
Sanal makine rolünün örnek sayımı özelliğini değiştirir.

## INDEKI

```
Set-WAPackVMRole -VMRole <VMRole> -InstanceCount <Int32> [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Tanım
Bu konular kullanım dışıdır ve gelecekte kaldırılacaktır.
Bu konuda, Microsoft Azure PowerShell modülünün 0.8.1 sürümündeki cmdlet açıklanmaktadır.
Kullandığınız modülün sürümünü bulmak için, Azure PowerShell konsolundan yazın `(Get-Module -Name Azure).Version` .

**Set-WAPackVMRole** cmdlet 'i, sanal makine rolünün örnek sayımı özelliğini değiştirir.

## ÖRNEKLERDEN

### Örnek 1: sanal makine rolü için örnek sayımını belirtme
```
PS C:\> $VMRole = Get-WAPackVMRole -Name "ContosoVMRole01"
PS C:\> Set-WAPackVMRole -VMRole $VMRole -InstanceCount 3
```

İlk komut ContosoVMRole01 adındaki sanal makine rolünü **Get-WAPackVMRole** cmdlet 'ini kullanarak alır ve bu nesneyi $VMRole değişkeninde depolar.

İkinci ve son komutu, $VMRole depolanan sanal makine rolünün yeni örnek sayısını 3 olarak ayarlar.
