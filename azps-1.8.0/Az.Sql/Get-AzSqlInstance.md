# Get-AzSqlInstance

## SYNOPSIS
Azure SQL yönetilen veritabanı örneği hakkında bilgi döndürür.

## INDEKI

```
Get-AzSqlInstance [[-Name] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Get-Azsqlınstance** cmdlet 'i, bir veya daha fazla Azure SQL yönetilen örneği hakkında bilgi döndürür.
Yalnızca bu örneğin bilgilerini görmek için bir örneğin adını belirtin.
