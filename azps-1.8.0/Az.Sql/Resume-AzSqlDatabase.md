# Resume-AzSqlDatabase

## SYNOPSIS
SQL veri ambarı veritabanını sürdürür.

## INDEKI

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Özgeçmiş-AzSqlDatabase** cmdlet 'ı Azure SQL veri ambarı veritabanını sürdürür.

## ÖRNEKLERDEN

### Örnek 1: Azure SQL veri ambarı veritabanını sürdürür
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Bu komut, askıya alınmış Azure SQL veri ambarı veritabanını sürdürür.
