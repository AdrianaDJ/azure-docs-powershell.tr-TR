# Remove-AzSqlServerThreatDetectionPolicy

## SYNOPSIS
Tehdit algılama ilkesini sunucudan kaldırır.

## INDEKI

```
Remove-AzSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Remove-AzSqlServerThreatDetectionPolicy cmdlet 'i Azure SQL Server 'dan tehdit algılama ilkesini kaldırır.
Bu cmdlet 'i kullanmak için, bu cmdlet 'in ilkeyi kaldırdığı sunucuyu belirlemek için ResourceGroupName ve ServerName parametrelerini belirtin.

## ÖRNEKLERDEN

### Örnek 1: veritabanı için tehdit algılama ilkesini kaldırma
```
PS C:\> Remove-AzSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Bu komut, tehdit algılama ilkesini server01 adlı sunucudan kaldırır.
