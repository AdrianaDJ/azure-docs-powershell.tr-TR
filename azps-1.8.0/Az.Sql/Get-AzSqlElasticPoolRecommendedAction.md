# Get-AzSqlElasticPoolRecommendedAction

## SYNOPSIS
Bir Azure SQL esnek havuz Danışmanı için bir veya daha fazla önerilen eylemi alır.

## INDEKI

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzSqlElasticPoolRecommendedAction** cmdlet 'ı BIR Azure SQL esnek havuz Danışmanı için önerilen bir veya birden çok eylemi alır.
