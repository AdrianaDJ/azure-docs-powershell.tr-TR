# New-AzSqlDatabaseCopy

## SYNOPSIS
Geçerli saatte anlık görüntüyü kullanan bir SQL veritabanının kopyasını oluşturur.

## INDEKI

### Vseçvet (varsayılan)
```
New-AzSqlDatabaseCopy [-DatabaseName] <String> [-ServiceObjectiveName <String>] [-ElasticPoolName <String>]
 [-Tags <Hashtable>] [-CopyResourceGroupName <String>] [-CopyServerName <String>] -CopyDatabaseName <String>
 [-AsJob] [-LicenseType <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Vcorebasevseçdatabase
```
New-AzSqlDatabaseCopy [-DatabaseName] <String> [-Tags <Hashtable>] [-CopyResourceGroupName <String>]
 [-CopyServerName <String>] -CopyDatabaseName <String> [-AsJob] -ComputeGeneration <String> -VCore <Int32>
 [-LicenseType <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**New-AzSqlDatabaseCopy** cmdlet 'i, geçerli zamanda verilerin anlık görüntüsünü kullanan BIR Azure SQL veritabanının kopyasını oluşturur. Tek seferlik veritabanı kopyası oluşturmak için Start-AzSqlDatabaseCopy cmdlet 'i yerine bu cmdlet 'i kullanın. Bu cmdlet kopyanın **veritabanı** nesnesini döndürür.
Not: veritabanında coğrafi çoğaltma yapılandırmak için New-AzSqlDatabaseSecondary cmdlet 'ini kullanın.
Bu cmdlet, Azure 'da SQL Server genişletme veritabanı hizmeti tarafından da desteklenir.
