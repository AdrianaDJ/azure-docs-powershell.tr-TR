# Disable-AzSqlServerAdvancedThreatProtection

## SYNOPSIS
Sunucuda Gelişmiş tehdit korumasını devre dışı bırakır.

## INDEKI

```
Disable-AzSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Disable-AzSqlServerAdvancedThreatProtection** cmdlet 'ı sunucuda Gelişmiş tehdit korumasını devre dışı bırakır.

## ÖRNEKLERDEN

### Örnek 1-Gelişmiş tehdit korumasını devre dışı bırakın
```powershell
PS C:\>  Disable-AzSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```

### Örnek 2-sunucu kaynağından Gelişmiş tehdit korumasını devre dışı bırakın
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Disable-AzSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```
