# Get-AzSqlElasticPoolActivity

## SYNOPSIS
Esnek havuzdaki işlemlerin durumunu alır.

## INDEKI

```
Get-AzSqlElasticPoolActivity [-ServerName] <String> [-ElasticPoolName] <String> [-OperationId <Guid>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Get-AzSqlElasticPoolActivity** cmdlet 'ı BIR Azure SQL veritabanı için esnek havuzda işlemlerin durumunu alır.
Havuz oluşturma ve yapılandırma güncelleştirmelerinin durumunu görebilirsiniz.

## ÖRNEKLERDEN

### Örnek 1: esnek bir havuzun işlemlerinin durumunu alma
```
PS C:\>Get-AzSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Bu komut, ElasticPool01 adlı esnek havuz işlemlerinin durumunu alır.
