# Remove-AzSqlServerDnsAlias

## SYNOPSIS
Azure SQL Server DNS diğer adını kaldırır.

## INDEKI

### Cmdlet Giriş parametrelerinden sunucu DNS diğer adını kaldırma
```
Remove-AzSqlServerDnsAlias -Name <String> -ServerName <String> [-ResourceGroupName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Azuressqlserverdnsaliasmodel örneği tanımından sunucu DNS diğer adlarını kaldırma
```
Remove-AzSqlServerDnsAlias -InputObject <AzureSqlServerDnsAliasModel> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Azure kaynak kimliğinden sunucu DNS diğer adını kaldırma
```
Remove-AzSqlServerDnsAlias -ResourceId <String> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Bu komutlar, sunucudan dokunulmadan Azure SQL Server DNS diğer adını sunucudan kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Remove-AzSqlServerDnsAlias -DnsAliasName aliasName -ServerName serverName -ResourceGroupName rg
```

Azure SQL Server DNS diğer adını sunucudan sunucuadı adı adıyla kaldırır.
