# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## SYNOPSIS
Yönetilen bir örneğin gelişmiş veri güvenliği ilkesini alır.

## INDEKI

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
Yönetilen bir örneğin gelişmiş veri güvenliği ilkesini gösteren **Get-Azsqlınstanceadvancevseçdatasecuritma** cmdlet 'i.

## ÖRNEKLERDEN

### Örnek 1-yönetilen örnek gelişmiş veri güvenliğini alır
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Örnek 2-yönetilen örneğe gelişmiş veri güvenliğini yönetilen örnek kaynağından alır
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
