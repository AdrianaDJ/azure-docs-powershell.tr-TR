# Get-AzLogProfile

## SYNOPSIS
Günlük profili alır.

## INDEKI

```
Get-AzLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzLogProfile** cmdlet 'i bir günlük profili alır.
