# New-AzUserAssignedIdentity

## SYNOPSIS
Yeni bir kullanıcı kimliği oluşturur veya mevcut bir kullanıcı tarafından atanan kimliği güncelleştirir.

## INDEKI

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**New-Azuseratanandentity** cmdlet 'i, yeni bir kullanıcı atanmış kimliği oluşturur. Var olan bir kimlikle kullanıldığında, kimliği güncelledi.
Kimliğe Azure Kaynak Yöneticisi etiketleri eklemek için, lütfen Set-AzResource cmdlet 'ini kullanın.
