# Get-AzConsumptionBudget

## SYNOPSIS
Bir abonelikteki veya kaynak grubundaki bütçelerin listesini alın.

## INDEKI

```
Get-AzConsumptionBudget [-DefaultProfile <IAzureContextContainer>] [-ResourceGroupName <String>]
 [-Name <String>] [<CommonParameters>]
```

## Tanım
Get-AzConsumptionBudget cmdlet 'i, bir abonelikteki veya kaynak grubundaki bütçelerin listesini alır.

## ÖRNEKLERDEN
