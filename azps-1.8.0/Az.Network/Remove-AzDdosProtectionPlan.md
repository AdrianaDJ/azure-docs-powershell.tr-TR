# Remove-AzDdosProtectionPlan

## SYNOPSIS
Bir Vseçdos koruma planını kaldırır.

## INDEKI

```
Remove-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Remove-AzDdosProtectionPlan cmdlet 'i, bir Vseçdos koruma planını kaldırır.

## ÖRNEKLERDEN

### Örnek 1: boş bir Vseçdos koruma planını kaldırma
```
D:\> Remove-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

Bu durumda, bir Vseçdos koruma planını belirtildiği gibi kaldırdık.

### Örnek 2: sanal ağla ilişkili bir Vseçdos koruma planını kaldırma
