# New-AzNetworkProfile

## SYNOPSIS
Yeni ağ profili oluşturur.

## INDEKI

```
New-AzNetworkProfile -ResourceGroupName <String> -Name <String> [-Location <String>] [-Tag <Hashtable>]
 [-ContainerNicConfig <PSContainerNetworkInterfaceConfiguration[]>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**New-AzNetworkProfile** cmdlet 'i, yeni bir ağ profili üst düzey kaynağı oluşturur.

## ÖRNEKLERDEN

### Örnek 1
```powershell
$networkProfile = New-AzNetworkProfile -Name np1 -ResourceGroupName rg1 -Location westus
```

Bu, yeni bir ağ profili üst düzey kaynağı oluşturur
