# Remove-AzPublicIpAddress

## SYNOPSIS
Genel IP adresini kaldırır.

## INDEKI

```
Remove-AzPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azpublicıpaddress** cmdlet 'ı bir Azure genel IP adresini kaldırır.

## ÖRNEKLERDEN

### 1: genel IP adresi kaynağını kaldırma
```
Remove-AzPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Bu komut, kaynak grubundaki $publicIpName adlı ortak IP adresi kaynağını kaldırır $rgName.
