# Test-AzDataLakeStoreAccount

## SYNOPSIS
Veri Lake Store hesabının varlığını sınar.

## INDEKI

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Test-AzDataLakeStoreAccount** cmdlet 'ı Data Lake Store hesabının varlığını sınar.

## ÖRNEKLERDEN

### Örnek 1: hesabı test etme
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Bu komut, ContosoADL adlı hesabın var olup olmadığını sınar.
