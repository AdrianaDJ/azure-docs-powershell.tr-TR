# New-AzVmssIpTagConfig

## SYNOPSIS
Bir VMSS 'nin ağ arabirimi için IP etiket nesnesi oluşturur.

## INDEKI

```
New-AzVmssIpTagConfig [-IpTagType] <String> [-Tag <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**New-AzVmssIpTagConfig** cmdlet 'ı sanal makine ölçek kümesi 'nın (VMSS) ağ arabirimi IÇIN bir IP etiketi yapılandırma nesnesi oluşturur.
Bu cmdlet 'i New-AzVmssIpConfig cmdlet 'in *ıptag* parametresi olarak belirtin.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $iptag = New-AzVmssIpTagConfig -IpTagType 'FirstPartyUsage' -Tag 'Sql'
PS C:\> $ipCfg = New-AzVmssIPConfig -Name 'test' -SubnetId $subnetId -IpTag $ipTag;
```

Bu komut, ' FirstPartyUsage ' türü ve ' SQL ' etiketiyle bir IP etiketi yerel nesnesi oluşturur ve bu IP etiketiyle bir IP yapılandırması oluşturur.
