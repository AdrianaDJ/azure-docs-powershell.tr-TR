# Resume-AzKustoCluster

## SYNOPSIS
Suspeneded kusto kümesini sürdürün.

## INDEKI

### Binaik Vseçresourcegroup (varsayılan)
```
Resume-AzKustoCluster [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Byresourceıd
```
Resume-AzKustoCluster [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Resume-AzKustoCluster [-InputObject] <PSKustoCluster> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Suspeneded kusto kümesini sürdürün.

## ÖRNEKLERDEN

### Örnek 1-askıya alınan bir kusto kümesini adla Sürdür

```
PS C:\> Resume-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster
```

Yukarıdaki komut, "mykustocluster" kaynak grubunda bulunan "" adlı askıya alınmış kusto kümesini sürdürür.

### Örnek 2-askıya alınan bir kusto kümesini boruları kullanarak sürdürme

```
PS C:\> Get-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster | Resume-AzKustoCluster
```

Yukarıdaki komut, cmdlet 'i kullanarak "mykustocluster" kaynak grubunda bulunan "" adlı kusto kümesini alır `Get-AzKustoCluster` ve sonra da sonucu `Resume-AzKustoCluster` devam ettirmek için buna yol gösterir.
