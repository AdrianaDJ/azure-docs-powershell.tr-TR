# Set-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Bir veri Lake Analytics hesabındaki güvenlik duvarı kuralını güncelleştirir.

## INDEKI

```
Set-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [[-StartIpAddress] <String>]
 [[-EndIpAddress] <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-Azdatalakeanalizleri Ticsfirewallrule** cmdlet 'i, bir Azure Data Lake Analytics hesabında güvenlik duvarı kuralını güncelleştirir.

## ÖRNEKLERDEN

### Örnek 1: güvenlik duvarı kuralını güncelleştirme
```
PS C:\>Set-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

Bu komut, yeni IP aralığına sahip olmak için "ContosoAdlAcct" hesabındaki "güvenlik duvarıyla
