# Enable-AzHDInsightOperationsManagementSuite

## SYNOPSIS
Bir HDInsight kümesinde Operations Management Suite (OMS) özelliğini etkinleştirir ve ilgili Günlükler etkinleştir sırasında belirtilen OMS çalışma alanına gönderilir.

## INDEKI

```
Enable-AzHDInsightOperationsManagementSuite [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Enable-AzHDInsightOperationsManagementSuite** cmdlet 'i, bir Azure HDInsight kümesindeki Operations Management SUITE (OMS) özelliğini etkinleştirir.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Enable-AzHDInsightOMS -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Operations Management Suite (OMS), HDInsight kümesinde etkinleştirilecek ve ilgili Günlükler, 1d364e89-bb71-4503-aa3d-a23535aea7bd kimliğine sahip OMS çalışma alanına gönderilecektir.

### Örnek 2
```
PS C:\> Enable-AzHDInsightOMS -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Operations Management Suite (OMS), HDInsight kümesinde etkinleştirilecek ve ilgili Günlükler, 1d364e89-bb71-4503-aa3d-a23535aea7bd kimliğine sahip OMS çalışma alanına gönderilecektir.
