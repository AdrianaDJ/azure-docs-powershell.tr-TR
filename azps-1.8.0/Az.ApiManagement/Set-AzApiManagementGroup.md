# Set-AzApiManagementGroup

## SYNOPSIS
Bir API yönetim grubu yapılandırır.

## INDEKI

```
Set-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-Name <String>]
 [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Set-Azapsananagementgroup** cmdlet 'ı bir API yönetim grubu yapılandırır.

## ÖRNEKLERDEN

### Örnek 1: yönetim grubu yapılandırma
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementGroup -Context $apimContext -Description "Updated Management Group" -Name "Group0001"
```

Bu komut, Group0001 adlı bir yönetim grubunu yapılandırır.
