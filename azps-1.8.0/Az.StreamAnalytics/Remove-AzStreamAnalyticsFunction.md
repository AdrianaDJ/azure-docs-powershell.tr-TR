# Remove-AzStreamAnalyticsFunction

## SYNOPSIS
Stream Analytics işinden bir işlevi siler.

## INDEKI

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzStreamAnalyticsFunction** cmdlet 'ı bir Azure Stream Analytics işinden zaman uyumsuz bir işlevi siler.

## ÖRNEKLERDEN

### Örnek 1: Stream Analytics işlevini kaldırma
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Bu komut, StreamJob22 adındaki işten ScoreTweet adındaki işlevi kaldırır.
