# Remove-AzBatchApplication

## SYNOPSIS
Bir uygulamayı toplu Iş hesabından siler.

## INDEKI

```
Remove-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationId] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Remove-AzBatchApplication** cmdlet 'ı bir Azure toplu hesabından uygulamayı siler.

## ÖRNEKLERDEN

### Örnek 1: bir uygulamayı toplu hesaptan silme
```
PS C:\>Remove-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "Litware"
```

Bu komut, ContosoBatch hesabından Litwin uygulamasını siler.
Uygulama herhangi bir paket içeriyorsa komut başarısız olur.
