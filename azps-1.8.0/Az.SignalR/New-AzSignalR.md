# New-AzSignalR

## SYNOPSIS
Bir SignalR hizmeti oluşturun.

## INDEKI

```
New-AzSignalR [-ResourceGroupName <String>] [-Name] <String> [-Location <String>] [-Sku <String>]
 [-UnitCount <Int32>] [-Tag <System.Collections.Generic.IDictionary`2[System.String,System.String]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Bir SignalR hizmeti oluşturun.
Aşağıdaki değerler belirtilmemişse parametreler için kullanılacaktır:
* `ResourceGroupName`: varsayılan kaynak grubu `Set-AzDefault -ResourceGroupName` .
* `Location`: kaynak grubunun konumu
* `Sku`: `Standard_S1`

## ÖRNEKLERDEN

### Bir Signall seri oluşturma
```powershell
PS C:\> New-AzSignalR -ResourceGroupName myResourceGroup1 -Name mysignalr1 -Location eastus -Sku Standard_S1

mysignalr1.service.signalr.net                     eastus         5002       5001       Succeeded         1.0-preview
```
