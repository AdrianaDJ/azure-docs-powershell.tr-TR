# Remove-AzureRmApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Otomatik ölçeklendirme yapılandırmasını uygulama ağ geçidinden kaldırır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzureRmApplicationGatewayAutoscaleConfiguration** cmdlet 'i var olan uygulama ağ geçidinden otomatik ölçeklendirme yapılandırmasını kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $gw = Get-AzureRmApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzureRmApplicationGateway -ApplicationGateway $gw
```

İlk komut uygulama ağ geçidini alır ve $gw değişkeninde depolar.
İkinci komut, applicationg ağ geçidinden otomatik ölçeklendirme yapılandırmasını kaldırır.
Üçüncü komut, Azure 'daki uygulama ağ geçidini güncelleştirir.
