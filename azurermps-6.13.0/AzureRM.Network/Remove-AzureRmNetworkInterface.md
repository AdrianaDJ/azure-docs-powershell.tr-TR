# Remove-AzureRmNetworkInterface

## SYNOPSIS
Ağ arabirimini kaldırır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmNetworkInterface -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azurermnetworkınterface** cmdlet 'ı bir Azure ağ arabirimini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: ağ arabirimini kaldırma
```
PS C:\>Remove-AzureRmNetworkInterface -Name "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

Bu komut, kaynak grubundaki ResourceGroup1 NetworkInterface1 ağ arabirimini kaldırır.
*Force* parametresi kullanılmadığından, kullanıcıdan bu eylemi onaylaması istenir.

### Örnek 2: ağ arabirimini kaldırma
```
PS C:\>Get-AzureRmNetworkInterface -ResourceGroupName "ResourceGroup1" | Remove-AzureRmNetworkInterface -Force
```

Bu komut, kaynak grubu ResourceGroup1 tüm ağ arabirimlerini kaldırır.
*Force* parametresi kullanıldığından, kullanıcıdan onay istenmez.
