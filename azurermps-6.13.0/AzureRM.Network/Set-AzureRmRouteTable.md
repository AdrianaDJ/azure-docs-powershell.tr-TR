# Set-AzureRmRouteTable

## SYNOPSIS
Yol tablosunun hedef durumunu ayarlar.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Set-AzureRmRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-AzureRmRouteTable** cmdlet 'ı bir Azure yol tablosunun hedef durumunu ayarlar.
