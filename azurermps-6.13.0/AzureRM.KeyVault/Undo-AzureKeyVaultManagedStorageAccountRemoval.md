# Undo-AzureKeyVaultManagedStorageAccountRemoval

## SYNOPSIS
Önceden silinmiş bir Keykasası yönetimli depolama hesabını kurtarır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### Varsayılan (varsayılan)
```
Undo-AzureKeyVaultManagedStorageAccountRemoval [-VaultName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Undo-AzureKeyVaultManagedStorageAccountRemoval
 [-InputObject] <PSDeletedKeyVaultManagedStorageAccountIdentityItem> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Undo-AzureKeyVaultManagedStorageAccountRemoval** komutu önceden silinmiş bir yönetilen depolama hesabını kurtarır, bu kasa için geçici yazılımdan silme özelliği etkin ve kurtarma girişimi de kurtarma aralığı sırasında gerçekleşir.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzureKeyVaultManagedStorageAccount -VaultName myVault -Name myAccount -InRemovedState
PS C:\> Undo-AzureKeyVaultManagedStorageAccountRemoval -VaultName myVault -Name myAccount

                      orage/storageAccounts/myaccount

Regeneration Period : 90.00:00:00
Enabled             : True
Created             : 4/25/2018 1:50:32 AM
Updated             : 4/25/2018 1:50:32 AM
Tags                :
```

Bu komut dizisi, belirtilen depolama hesabının kasada silinmiş bir durumda olup olmadığını belirler; Sonraki komut silinmiş depolama hesabını kurtarır ve etkin bir duruma geri döner.
