# AzureRM. DeviceProvisioningServices modülü
## Tanım
Azure IoT Hub cihaz sağlama hizmetini yönetme komutları.

## AzureRM. DeviceProvisioningServices cmdlet 'Leri
### [Add-AzureRmIoTDeviceProvisioningServiceAccessPolicy](Add-AzureRmIoTDeviceProvisioningServiceAccessPolicy.md)
Azure IoT Hub cihaz sağlama hizmetine yeni bir paylaşılan erişim ilkesi ekleyin.

### [Add-AzureRmIoTDeviceProvisioningServiceCertificate](Add-AzureRmIoTDeviceProvisioningServiceCertificate.md)
Bir Azure IoT Hub cihaz sağlama hizmeti sertifikası oluşturun/güncelleştirin.

### [Add-AzureRmIoTDeviceProvisioningServiceLinkedHub](Add-AzureRmIoTDeviceProvisioningServiceLinkedHub.md)
Bir Azure IoT Hub cihaz sağlama hizmetine bağlı IoT Hub.

### [Get-AzureRmIoTDeviceProvisioningService](Get-AzureRmIoTDeviceProvisioningService.md)
Tüm bilgileri Listele veya Azure IoT Hub cihaz sağlama hizmetlerinin ayrıntılarını göster.

### [Get-AzureRmIoTDeviceProvisioningServiceAccessPolicy](Get-AzureRmIoTDeviceProvisioningServiceAccessPolicy.md)
Azure IoT Hub cihaz sağlama hizmetinde tüm paylaşılan erişim ilkelerinin ayrıntılarını listeler veya ayrıntılarını gösterin.

### [Get-AzureRmIoTDeviceProvisioningServiceCertificate](Get-AzureRmIoTDeviceProvisioningServiceCertificate.md)
Tüm sertifikaları veya Azure IoT Hub cihaz sağlama hizmeti içindeki belirli bir sertifikayı listeler.

### [Get-AzureRmIoTDeviceProvisioningServiceLinkedHub](Get-AzureRmIoTDeviceProvisioningServiceLinkedHub.md)
Bir Azure IoT Hub cihaz sağlama hizmetinde, bağlantılı IoT Hub 'larının tüm ayrıntılarını listeler veya ayrıntılarını gösterin.

### [Yeni-AzureRmIoTDeviceProvisioningService](New-AzureRmIoTDeviceProvisioningService.md)
Azure IoT Hub cihazı sağlama hizmeti oluşturun.

### [New-AzureRmIoTDeviceProvisioningServiceCertificateVerificationCode](New-AzureRmIoTDeviceProvisioningServiceCertificateVerificationCode.md)
Azure IoT Hub cihaz sağlama hizmeti sertifikası için bir doğrulama kodu oluşturun.

### [Remove-AzureRmIoTDeviceProvisioningService](Remove-AzureRmIoTDeviceProvisioningService.md)
Azure IoT Hub cihazı sağlama hizmetini silme.

### [Remove-AzureRmIoTDeviceProvisioningServiceAccessPolicy](Remove-AzureRmIoTDeviceProvisioningServiceAccessPolicy.md)
Azure IoT Hub cihaz sağlama hizmetinde paylaşılan erişim ilkelerini silme.

### [Remove-AzureRmIoTDeviceProvisioningServiceCertificate](Remove-AzureRmIoTDeviceProvisioningServiceCertificate.md)
Azure IoT Hub cihaz sağlama hizmeti sertifikasını silme.

### [Remove-AzureRmIoTDeviceProvisioningServiceLinkedHub](Remove-AzureRmIoTDeviceProvisioningServiceLinkedHub.md)
Bir Azure IoT Hub cihaz sağlama hizmetinde bağlantılı bir IoT Hub 'ı silin.

### [Set-AzureRmIoTDeviceProvisioningServiceCertificate](Set-AzureRmIoTDeviceProvisioningServiceCertificate.md)
Bir Azure IoT Hub cihaz sağlama hizmeti sertifikasını doğrulayın.

### [Update-AzureRmIoTDeviceProvisioningService](Update-AzureRmIoTDeviceProvisioningService.md)
Azure IoT Hub cihazı sağlama hizmetini güncelleştirme.

### [Güncelleştirme-AzureRmIoTDeviceProvisioningServiceAccessPolicy](Update-AzureRmIoTDeviceProvisioningServiceAccessPolicy.md)
Azure IoT Hub cihaz sağlama hizmetinde paylaşılan bir Access ilkesini güncelleyin.

### [Güncelleştirme-AzureRmIoTDeviceProvisioningServiceLinkedHub](Update-AzureRmIoTDeviceProvisioningServiceLinkedHub.md)
Bir Azure IoT Hub cihaz sağlama hizmetinde bağlantılı bir IoT Hub 'ı güncelleştirin.
