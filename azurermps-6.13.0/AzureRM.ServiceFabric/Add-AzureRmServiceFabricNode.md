# Add-AzureRmServiceFabricNode

## SYNOPSIS
Kümedeki belirli bir düğüme düğüm ekleme.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Add-AzureRmServiceFabricNode -NumberOfNodesToAdd <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Belirli bir düğüm türüne düğüm eklemek için **Add-AzureRmServiceFabricNode** öğesini kullanın. Yalnızca düğüm türüne eklemek istediğiniz düğüm sayısını belirtmeniz gerekir.

## ÖRNEKLERDEN

### Örnek 1
```
PS c:> Add-AzureRmServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Bu komut, ' N1 ' düğüm türüne 2 düğüm ekler.
