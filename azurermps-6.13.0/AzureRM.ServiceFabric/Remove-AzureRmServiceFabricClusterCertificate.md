# Remove-AzureRmServiceFabricClusterCertificate

## SYNOPSIS
Küme güvenliği için kullanılan küme sertifikasını kaldırma.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
Küme sertifikasını kümeden kaldırmak için, kümede zaten kullanımda olan başka bir sertifika olduğu sürece **Remove-AzureRmServiceFabricClusterCertificate** öğesini kullanın.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Remove-AzureRmServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Bu komut, küme güvenliği için parmak izi 5F3660C715EBBDA31DB1FFDCF508302348DE8E7A kullanılmasını kaldırır.
