# Start-AzureRmVM

## SYNOPSIS
Azure sanal makinesini başlatır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### ResourceGroupNameParameterSetName (varsayılan)
```
Start-AzureRmVM [-Name] <String> [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Idparametersetname
```
Start-AzureRmVM [-Name] <String> [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Start-AzureRmVM** cmdlet 'ı bir Azure sanal makinesini başlatır.

## ÖRNEKLERDEN

### Örnek 1: sanal makine başlatma
```
PS C:\> Start-AzureRmVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Bu komut, ResourceGroup11 'da VirtualMachine07 adındaki sanal makineyi başlatır.
