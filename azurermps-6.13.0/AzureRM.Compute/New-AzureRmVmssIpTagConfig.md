# New-AzureRmVmssIpTagConfig

## SYNOPSIS
Bir VMSS 'nin ağ arabirimi için IP etiket nesnesi oluşturur.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
New-AzureRmVmssIpTagConfig [-IpTagType] <String> [-Tag <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Yeni-AzureRmVmssIpTagConfig** cmdlet 'ı sanal makine ölçek kümesinin (VMSS) ağ arabirimi IÇIN bir IP etiketi yapılandırma nesnesi oluşturur.
Bu cmdlet 'i New-AzureRmVmssIpConfig cmdlet 'in *ıptag* parametresi olarak belirtin.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $iptag = New-AzureRmVmssIpTagConfig -IpTagType 'FirstPartyUsage' -Tag 'Sql'
PS C:\> $ipCfg = New-AzureRmVmssIPConfig -Name 'test' -SubnetId $subnetId -IpTag $ipTag;
```

Bu komut, ' FirstPartyUsage ' türü ve ' SQL ' etiketiyle bir IP etiketi yerel nesnesi oluşturur ve bu IP etiketiyle bir IP yapılandırması oluşturur.
