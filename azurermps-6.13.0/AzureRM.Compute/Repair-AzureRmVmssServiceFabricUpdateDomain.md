# Repair-AzureRmVmssServiceFabricUpdateDomain

## SYNOPSIS
El ile platform güncelleştirme etki alanı hizmet yapısı sanal makine ölçek kümesindeki sanal makineleri güncelleştirmeye yönelik ilerleme.

## Tanım
El ile platform güncelleştirme etki alanını zorla hizmet yapısı sanal makine ölçek kümesindeki sanal makineleri güncelleştirmeye ilerleme.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Repair-AzureRmVmssServiceFabricUpdateDomain -ResourceGroupName $rgname -VMScaleSetName $vmssName -PlatformUpdateDomain 0
```

Bu komut, kaynak grubu adı ve ölçek kümesi adıyla belirtilen sanal makine ölçek kümesi

### Örnek 2
```
PS C:\> $vmss = Get-AzureRmVmss -ResourceGroupName $rgname -VMScaleSetName $vmssName
PS C:\> Repair-AzureRmVmssServiceFabricUpdateDomain -VirtualMachineScaleSet $vmss -PlatformUpdateDomain 1
```

Bu komut, VM Ölçek kümesi nesnesinin belirttiği sanal makine ölçeği kümesi için UD 1 ' de hizmet yapısı güncelleştirmesini zorlar.

### Örnek 3
```
PS C:\> Repair-AzureRmVmssServiceFabricUpdateDomain -ResourceId $resoureId  -PlatformUpdateDomain 2;
```

Bu komut, kaynak kimliği tarafından belirtilen sanal makine ölçeği kümesi için UD 2 ' ye hizmet yapısı güncelleştirmesini zorlar.
