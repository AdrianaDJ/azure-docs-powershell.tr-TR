# Remove-AzureRmDataLakeStoreAccount

## SYNOPSIS
Data Lake Store hesabını kalıcı olarak siler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzureRmDataLakeStoreAccount** cmdlet 'ı bir Data Lake Store hesabını kalıcı olarak siler.

## ÖRNEKLERDEN

### Örnek 1: veri Lake Store hesabını kaldırma
```
PS C:\>Remove-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Bu komut, Data Lake Store 'dan ContosoADL adlı hesabı kaldırır.
