# AzureRM. ManagementPartner modülü
## Tanım
Azure Resource Manager için yönetim ortağı cmdlet 'leri.

## AzureRM. Yönetimiş ortağı cmdlet 'Leri
### [Get-AzureRmManagementPartner](Get-AzureRmManagementPartner.md)
Geçerli kimliği doğrulanmış kullanıcı veya hizmet sorumlusunun Microsoft Iş ortağı ağı (MPN) KIMLIĞINI alır. 

### [Yeni-AzureRmManagementPartner](New-AzureRmManagementPartner.md)
Bir Microsoft Iş ortağı ağının (MPN) KIMLIĞINI geçerli kimlik doğrulamalı Kullanıcı veya hizmet sorumlusuna ilişkilendirir.

### [Remove-AzureRmManagementPartner](Remove-AzureRmManagementPartner.md)
Geçerli kimliği doğrulanmış kullanıcı veya hizmet sorumlusunun Microsoft Iş ortağı ağı (MPN) KIMLIĞINI silin.

### [Güncelleştirme-AzureRmManagementPartner](Update-AzureRmManagementPartner.md)
Geçerli kimliği doğrulanmış kullanıcı veya hizmet sorumlusunun Microsoft Iş ortağı ağı (MPN) KIMLIĞINI güncelleştirir.
