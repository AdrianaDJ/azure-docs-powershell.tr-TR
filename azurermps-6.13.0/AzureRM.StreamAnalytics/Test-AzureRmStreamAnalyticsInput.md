# Test-AzureRmStreamAnalyticsInput

## SYNOPSIS
Girişin bağlantı durumunu sınar.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Test-AzureRmStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Test-AzureRmStreamAnalyticsInput** cmdlet 'ı, akış çözümlemelerini bir girişe bağlanma olanağını sınar.

## ÖRNEKLERDEN

### Örnek 1: Giriş akışının bağlantı durumunu sınama
```
PS C:\>Test-AzureRmStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Bu, StreamingJob 'daki giriş girişi akışının bağlantı durumunu sınar.
