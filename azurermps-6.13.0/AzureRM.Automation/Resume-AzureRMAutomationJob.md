# Resume-AzureRmAutomationJob

## SYNOPSIS
Askıya alınan Otomasyon işini sürdürür.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Resume-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Özgeçmiş-AzureRmAutomationJob** cmdlet 'i askıya alınmış bir Azure Otomasyonu işini sürdürür.
Askıya alınan işi belirtin.
Bir işi askıya almak için Suspend-AzureRmAutomationJob cmdlet 'ini kullanın.

## ÖRNEKLERDEN

### Örnek 1: askıya alınan işi sürdürme
```
PS C:\>Resume-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Bu komut belirtilen KIMLIĞE sahip işi sürdürür.
