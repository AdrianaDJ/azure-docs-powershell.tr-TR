# Update-AzureRmAutomationSourceControl

## SYNOPSIS
Azure Otomasyonu kaynak denetimini güncelleştirir.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Update-AzureRmAutomationSourceControl -Name <String> [-AccessToken <SecureString>] [-FolderPath <String>]
 [-Branch <String>] [-Description <String>] [-AutoSync <Boolean>] [-PublishRunbook <Boolean>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Update-AzureRmAutomationSourceControl cmdlet 'i, Azure Otomasyonu 'nda kaynak denetimindeki bir alanın değerini değiştirir.

## ÖRNEKLERDEN

### Örnek 1
Bu komutlar, Devrunbook alanını, Devsnative adlı hesap adındaki VSTSNative için false olarak ayarlar.


```powershell
Update-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                      -AutomationAccountName "devAccount" `
                                      -Name "VSTSNative" `
                                      -PublishRunbook $false

----            ---------- ------ ----------  -------- -------------- -------

```
