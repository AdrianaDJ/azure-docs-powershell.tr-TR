# Remove-AzureRmApiManagementGroup

## SYNOPSIS
Var olan bir API yönetim grubunu kaldırır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azurermapsananagementgroup** cmdlet 'i var olan bir API yönetim grubunu kaldırır.

## ÖRNEKLERDEN

### Örnek 1: var olan bir yönetim grubunu kaldırma
```powershell
PS C:\>$apimContext = New-AzureRmApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzureRmApiManagementGroup -Context $apimContext -GroupId "Group0001" -Force
```

Bu komut Group0001 adlı bir yönetim grubunu kaldırır ve kullanıcıya onay istemez.
