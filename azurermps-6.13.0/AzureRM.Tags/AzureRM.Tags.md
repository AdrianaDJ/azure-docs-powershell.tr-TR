# AzureRM. Etiketler modülü
## Tanım
Bu konuda, Azure etiket cmdlet 'Lerinin yardım konularını görüntüler.

## AzureRM. etiket cmdlet 'Leri
### [Get-AzureRmTag](Get-AzureRmTag.md)
Önceden tanımlanmış Azure etiketlerini alır.

### [Yeni-AzureRmTag](New-AzureRmTag.md)
Önceden tanımlanmış bir Azure etiketi oluşturur veya varolan bir etikete değerler ekler.

### [Remove-AzureRmTag](Remove-AzureRmTag.md)
Önceden tanımlanmış Azure etiketlerini veya değerlerini siler.
