# Get-AzureRmConsumptionReservationDetail

## SYNOPSIS
Sağlanan tarih aralığı için rezervasyonlar bilgilerini alın.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmConsumptionReservationDetail -StartDate <DateTime> -EndDate <DateTime> -ReservationOrderId <String>
 [-ReservationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azurermtüketimptionrezervationdetail** cmdlet 'i, sağlanan tarih aralığı için rezervasyonlar ayrıntılarını alır.
