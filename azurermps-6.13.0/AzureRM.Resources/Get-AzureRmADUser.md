# Get-AzureRmADUser

## SYNOPSIS
Active Directory kullanıcılarına filtre uygular.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### EmptyParameterSet (varsayılan)
```
Get-AzureRmADUser [-UserPrincipalName <String>] [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### SearchStringParameterSet
```
Get-AzureRmADUser -StartsWith <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Get-AzureRmADUser -DisplayName <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### NesneKimliği
```
Get-AzureRmADUser -ObjectId <Guid> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### UPNParameterSet
```
Get-AzureRmADUser -UserPrincipalName <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### MailParameterSet
```
Get-AzureRmADUser -Mail <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

## Tanım
Active Directory kullanıcılarına filtre uygular.

## ÖRNEKLERDEN

### Örnek 1-tüm kullanıcıları listeler

```
PS C:\> Get-AzureRmADUser
```

Kiracıdaki tüm reklam kullanıcılarını listeler.

### Örnek 2-sayfalama kullanan tüm kullanıcıları listeler

```
PS C:\> Get-AzureRmADUser -First 100
```

Kiracıdaki ilk 100 AD kullanıcılarını listeler.

### Örnek 3-Kullanıcı asıl adına göre AD kullanıcısını alma

```
PS C:\> Get-AzureRmADUser -UserPrincipalName foo@domain.com
```

AD kullanıcısını Kullanıcı asıl adı "" ile alır foo@domain.com .

### Örnek 4-arama dizesine göre liste

```
PS C:\> Get-AzureRmADUser -SearchString Joe
```

Görünen adı "Joe" ile başlayan tüm AD kullanıcılarını listeler.
