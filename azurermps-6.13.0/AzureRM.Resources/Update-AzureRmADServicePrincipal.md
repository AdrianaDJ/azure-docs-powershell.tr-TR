# Update-AzureRmADServicePrincipal

## SYNOPSIS
Var olan bir Azure Active Directory hizmet sorumlusunu güncelleştirir.

## Tanım
Var olan bir Azure Active Directory hizmet sorumlusunu güncelleştirir. Bu hizmet sorumlusunun ilişkili kimlik bilgilerini güncelleştirmek için lütfen New-AzureRmADSpCredential cmdlet 'ini kullanın. Temel uygulamayla ilişkili özellikleri güncelleştirmek için lütfen Update-AzureRmADApplication cmdlet 'ini kullanın.

## ÖRNEKLERDEN

### Örnek 1-hizmet sorumlusunun görünen adını güncelleyin

```
PS C:\> Update-AzureRmADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 -DisplayName MyNewDisplayName
```

' 784136ca-3ae2-4fdd-A388-89r793e7c780 ' ile hizmet sorumlusunun görünen adını ' Yenisdisplayname ' olarak güncelleştirir.

### Örnek 2-boru kullanarak hizmet sorumlusunun görünen adını güncelleştirme

```
PS C:\> Get-AzureRmADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 | Update-AzureRmADServicePrincipal -DisplayName MyNewDisplayName
```

' 784136ca-3ae2-4fdd-A388-89r793e7c780 ' ile hizmet sorumlusunu ve hizmet sorumlusunun görünen adını "Yenisör" olarak güncelleştirmek için Update-AzureRmADServicePrincipal cmdlet 'ine sahip kanalları alır.
