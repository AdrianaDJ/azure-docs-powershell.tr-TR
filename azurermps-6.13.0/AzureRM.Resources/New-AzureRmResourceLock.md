## Tanım
**New-AzureRmResourceLock** cmdlet 'i kaynak kilidi oluşturur.

## ÖRNEKLERDEN

### Örnek 1: Web sitesinde kaynak kilidi oluşturma
```
PS C:\>New-AzureRmResourceLock -LockLevel CanNotDelete -LockNotes "My lock notes" -LockName "ContosoSiteLock" -ResourceName "ContosoSite" -ResourceType "microsoft.web/sites"
```

Bu komut, Web sitesinde kaynak kilidi oluşturur.
