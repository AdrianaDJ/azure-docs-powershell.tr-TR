# Stop-AzureRmSqlServerUpgrade

## SYNOPSIS
SQL veritabanı sunucusunu yükseltmeyi durdurur.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Stop-AzureRmSqlServerUpgrade [-Force] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Stop-AzureRmSqlServerUpgrade** cmdlet 'ı BIR Azure SQL veritabanı sunucusunun yükseltmesini durdurur.

## ÖRNEKLERDEN

### Örnek 1: sunucu yükseltmesini durdurma
```
PS C:\>Stop-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server02"
```

Bu komut, ResourceGroup01 adındaki kaynak grubuna atanmış Server02 adlı sunucuyu yükseltme isteğini durdurur.
