# Get-AzureRmSqlElasticPool

## SYNOPSIS
Bir Azure SQL veritabanında esnek havuzlar ve özellik değerlerini alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azurermsqlelaraz Pool** cmdlet 'i esnek havuzlar ve özellik değerlerini alır.
Yalnızca bu havuzun özellik değerlerini görmek için varolan esnek havuzun adını belirtin.
