# New-AzureRmSqlInstanceDatabase

## SYNOPSIS
Azure SQL yönetilen örnek veritabanı oluşturur.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### CreateNewInstanceDatabaseFromInputParameters (varsayılan)
```
New-AzureRmSqlInstanceDatabase [-Name] <String> [-InstanceName] <String> [-ResourceGroupName] <String>
 [-Collation <String>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

```

```

```

 [-InstanceResourceId] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Yeni-Azurermsqlınstancedatabase** cmdlet 'ı Azure SQL yönetilen örnek veritabanı oluşturur.
