# Get-AzureRmSqlServerUpgrade

## SYNOPSIS
Azure SQL veritabanı sunucusu yükseltmesinin durumunu alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlServerUpgrade -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Get-AzureRmSqlServerUpgrade** cmdlet 'ı, Azure SQL veritabanı sunucusu yükseltmesinin durumunu alır.

## ÖRNEKLERDEN

### Örnek 1: yükseltme durumunu alma
```
PS C:\>Get-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Format-List
ResourceGroupName               : resourcegroup01
ServerName                      : server01
Status                          : Queued
```

Bu komut, ResourceGroup01 adındaki kaynak grubundaki server01 adındaki sunucudan yükseltme durumunu alır.
