# Get-AzureRmSqlInstance

## SYNOPSIS
Azure SQL yönetilen veritabanı örneği hakkında bilgi döndürür.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### Getınstancebyresourcegroup (varsayılan)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Getınstancebinaik Vseçresourcegroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azurermsqlınstance** cmdlet 'i, bir veya daha fazla Azure SQL yönetilen örneği hakkında bilgi döndürür.
Yalnızca bu örneğin bilgilerini görmek için bir örneğin adını belirtin.
