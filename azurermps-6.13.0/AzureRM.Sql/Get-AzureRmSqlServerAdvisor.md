# Get-AzureRmSqlServerAdvisor

## SYNOPSIS
Bir Azure SQL Server için bir veya daha fazla danışmanları alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzureRmSqlServerAdvisor** cmdlet 'i, BIR Azure SQL Server için bir veya daha fazla Azure SQL Server danışmanını alır.
