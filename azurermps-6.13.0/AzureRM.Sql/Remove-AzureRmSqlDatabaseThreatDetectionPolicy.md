# Remove-AzureRmSqlDatabaseThreatDetectionPolicy

## SYNOPSIS
Tehdit algılama ilkesini veritabanından kaldırır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmSqlDatabaseThreatDetectionPolicy [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Remove-AzureRmSqlDatabaseThreatDetectionPolicy** cmdlet 'i, tehdit algılama Ilkesini Azureazma SQL veritabanından kaldırır.
Bu cmdlet 'i kullanmak için, bu cmdlet 'in ilkeyi kaldırdığı veritabanını tanımlamak için *Resourcegroupname* ve *ServerName* parametrelerini belirtin.

## ÖRNEKLERDEN

### Örnek 1: veritabanı için tehdit algılama ilkesini kaldırma
```
PS C:\>Remove-AzureRmSqlDatabaseThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

Bu komut, server01 adındaki sunucudaki Database01 adlı veritabanından tehdit algılama ilkesini kaldırır.
