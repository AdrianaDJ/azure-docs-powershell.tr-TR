# Get-AzureRmSqlDatabaseAdvisor

## SYNOPSIS
Bir Azure SQL veritabanı için bir veya daha fazla danışmanları alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Get-AzureRmSqlDatabaseAdvisor** cmdlet 'i, BIR Azure SQL veritabanı için bir veya birden çok Azure SQL veritabanı danışmanını alır.
