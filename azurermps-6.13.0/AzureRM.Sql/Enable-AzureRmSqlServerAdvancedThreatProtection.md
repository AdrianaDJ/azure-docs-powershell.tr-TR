# Enable-AzureRmSqlServerAdvancedThreatProtection

## SYNOPSIS
Sunucuda Gelişmiş tehdit korumasını etkinleştirmiştir.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Enable-AzureRmSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Enable-AzureRmSqlServerAdvancedThreatProtection** cmdlet 'ı sunucuda Gelişmiş tehdit korumasını etkinleştirir.

## ÖRNEKLERDEN

### Örnek 1-sunucu Gelişmiş tehdit korumasını etkinleştir
```powershell
PS C:\>  Enable-AzureRmSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Örnek 2-sunucu kaynağından Gelişmiş tehdit korumasını etkinleştir
```powershell
PS C:\>  Get-AzureRmSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzureRmSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
