# AzureRM. Haritalar modülü
## Tanım
Bu konuda, Azure haritalar cmdlet 'Lerinin yardımı görüntülenir.

## AzureRM. Haritalar cmdlet 'Leri
### [Get-AzureRmMapsAccount](Get-AzureRmMapsAccount.md)
Hesabı alır.

### [Get-AzureRmMapsAccountKey](Get-AzureRmMapsAccountKey.md)
Bir hesabın API anahtarlarını alır.
Bu tuşlar, Azure Maps 'e sonraki çağrılarda kullanılan kimlik doğrulama mekanizmasıdır.

### [Yeni-AzureRmMapsAccount](New-AzureRmMapsAccount.md)
Azure haritalar hesabı oluşturur.

### [New-AzureRmMapsAccountKey](New-AzureRmMapsAccountKey.md)
Hesap anahtarını yeniden oluşturur.

### [Remove-AzureRmMapsAccount](Remove-AzureRmMapsAccount.md)
Azure haritalar hesabını siler.
