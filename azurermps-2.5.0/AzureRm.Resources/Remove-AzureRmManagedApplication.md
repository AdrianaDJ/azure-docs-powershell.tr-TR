# Remove-AzureRmManagedApplication

## SYNOPSIS
Yönetilen uygulamayı kaldırır

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### Removebynaik Vseçresourcegroup (varsayılan)
```
Remove-AzureRmManagedApplication -Name <String> -ResourceGroupName <String> [-Force] [-ApiVersion <String>]
 [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Removebyıd
```
Remove-AzureRmManagedApplication -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzureRmManagedApplication** cmdlet 'i yönetilen uygulamayı kaldırır

## ÖRNEKLERDEN

### Örnek 1: yönetilen uygulamayı kaynak KIMLIĞIYLE kaldırma
```
PS C:\>$Application = Get-AzureRmManagedApplication -Name "myApp" -ResourceGroupName "myRG"
PS C:\>Remove-AzureRmManagedApplication -Id $Application.ResourceId -Force
```

İlk komut, Uygulamam adlı bir yönetilen uygulamayı Get-AzureRmManagedApplication cmdlet 'i kullanarak alır.
Komut, $Application değişkeninde depolar.

İkinci komut $Application 'in **RESOURCEID** özelliğiyle tanımlanan yönetilen uygulamayı kaldırır.
