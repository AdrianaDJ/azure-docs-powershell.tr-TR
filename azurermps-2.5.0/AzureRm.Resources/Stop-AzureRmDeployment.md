# Stop-AzureRmDeployment

## SYNOPSIS
Çalışan bir dağıtım

## INDEKI

### StopByDeploymentName (varsayılan)
```
Stop-AzureRmDeployment [-Name] <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzureRmDeployment -Id <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzureRmDeployment -InputObject <PSDeployment> [-PassThru] [-ApiVersion <String>] [-Pre]

```

## Tanım
**Stop-AzureRmDeployment** cmdlet 'i, abonelik kapsamındaki ancak tamamlanmamış bir dağıtımı iptal eder.
Bir dağıtımı durdurmak için dağıtımın, sağlama gibi tamamlanmamış bir sağlama durumu olmalıdır; bunun sağlanması veya başarısız gibi tamamlanmamış bir durumda olmaması gerekir.

Abonelik kapsamında dağıtım oluşturmak için New-AzureRmDeployment cmdlet 'ini kullanın.

Bu cmdlet yalnızca bir çalışan dağıtımı durdurur. Belirli bir dağıtımı durdurmak için *Name* parametresini kullanın.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\>Stop-AzureRmDeployment -Name "deployment01"
```

Bu komut, geçerli abonelik kapsamında çalışan "deployment01" dağıtımını iptal eder.

### Örnek 2
```
PS C:\>Get-AzureRmDeployment -Name "deployment01" | Stop-AzureRmDeployment
```

Bu komut, geçerli abonelik kapsamındaki "deployment01" dağıtımını alır ve iptal eder.
