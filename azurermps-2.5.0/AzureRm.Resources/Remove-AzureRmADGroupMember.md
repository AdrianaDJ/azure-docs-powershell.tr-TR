# Remove-AzureRmADGroupMember

## SYNOPSIS

## Tanım
Bir AD grubundan kullanıcıyı kaldırır.

## ÖRNEKLERDEN

### Örnek 1-Group By nesne kimliğinden Kullanıcı kaldırma

```
PS C:\> Remove-AzureRmADGroup -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405 -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Nesne kimliği ' D9076BBC-D62C-4105-9C78-A7F5BC4A3405 ' olan kullanıcıyı nesne kimliği ' 85F89C90-780E-4AA6-9F4F-6F268VSEÇ322EEE ' olan gruptan kaldırır.

### Örnek 2-şeridi kullanarak gruptan bir kullanıcıyı kaldırma

```
PS C:\> Get-AzureRmADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Remove-AzureRmADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405
```

Nesne kimliği ' 85F89C90-780E-4AA6-9F4F-6F268VSEÇ322EEE ' ve Remove-AzureRmADGroupMember yöneltmeleri
