# Add-AzureRmADGroupMember

## SYNOPSIS
Var olan bir AD grubuna kullanıcı ekler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

### Memberobjectivseçwithgroupobjectid (varsayılan)


## Tanım
Var olan bir AD grubuna kullanıcı ekler.

## ÖRNEKLERDEN

### Örnek 1-gruba göre kullanıcıya ekleme kimliği

```
PS C:\> Add-AzureRmADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405 -TargetGroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Nesne kimliği ' D9076BBC-D62C-4105-9C78-A7F5BC4A3405 ' olan kullanıcıyı nesne kimliği ' 85F89C90-780E-4AA6-9F4F-6F268VSEÇ322EEE ' olan gruba ekler.

### Örnek 2-boru by bir gruba kullanıcı ekleme

```
PS C:\> Get-AzureRmADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Add-AzureRmADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405
```

Nesne kimliği ' 85F89C90-780E-4AA6-9F4F-6F268VSEÇ322EEE ' ve yöneltmeleri, kullanıcıyı o gruba eklemek için Add-AzureRmADGroupMember cmdlet 'e alır.
