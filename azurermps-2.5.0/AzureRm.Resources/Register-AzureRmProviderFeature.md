# Register-AzureRmProviderFeature

## SYNOPSIS
Hesabınızda bir Azure sağlayıcı özelliği kaydeder.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Register-AzureRmProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Register-AzureRmProviderFeature** cmdlet 'i hesabınıza bir Azure sağlayıcı özelliği kaydeder.

## ÖRNEKLERDEN

### Örnek 1: bir özelliği kaydetme
```
PS C:\>Register-AzureRmProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Bu, Microsoft. Network için AllowApplicationSecurityGroups özelliğini hesabınıza ekler.
