# Remove-AzureKeyVaultCertificateOperation

## SYNOPSIS
Anahtar kasasından sertifika işlemini siler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzureKeyVaultCertificateOperation** cmdlet 'i bir anahtar kasasından sertifika işlemini siler.

## ÖRNEKLERDEN

### Örnek 1: sertifika işlemini kaldırma
```
PS C:\>Remove-AzureKeyVaultCertificateOperation -VaultName "ContosoKV01" -Name "TestCert01" -Force
```

Bu komut, TestCert01 adındaki sertifika işlemini, onay istemeden ContosoKV01 anahtar kasasından kaldırır.
