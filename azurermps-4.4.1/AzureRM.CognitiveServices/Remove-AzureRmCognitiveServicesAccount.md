# Remove-AzureRmCognitiveServicesAccount

## SYNOPSIS
Bir öğretici hizmetler hesabını siler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azurermöğretici Iveservicesaccount** cmdlet 'ı belirtilen öğretici sitif hizmetler hesabını siler.
