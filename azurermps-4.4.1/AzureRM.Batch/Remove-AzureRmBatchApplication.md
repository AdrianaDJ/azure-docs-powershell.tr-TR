# Remove-AzureRmBatchApplication

## SYNOPSIS
Bir uygulamayı toplu Iş hesabından siler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationId] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Remove-AzureRmBatchApplication** cmdlet 'ı bir Azure toplu hesabından uygulamayı siler.

## ÖRNEKLERDEN

### Örnek 1: bir uygulamayı toplu hesaptan silme
```
PS C:\>Remove-AzureRmBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "Litware"
```

Bu komut, ContosoBatch hesabından Litwin uygulamasını siler.
Uygulama herhangi bir paket içeriyorsa komut başarısız olur.
