# Remove-AzureBatchJobSchedule

## SYNOPSIS
Toplu iş çizelgesini kaldırır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureBatchJobSchedule [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzureBatchJobSchedule** cmdlet 'ı bir Azure toplu iş çizelgesini kaldırır.
