# Get-AzureRmBatchApplication

## SYNOPSIS
Belirtilen uygulama hakkında bilgi alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [[-ApplicationId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzureRmBatchApplication** cmdlet 'ı bir Azure toplu hesabındaki bir uygulama hakkında bilgi alır.

## ÖRNEKLERDEN

### Örnek 1: uygulamaları bir toplu Iş hesabında görüntüleme
```
PS C:\>Get-AzureRmBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup"
ApplicationId AllowUpdates DisplayName

------------- ------------ ----------------------------

litware       False        Litware Advanced Reticulator
```

Bu komut, ContosoBatch hesabındaki tüm uygulamaları görüntüler.
