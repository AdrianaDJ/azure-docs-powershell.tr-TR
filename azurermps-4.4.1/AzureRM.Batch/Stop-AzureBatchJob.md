# Stop-AzureBatchJob

## SYNOPSIS
Toplu işi durdurur.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Stop-AzureBatchJob [-Id] <String> [[-TerminateReason] <String>] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Stop-AzureBatchJob** cmdlet 'ı bir Azure toplu işlemini durdurur.
Bu komut işi tamamlandı olarak işaretler.

## ÖRNEKLERDEN

### Örnek 1: toplu işi durdurma
```
PS C:\>Stop-AzureBatchJob -Id "Job-000001" -TerminateReason "No more tasks to run" -BatchContext $Context
```

Bu komut, Iş KIMLIĞI-000001 olan işi durdurur.
Komut, işi durdurmayı seçtiğiniz bir nedeni belirtir.
$Context değişkenine bağlam atamak için Get-AzureRmBatchAccountKeys cmdlet 'ini kullanın.
