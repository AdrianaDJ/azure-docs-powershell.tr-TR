# Set-AzureRmBatchAccount

## SYNOPSIS
Bir toplu Iş hesabını güncelleştirir.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Set-AzureRmBatchAccount [-AccountName] <String> [-Tag] <Hashtable> [-ResourceGroupName <String>]
 [-AutoStorageAccountId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Set-AzureRmBatchAccount** cmdlet 'ı bir Azure toplu hesabını güncelleştirir.
Şu anda bu cmdlet yalnızca etiketleri güncelleştirebilir.

## ÖRNEKLERDEN

### Örnek 1: toplu Iş hesabındaki etiketleri güncelleştirme
```
PS C:\>Set-AzureRmBatchAccount -AccountName "cmdletexample" -Tag @{key0="value0";key1=$null;key2="value2"}
AccountName                  : cmdletexample
Location                     : westus
ResourceGroupName            : CmdletExampleRG
CoreQuota                    : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
                               Name  Value
                               ====  ======
                               key0  value0
                               key1
                               key2  value2
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
```

Bu komut, pfuller adlı hesaptaki etiketleri güncelleştirir.
