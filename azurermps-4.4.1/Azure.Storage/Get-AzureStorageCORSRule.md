# Get-AzureStorageCORSRule

## SYNOPSIS
Depolama hizmeti türü için CORS kurallarını alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureStorageCORSRule [-ServiceType] <StorageServiceType> [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>] [-ConcurrentTaskCount <Int32>]
 [<CommonParameters>]
```

## Tanım
**Get-AzureStorageCORSRule** cmdlet 'ı bir Azure depolama hizmeti türü Için çapraz kaynak PAYLAŞıMı (CORS) kurallarını alır.

## ÖRNEKLERDEN

### Örnek 1: Blob hizmetinin CORS kurallarını alma
```
PS C:\>Get-AzureStorageCORSRule -ServiceType Blob
```

Bu komut, blob hizmet türü için CORS kurallarını alır.
