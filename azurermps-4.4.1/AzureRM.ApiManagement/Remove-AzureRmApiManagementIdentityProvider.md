# Remove-AzureRmApiManagementIdentityProvider

## SYNOPSIS
Var olan kimlik sağlayıcı yapılandırmasını kaldırır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
Var olan kimlik sağlayıcı yapılandırmasını kaldırır.

## ÖRNEKLERDEN

### Facebook kimlik sağlayıcısı ayarlarını, Apımanayönetimi hizmetinden kaldırır
```
Remove-AzureRmApiManagementIdentityProvider -Context $apimContext -Type 'Facebook' -PassThru
```

Facebook Identity Provider yapılandırması ile ilgili yapılandırmayı siler.
