# New-AzureRmDataFactoryHub

## SYNOPSIS
Azure Veri Fabrikası için bir hub oluşturur.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### ByFactoryName (varsayılan)
```
New-AzureRmDataFactoryHub [-Name] <String> [-File] <String> [-Force] [-DataFactoryName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
New-AzureRmDataFactoryHub [-Name] <String> [-File] <String> [-Force] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**New-AzureRmDataFactoryHub** cmdlet 'ı belirtilen Azure Kaynak grubunda ve belirtilen veri fabrikası içinde belirtilen dosya tanımıyla Azure Veri Fabrikası için bir hub oluşturur.
Hub 'ı oluşturduktan sonra, bir grupta bağlantılı hizmetleri depolamak ve yönetmek için kullanabilirsiniz ve merkeze ardışık düzen ekleyebilirsiniz.

## ÖRNEKLERDEN

### Örnek 1: hub oluşturma
```
PS C:\>New-AzureRmDataFactoryHub -ResourceGroupName "ADFResourceGroup" -DataFactoryName "ADFDataFactory" -Name "ContosoDataHub" -File "C:\Hub.json"
```

Bu komut, ADFResourceGroup kaynak grubunda ContosoDataHub adlı bir hub ve ADFDataFactory adlı veri fabrikası oluşturur.
