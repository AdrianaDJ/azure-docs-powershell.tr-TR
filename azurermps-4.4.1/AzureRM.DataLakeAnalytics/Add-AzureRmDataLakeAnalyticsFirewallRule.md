# Add-AzureRmDataLakeAnalyticsFirewallRule

## SYNOPSIS
Bir veri Lake Analytics hesabına güvenlik duvarı kuralı ekler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Add-AzureRmDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Add-Azurermdatalakeanalyzer Ticsfirewallrule** cmdlet 'ı Azure Data Lake Analytics hesabına bir güvenlik duvarı kuralı ekler.

## ÖRNEKLERDEN

### Örnek 1: güvenlik duvarı kuralı ekleme
```
PS C:\>Add-AzureRmDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

Bu komut "ContosoAdlAcct" hesabından "My Firewall Rule" adlı güvenlik duvarı kuralını IP aralığıyla ekler: 127.0.0.1-127.0.0.10
