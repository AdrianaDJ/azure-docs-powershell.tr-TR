# Add-AzureRmDataLakeAnalyticsDataSource

## SYNOPSIS
Data Lake Analytics hesabına veri kaynağı ekler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### Veri Lake Storage hesabı ekleme
```
Add-AzureRmDataLakeAnalyticsDataSource [-Account] <String> [-DataLakeStore] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### BLOB depolama hesabı ekleme
```
Add-AzureRmDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-AccessKey] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Add-Azurermdatalakeanalizleri** , bir Azure Data Lake Analytics hesabına veri kaynağı ekler.

## ÖRNEKLERDEN

### Örnek 1: hesaba veri kaynağı ekleme
```
PS C:\>Add-AzureRmDataLakeAnalyticsDataSource -Account "ContosoAdlA" -DataLakeStore "ContosoAdlS"
```

Bu komut, Data Lake Analytics hesabına bir Data Lake Store veri kaynağı ekler.
