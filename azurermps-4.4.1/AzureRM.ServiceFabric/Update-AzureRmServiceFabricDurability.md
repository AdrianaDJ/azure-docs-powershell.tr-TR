# Update-AzureRmServiceFabricDurability

## SYNOPSIS
Kümedeki bir düğümün dayanıklılık katmanını veya VmSku 'Yu güncelleyin.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Update-AzureRmServiceFabricDurability [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 -DurabilityLevel <DurabilityLevel> [-Sku <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
, Kümenin dayanıklılık veya SKU 'YU güncelleştirmek için **Update-Azurermservicefabricdayanıklılık** 'yi kullanın.

## ÖRNEKLERDEN

### Örnek 1
```
PS c:> Update-AzureRmServiceFabricDurability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -DurabilityLevel Silver -NodeType nt1
```

Bu komut, NodeType 'in dayanıklılık katmanını ' NT1 ' ile gümüş 'ya değiştirir.
