# Get-AzureRmSqlElasticPoolAdvisor

## SYNOPSIS
Bir Azure SQL esnek havuzunun bir veya birden çok danışmanına sahip olur.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Get-AzureRmSqlElasticPoolAdvisor** cmdlet 'ı BIR Azure SQL esnek havuzunun bir veya birden çok Azure SQL esnek havuz danışmanına sahiptir.
