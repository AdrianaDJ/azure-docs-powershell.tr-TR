# Remove-AzureRmSqlElasticPool

## SYNOPSIS
Elastik bir veritabanı havuzunu siler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmSqlElasticPool [-ElasticPoolName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Remove-Azurermsqlelak,** BIR Azure SQL veritabanı esnek havuzunu siler.

## ÖRNEKLERDEN

### Örnek 1: esnek havuz silme
```
PS C:\>Remove-AzureRmSqlElasticPool -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Bu komut, ElasticPool01 adlı bir esnek havuzu siler.
