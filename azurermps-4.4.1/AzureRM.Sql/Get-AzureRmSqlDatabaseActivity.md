# Get-AzureRmSqlDatabaseActivity

## SYNOPSIS
Elastik veritabanlarının taşınması durumunu alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlDatabaseActivity [-ServerName] <String> -ElasticPoolName <String> -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Get-AzureRmSqlDatabaseActivity** cmdlet 'ı, Azure SQL veritabanında esnek bir veritabanı havuzuna taşınan veya olmayan elastik veritabanlarının durumunu alır.

## ÖRNEKLERDEN

### Örnek 1: tüm SQL veritabanı örneklerinin durumunu al
```
PS C:\>Get-AzureRmSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Bu komut, ElasticPool01 adındaki bir esnek havuzda tüm SQL veritabanı örneklerinin işlem durumunu döndürür.
