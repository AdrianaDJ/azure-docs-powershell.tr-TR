# Get-AzureRmSqlServerThreatDetectionPolicy

## SYNOPSIS
Sunucu için tehdit algılama ilkesini alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlServerThreatDetectionPolicy -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Get-AzureRmSqlServerThreatDetectionPolicy** cmdlet 'ı BIR Azure SQL Server 'ın tehdit algılama ilkesini alır.
Bu cmdlet 'i kullanmak için, bu cmdlet 'in ilkeyi aldığı sunucuyu tanımlayacak *Resourcegroupname* ve *ServerName* parametrelerini belirtin.

## ÖRNEKLERDEN

### Örnek 1: sunucunun tehdit algılama ilkesini alma
```
PS C:\>Get-AzureRmSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Bu komut, server01 adlı bir sunucuda tehdit algılama ilkesini alır.
Sunucu, ResourceGroup11 kaynak grubuna atanır.
