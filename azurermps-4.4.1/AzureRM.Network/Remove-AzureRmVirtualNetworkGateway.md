# Remove-AzureRmVirtualNetworkGateway

## SYNOPSIS
Sanal ağ geçidi siler

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmVirtualNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Sanal ağ geçidi, Azure 'daki ağ geçidini temsil eden nesnedir.

**Get-AzureRmVirtualNetworkGateway** cmdlet 'ı, ad ve kaynak grubu adına dayalı olarak Azure 'daki ağ geçidinizin nesnesini döndürür.

## ÖRNEKLERDEN

### 1: sanal ağ geçidini silme
```
Remove-AzureRmVirtualNetworkGateway -Name myGateway -ResourceGroupName myRG
```

Sanal ağ geçidinin, "myRG" kaynak grubundaki "myGateway" adıyla

Not: **Remove-AzureRmVirtualNetworkGatewayConnection** cmdlet 'Ini kullanarak sanal ağ geçitindeki tüm bağlantıları silmelisiniz.
