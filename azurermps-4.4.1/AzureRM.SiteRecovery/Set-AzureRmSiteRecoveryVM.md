# Set-AzureRmSiteRecoveryVM

## SYNOPSIS
Site kurtarma koruma varlığının kurtarma tarafı seçeneklerini ayarlar.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Set-AzureRmSiteRecoveryVM -VirtualMachine <ASRVirtualMachine> [-Name <String>] [-Size <String>]
 [-PrimaryNic <String>] [-RecoveryNetworkId <String>] [-RecoveryNicSubnetName <String>]
 [-RecoveryNicStaticIPAddress <String>] [-NicSelectionType <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Set-AzureRmSiteRecoveryVM** cmdlet 'ı, Azure Site Recovery koruma varlıkları için kurtarma sanal makine boyutu ve kurtarma sanal makine ağı gibi kurtarma tarafı koruma seçeneklerini ayarlar.
