# Get-AzureRmSiteRecoveryVaultSettings

## SYNOPSIS
Geçerli site kurtarma Kasası için ayarlar bilgilerini alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSiteRecoveryVaultSettings [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzureRmSiteRecoveryVaultSettings** cmdlet 'ı geçerli Azure Site Recovery kasası ile ilgili ayarları alır.
