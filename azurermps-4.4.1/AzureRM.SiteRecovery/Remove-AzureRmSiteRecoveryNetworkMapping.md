# Remove-AzureRmSiteRecoveryNetworkMapping

## SYNOPSIS
Geçerli site kurtarma kasasından Ağ eşlemesini kaldırır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureRmSiteRecoveryNetworkMapping -NetworkMapping <ASRNetworkMapping>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Remove-AzureRMSiteRecoveryNetworkMapping** cmdlet 'i, geçerli Azure Site Recovery kasasından bir ağ eşlemesini kaldırır.
