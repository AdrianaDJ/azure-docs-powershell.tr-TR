# Get-AzNetworkVirtualApplianceSku

## SYNOPSIS
Stoktaki kullanılabilir ağ sanal uygulaması SKU 'Larını alın veya listeleyin.

## INDEKI

```
Get-AzNetworkVirtualApplianceSku [-SkuName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
Get-AzNetworkVirtualApplianceSku, Microsoft Azure envanter'ndaki kullanılabilir ağ sanal gereç SKU 'Larını alır veya listeler.
