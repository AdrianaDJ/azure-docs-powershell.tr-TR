# Test-AzDnsAvailability

## SYNOPSIS
Cloudapp.azure.com bölgesindeki bir etki alanı adının kullanıma uygun olup olmadığını denetler.

## INDEKI

```
Test-AzDnsAvailability -DomainNameLabel <String> -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
Cloudapp.azure.com bölgesindeki bir etki alanı adının kullanıma uygun olup olmadığını denetler.

## ÖRNEKLERDEN

### Örnek 1: contoso.westus.cloudapp.azure.com 'in kullanılabilir olup olmadığını denetleyin.
```
Test-AzDnsAvailability -DomainNameLabel contoso -Location westus
```
