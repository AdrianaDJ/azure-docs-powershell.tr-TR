# Remove-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Uygulama ağ geçidinden kimlik doğrulama sertifikasını kaldırır.

## INDEKI

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzApplicationGatewayAuthenticationCertificate** cmdlet 'i, bir Azure uygulama ağ geçidinden kimlik doğrulama sertifikasını kaldırır.

## ÖRNEKLERDEN

### Örnek 1: uygulama ağ geçidinden kimlik doğrulama sertifikası kaldırma
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

İlk komut appGwName adlı uygulama ağ geçidini alır ve sonucu $appgw değişkenine depolar.
İkinci komut cert01 adındaki kimlik doğrulama sertifikasını uygulama ağ geçidinden kaldırır.
Üçüncü komut, uygulama ağ geçidini güncelleştirir.
