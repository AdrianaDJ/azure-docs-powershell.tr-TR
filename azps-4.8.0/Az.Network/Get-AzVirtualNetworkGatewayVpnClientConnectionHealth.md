# Get-AzVirtualNetworkGatewayVpnClientConnectionHealth

## SYNOPSIS 
VPN istemci bağlantısı için bir Azure sanal ağ ağ geçidinin VPN istemci bağlantısı durumu listesini alma

## INDEKI

```
Get-AzVirtualNetworkGatewayVpnClientConnectionHealth -ResourceName <String> -ResourceGroupName <String> -InputObject <PSVirtualNetworkGateway> -ResourceId <ResourceId>
 [-AsJob] [<CommonParameters>]
```

## Tanım
Tüm bağlanılan VPN istemcisi bağlantı durumunu listeleyin. Bu da dahildir: Vpnconnectionıd vpnConnectionDuration vpnConnectionTime Publicıpaddress Privateıpaddress, Kullanıcı adı ço, testransferred Spacketlersbytestransferred maxPacketsPerSecond
