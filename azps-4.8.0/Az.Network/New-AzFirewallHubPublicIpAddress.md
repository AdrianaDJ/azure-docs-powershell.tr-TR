# New-AzFirewallHubPublicIpAddress

## SYNOPSIS
Sanal hub 'da ortak IP 'ler güvenlik duvarına katılır

## INDEKI

```
New-AzFirewallHubPublicIpAddress [-Count <Int32>] [-Addresses <PSAzureFirewallPublicIpAddress[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Sanal hub 'da ortak IP 'ler güvenlik duvarına katılır

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> New-AzFirewallHubPublicIpAddress -Count 2
```

Bu, sanal hub 'a bağlı güvenlik duvarında 2 genel IP oluşturur. Bu, IP adresini arka planda oluşturur. Yeni bir güvenlik duvarı için özel olarak IP adresi sağlayamıyoruz.

### Örnek 2
```powershell
PS C:\> $publicIp1 = New-AzFirewallPublicIpAddress -Address 10.2.3.4
PS C:\> $publicIp2 = New-AzFirewallPublicIpAddress -Address 20.56.37.46
PS C:\> New-AzFirewallHubPublicIpAddress -Count 3 -Addresses $publicIp1, $publicIp2
```

Bu, güvenlik duvarında $publicIp 1 ' i korur $publicIp 2 ' yi güvenlik duvarında var olan yeni genel IP 'yi oluşturur.
