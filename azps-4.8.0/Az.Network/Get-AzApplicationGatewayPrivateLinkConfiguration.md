# Get-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Uygulama ağ geçidinin özel bağlantı yapılandırmasını alır.

## INDEKI

```
Get-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway <PSApplicationGateway> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzApplicationGatewayPrivateLinkConfiguration** cmdlet 'i, bir uygulama ağ geçidinin özel bağlantı yapılandırmasını alır.

## ÖRNEKLERDEN

### Örnek 1: belirtilen özel bağlantı yapılandırmasını alma
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $PrivateLinkConfiguration = Get-AzApplicationGatewayPrivateLinkConfiguration -Name "privateLinkConfig01" -ApplicationGateway $AppGw
```

İlk komut, ResourceGroup01 adındaki kaynak grubundan ApplicationGateway01 adlı bir uygulama ağ geçidini alır ve $AppGw değişkeninde depolar.
İkinci komut, $AppGw privateLinkConfig01 adındaki özel bağlantı yapılandırmasını alır ve bunu $PrivateLinkConfiguration değişkeninde depolar.

### Örnek 2: özel bağlantı yapılandırmasının listesini alma
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $PrivateLinkConfigurations = Get-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw
```

İlk komut, ResourceGroup01 adındaki kaynak grubundan ApplicationGateway01 adlı bir uygulama ağ geçidini alır ve $AppGw değişkeninde depolar.
İkinci komut $AppGw 'dan tüm özel bağlantı yapılandırmalarını alır ve $PrivateLinkConfigurations değişkeninde depolar.
