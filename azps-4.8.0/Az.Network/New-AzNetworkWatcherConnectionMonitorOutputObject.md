# New-AzNetworkWatcherConnectionMonitorOutputObject

## SYNOPSIS
Bağlantı izleme çıkışı hedef nesnesi oluşturma.

## INDEKI

```
New-AzNetworkWatcherConnectionMonitorOutputObject [-OutputType <String>] -WorkspaceResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
New-AzNetworkWatcherConnectionMonitorOutputObject cmdlet 'i Bağlantı İzleyicisi çıkış hedef nesnesi oluşturur.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> New-AzNetworkWatcherConnectionMonitorOutputObject -OutputType "workspace" -ResourcWorkspaceResourceId MyWSResourceId
```

"Çalışma alanı" çalışma alanı ayarları: {"WorkspaceResourceId": "Mywsresourceıd"}
