# Get-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Uygulama ağ geçidinin otomatik ölçeklendirme yapılandırmasını alır.

## INDEKI

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzApplicationGatewayAutoscaleConfiguration** cmdlet 'ı uygulama ağ geçidinin otomatik ölçeklendirme yapılandırmasını alır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

İlk komut uygulama ağ geçidini alır ve $gw değişkeninde depolar.
İkinci komut, uygulama ağ geçidinden otomatik ölçeklendirme yapılandırmasını ayıklar.
