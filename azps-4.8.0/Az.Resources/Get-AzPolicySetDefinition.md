## SYNOPSIS

## Tanım
**Get-AzPolicySetDefinition** cmdlet 'i, ilke kümesi tanımlarının veya ad veya kimlikle tanımlanan belirli bir ilke kümesi tanımı koleksiyonunu alır.

## ÖRNEKLERDEN

### Örnek 1: tüm ilke kümesi tanımlarını alma
```
PS C:\> Get-AzPolicySetDefinition
```

Bu komut, tüm ilke kümesi tanımlarını alır.

### Örnek 2: geçerli abonelikten ilke kümesi tanımını ada göre alma
```
PS C:\> Get-AzPolicySetDefinition -Name 'VMPolicySetDefinition'
```

Bu komut, VMPolicySetDefinition adındaki ilke kümesi tanımını geçerli varsayılan abonelikten alır.

### Örnek 3: ad ile abonelikten ilke kümesi tanımını alma
```
PS C:\> Get-AzPolicySetDefinition -Name 'VMPolicySetDefinition' -subscriptionId '3bf44b72-c631-427a-b8c8-53e2595398ca'
```

Bu komut, VMPolicySetDefinition adlı aboneliğin KIMLIĞI 3bf44b72-c631-427A-b8c8-53e2595398ca olan abonelikten alır.

### Örnek 4: tüm özel ilke kümesi tanımlarını yönetim grubundan alma
```
PS C:\> Get-AzPolicySetDefinition -ManagementGroupName 'Dept42' -Custom
```

Bu komut, Dept42 adlı yönetim grubundan tüm özel ilke kümesi tanımlarını alır.

### Örnek 5: verilen kategoriden ilke kümesi tanımları alma
```
PS C:\> Get-AzPolicySetDefinition | where-object {$_.Properties.metadata.category -eq "Virtual Machine"}
```

Bu komut, "sanal makine" kategorisindeki tüm ilke kümesi tanımlarını alır.
