## Tanım
**Set-Azkaynak** cmdlet 'i mevcut bir Azure kaynağını değiştirir.
Ada göre değiştirmek ve KIMLIK yazmak için kaynak belirtin.

## ÖRNEKLERDEN

### Örnek 1: kaynağı değiştirme
```
PS C:\> $Resource = Get-AzResource -ResourceType Microsoft.Web/sites -ResourceGroupName ResourceGroup11 -ResourceName ContosoSite
PS C:\> $Resource.Properties.Enabled = "False"
PS C:\> $Resource | Set-AzResource -Force
```

İlk komut, Get-AzResource cmdlet 'ini kullanarak ContosoSite adlı kaynağı alır ve $Resource değişkeninde depolar.
İkinci komut $Resource özelliğini değiştirir.
Son komutu $Resource eşleşecek şekilde güncelleştirir.

### Örnek 2: belirli bir kaynak grubundaki tüm kaynakları değiştirme
```
PS C:\> $Resource = Get-AzResource -ResourceGroupName testrg
PS C:\> $Resource | ForEach-Object { $_.Tags.Add("testkey", "testval") }
PS C:\> $Resource | Set-AzResource -Force
```
