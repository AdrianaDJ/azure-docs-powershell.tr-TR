# Get-AzTenantDeploymentOperation

## SYNOPSIS
Kiracı kapsamındaki dağıtım için dağıtım işlemini alma

## INDEKI

### GetByDeploymentName (varsayılan)
```
Get-AzTenantDeploymentOperation -DeploymentName <String> [-OperationId <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByDeploymentObject
```
Get-AzTenantDeploymentOperation -DeploymentObject <PSDeployment> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzTenantDeploymentOperation** cmdlet 'i, belirli bir dağıtımın başarısız olduğu tam işlemlerle ilgili daha fazla bilgi tanımlamanıza yardımcı olması için geçerli kiracı kapsamında dağıtımın bir parçası olan tüm işlemleri listeler.

## ÖRNEKLERDEN

### Örnek 1: dağıtım adı verilen dağıtım işlemlerini alma
```
PS C:\>Get-AzTenantDeploymentOperation -DeploymentName Deploy01
```

Geçerli kiracı kapsamındaki "Deploy01" adlı dağıtım işlemlerini alır.

### Örnek 2: dağıtım alma ve dağıtım işlemlerini alma
```
PS C:\>Get-AzTenantDeployment -Name Deploy01 | Get-AzTenantDeploymentOperation
```

Bu komut, geçerli kiracı kapsamındaki "Deploy01" dağıtımını alır ve dağıtım işlemlerini alır.
