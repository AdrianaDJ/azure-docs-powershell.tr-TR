# Get-AzApiManagementGatewayHostnameConfiguration

## SYNOPSIS
Var olan ağ geçidi için tüm veya belirli ana bilgisayar yapılandırmasını alır.

## INDEKI

### Getbygatewayıd (varsayılan)
```
Get-AzApiManagementGatewayHostnameConfiguration -Context <PsApiManagementContext> -GatewayId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Getbygatewayhostnameıd
```
Get-AzApiManagementGatewayHostnameConfiguration -Context <PsApiManagementContext> -GatewayId <String>
 -GatewayHostnameConfigurationId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azapsananagementgatewayhostnameconfiguration** cmdlet 'i, mevcut ağ geçidi için tüm veya belirli ana bilgisayar yapılandırmasını alır.

## ÖRNEKLERDEN

### Örnek 1: ağ geçidi için tüm ana bilgisayar yapılandırmalarını alma
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGatewayHostnameConfiguration -Context $apimContext  -GatewayId "0123456789"
```

Bu komut, "0123456789" ağ geçidi için tüm ana bilgisayar yapılandırmalarını alır.

### Örnek 2: ağ geçidi için bir ana bilgisayar yapılandırması alma
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGatewayHostnameConfiguration -Context $apimContext -GatewayId "0123456789" -GatewayHostnameConfigurationId "123"
```

Bu komut "0123456789" ağ geçidi için "123" hostname yapılandırmasını alır.
