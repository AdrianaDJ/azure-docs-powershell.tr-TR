# Get-AzApiManagementIdentityProviderClientSecret

## SYNOPSIS
Kimlik sağlayıcısı istemci gizliliğini edinin.

## INDEKI

```
Get-AzApiManagementIdentityProviderClientSecret -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Kimlik sağlayıcısı istemci gizliliğini edinin.

## ÖRNEKLERDEN

### Örnek 1: AAD türü kimlik sağlayıcısının istemci gizliliğini alma
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\> Get-AzApiManagementIdentityProviderClientSecret -Context $apimContext -Type Aad
```

Azure Active Directory 'nin kimlik sağlayıcısı yapılandırmasının istemci gizliliğini alır.
