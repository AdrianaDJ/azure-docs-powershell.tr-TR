# Get-AzIotHubDeployment

## SYNOPSIS
Tüm veya belirli bir IoT Edge dağıtımını listeler.

## INDEKI

### ResourceSet (varsayılan)
```
Get-AzIotHubDeployment [-ResourceGroupName] <String> [-IotHubName] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Inputobjectset
```
Get-AzIotHubDeployment [-InputObject] <PSIotHub> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Resourceıdset
```
Get-AzIotHubDeployment [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
IoT Hub 'ındaki bir IoT uç dağıtımının veya liste IoT uç dağıtımlarının ayrıntılarını edinin.
https://docs.microsoft.com/azure/iot-edge/module-deployment-monitoringDaha fazla bilgi için bkz.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "deploy1"
```

IoT Edge dağıtımının ayrıntılarını edinin.

### Örnek 2
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

IoT Hub 'ındaki tüm IoT Edge dağıtımlarını listeleyin.
