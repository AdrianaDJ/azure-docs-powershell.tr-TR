# Invoke-AzIotHubManualFailover

## SYNOPSIS
IoT Hub 'ı coğrafi olarak eşdeğerli olağanüstü durum kurtarma bölgesine yönelik yük devretme işlemini çağırın.

## INDEKI

### ResourceSet (varsayılan)
```
Invoke-AzIotHubManualFailover [-ResourceGroupName] <String> [-Name] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobjectset
```
Invoke-AzIotHubManualFailover [-InputObject] <PSIotHub> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceıdset
```
Invoke-AzIotHubManualFailover [-ResourceId] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
IoT Hub 'ınızı ikincil konuma tetikler. Bu eylem, çözümünüze zaman ve telemetri kaybını sağlayacaktır. Bu, uzun süren bir işlemdir ve tamamlanması birkaç dakika sürebilir. Lütfen kullanırken dikkatli olun.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Invoke-AzIotHubManualFailover -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

"Myiothub" IoT Hub 'ın yük devretme işlemi başlatılıyor.

### Örnek 2
```powershell
PS C:\> Invoke-AzIotHubManualFailover -ResourceGroupName "myresourcegroup" -Name "myiothub" -AsJob
```

Arka planda "myiothub" IoT Hub 'ın yük devretme işlemi başlatılıyor.
