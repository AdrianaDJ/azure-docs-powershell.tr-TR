# Get-AzSqlInstancePool

## SYNOPSIS
Azure SQL örnek havuzuyla ilgili bilgileri döndürür.

## INDEKI

### ListBySubOrResourceGroupParameterSet (varsayılan)
```
Get-AzSqlInstancePool [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ListByInstancePoolDefaultsParameterSet
```
Get-AzSqlInstancePool -ResourceGroupName <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Getınstancepoolbyınstancepoolresourceıdentifierparameterset
```
Get-AzSqlInstancePool [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azsqlınstancepool** cmdlet 'i, bir veya daha fazla Azure SQL örneği havuzuyla ilgili bilgileri döndürür.
Yalnızca bu örnek havuzunun bilgilerini görmek için örnek havuzun adını belirtin.
