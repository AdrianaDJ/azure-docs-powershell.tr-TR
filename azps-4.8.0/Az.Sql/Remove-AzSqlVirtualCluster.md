# Remove-AzSqlVirtualCluster

## SYNOPSIS
Azure SQL sanal kümesini kaldırır.


## Tanım
**Remove-AzSqlVirtualCluster** cmdlet 'ı BIR Azure SQL sanal kümesini kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:> Remove-AzSqlVirtualCluster -Name VirtualCluster1 -ResourceGroupName ResourceGroup01
```

Bu komut, kaynak grubuna atanmış VirtualCluster1 adındaki sanal kümeyi kaldırır.
