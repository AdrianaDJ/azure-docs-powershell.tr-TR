# Invoke-AzSqlElasticPoolFailover

## SYNOPSIS
Esnek bir havuzun yerine çalışma.

## INDEKI

```
Invoke-AzSqlElasticPoolFailover [-ElasticPoolName] <String> [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
Invoke-AzSqlElasticPoolFailover cmdlet 'i esnek bir havuzda kullanın. Bu cmdlet çalıştırıldıktan sonra, esnek havuzundaki tüm veritabanlarında yük devretme gerçekleşir.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Invoke-AzSqlElasticPoolFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Bu komut, "Server01" adlı sunucudaki "ElasticPool01" adlı esnek havuzun yerine çalışır.  Bu, yerine çalışma, "ElasticPool01" adlı elastik havuzdaki tüm veritabanlarında gerçekleşir.
