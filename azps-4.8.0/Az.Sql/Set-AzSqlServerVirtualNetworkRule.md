# Set-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Azure SQL Server sanal ağ kuralının yapılandırmasını değiştirir.

## INDEKI

```
Set-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Bu komut, Azure SQL Server sanal ağ kuralının yapılandırmasını değiştirir.
Sunucudaki sanal ağ kuralları kümesini denetlemek için bunun yerine ' Add-AzSqlServerVirtualNetworkRule ' ve ' Remove-AzSqlServerVirtualNetworkRule ' kullanın.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> $virtualNetworkRule = Set-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Yeni sanal ağ hakkında bilgi içeren yeni sanal ağ alt ağı kimliği ile mevcut bir sanal ağ kuralını değiştirir
