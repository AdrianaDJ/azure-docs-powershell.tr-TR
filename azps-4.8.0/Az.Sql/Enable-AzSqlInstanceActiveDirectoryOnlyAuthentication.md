# Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication

## SYNOPSIS
Belirli bir SQL yönetilen örneği için Azure AD only kimlik doğrulamasını etkinleştirmektedir.

## INDEKI

### Useresourcegroupandınstancenameparameterset (varsayılan)
```
Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication [-ResourceGroupName] <String> [-InstanceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UseInputObjectParameterSet
```
Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication -InputObject <AzureSqlManagedInstanceModel>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Userresourceıdparameterset
```
Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Enable-Azsqlınstanceactivedirectoryonlyauthentication** cmdlet 'i geçerli abonelikteki bir AzureSQL yönetilen örneği Için Azure Active Directory (Azure AD) kimlik doğrulama gereksinimini etkinleştirir.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\>Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication -ResourceGroupName "ResourceGroup01" -InstanceName "ManagedInstance01"
ResourceGroupName InstanceName        AzureADOnlyAuthentication
----------------- ---------- ----------- -------- -----------
ResourceGroup01   ManagedInstance01   True
```

Bu komut, ResourceGroup01 adlı bir kaynak grubuyla ilişkili ManagedInstance01 adındaki bir AzureSQL yönetilen örneği için Azure Active Directory (Azure AD) kimlik doğrulama gereksinimini etkinleştirmektedir.
