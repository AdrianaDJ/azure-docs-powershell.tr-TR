# Get-AzSqlInstanceDatabase

## SYNOPSIS
Azure SQL yönetilen örnek veritabanı hakkında bilgi döndürür.

## INDEKI

### Getınstancedatabasefrominınputparameters (varsayılan)
```
Get-AzSqlInstanceDatabase [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetInstanceDatabaseFromAzureResourceId
```
Get-AzSqlInstanceDatabase [[-Name] <String>] [-InstanceResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Getınstancedatabasefromınstanceobject
```
Get-AzSqlInstanceDatabase [[-Name] <String>] [-InstanceObject] <AzureSqlManagedInstanceModel>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azsqlınstancedatabase** cmdlet 'ı BIR Azure SQL veritabanı yönetilen örneğinden bir veya birden çok Azure SQL veritabanını alır.
