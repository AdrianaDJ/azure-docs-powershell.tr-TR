# New-AzOperationalInsightsComputerGroup

## SYNOPSIS
Bilgisayar grubu oluşturur.

## INDEKI

```
New-AzOperationalInsightsComputerGroup [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-SavedSearchId] <String> [-DisplayName] <String> [-Category] <String> [-Query] <String> [[-Version] <Int64>]
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**New-Azoperationalınsightscomputergroup** cmdlet 'i bir kaynak grubunda ve konumda bir bilgisayar grubu oluşturur.

## ÖRNEKLERDEN

### Örnek 1

Bilgisayar grubu oluşturur. oluşturulmuş

