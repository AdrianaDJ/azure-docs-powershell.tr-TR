# Set-AzSecurityAssessment

## SYNOPSIS
Kaynakta güvenlik değerlendirme sonucu oluşturma veya güncelleştirme

## INDEKI

### SubscriptionLevelResource (varsayılan)
```
Set-AzSecurityAssessment -Name <String> [-StatusCode <String>] [-StatusCause <String>]
 [-StatusDescription <String>]
 [-AdditionalData <System.Collections.Generic.Dictionary`2[System.String,System.String]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceıdlevelresource
```
Set-AzSecurityAssessment -Name <String> -AssessedResourceId <String> -StatusCode <String>
 [-StatusCause <String>] [-StatusDescription <String>]
 [-AdditionalData <System.Collections.Generic.Dictionary`2[System.String,System.String]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Kaynakta güvenlik değerlendirme sonucu oluşturma veya güncelleştirme, var olan sonucun durumunu değiştirmek veya başka veriler eklemek için kullanılabilir.
yalnızca "Customeryönetilebilir" değerlendirme türleri için ve eşleşen değerlendirme meta verileri oluşturulduktan sonra kullanılabilir.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Set-AzSecurityAssessment -Name 4FB6C0A0-1137-42C7-A1C7-4BD37C91DE8D -StatusCode "Unhealthy"
```

Abonelik sonucunu "4FB6C0A0-1137-4237-A1C7-4BD37C91DE8D" türünün değerlendirmesi için "sağlıksız" olarak işaretler
