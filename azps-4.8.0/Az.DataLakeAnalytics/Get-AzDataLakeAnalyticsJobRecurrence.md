# Get-AzDataLakeAnalyticsJobRecurrence

## SYNOPSIS
Veri Lake Analytics Iş yinelemesi veya yinelenmeleri alır.

## INDEKI

### GetAllInAccount (varsayılan)
```
Get-AzDataLakeAnalyticsJobRecurrence [-Account] <String> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Getbyspecificjobyinelenme
```
Get-AzDataLakeAnalyticsJobRecurrence [-Account] <String> [-RecurrenceId] <Guid>
 [-SubmittedAfter <DateTimeOffset>] [-SubmittedBefore <DateTimeOffset>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azdatalakeçözümlerken** , belirli bir Azure Data Lake Analytics iş yinelemesini veya yinelenme listesini alır.

## ÖRNEKLERDEN

### Örnek 1: belirli bir yinelenme alma
```
PS C:\>Get-AzDataLakeAnalyticsJobRecurrence -Account "contosoadla" -RecurrenceId 83cb7ad2-3523-4b82-b909-d478b0d8aea3
```

Bu komut, ' contosoadla ' hesabında kimliği ' 83cb7ad2-3523-4b82-b909-vseç478b0vseç8aea3 ' olan belirtilen iş yinelemesini alır.

### Örnek 2: hesaptaki tüm yinelenmeleri listesini alma
```
PS C:\>Get-AzDataLakeAnalyticsJobRecurrence -AccountName "contosoadla"
```

Bu komut, "contosoadla" hesabındaki tüm Yinelenmeler listesini alır
