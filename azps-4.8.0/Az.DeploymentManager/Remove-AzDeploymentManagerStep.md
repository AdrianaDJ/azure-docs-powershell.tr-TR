# Remove-AzDeploymentManagerStep

## SYNOPSIS
Adımı siler.

## INDEKI

### Etkileşimli (varsayılan)
```
Remove-AzDeploymentManagerStep [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Içermiyor
```
Remove-AzDeploymentManagerStep [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzDeploymentManagerStep** cmdlet 'i bir adımı siler.
Adımı adına ve kaynak grubu adına göre belirtin. Alternatif olarak, Step nesnesini veya RESOURCEID 'yi de sağlayabilirsiniz.

## ÖRNEKLERDEN

### Örnek 1: adımı kaldırma
```powershell
PS C:\> Remove-AzDeploymentManagerStep -ResourceGroupName ContosoResourceGroup -Name ContosoService1WaitStep
```

Bu komut, ContosoResourceGroup 'ta ContosoService1WaitStep adındaki bir adımı siler.

### Örnek 2: kaynak tanımlayıcısını kullanarak bir adımı kaldırma
```powershell
PS C:\> Remove-AzDeploymentManagerStep -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/steps/ContosoService1WaitStep"
```

Bu komut, ContosoResourceGroup 'ta ContosoService1WaitStep adındaki bir adımı siler.

### Örnek 3: New-AzDeploymentManagerStep tarafından döndürülen bir nesneyi kullanarak adımı kaldırma
### Örnek 1
```powershell
PS C:\> Remove-AzDeploymentManagerStep -InputObject $stepObject
```

 Bu komut, adı ve ResourceGroup 'in sırasıyla $stepObject Name ve ResourceGroupName özellikleriyle eşleşen bir adımını siler.
