# New-AzDiskAccess

## SYNOPSIS
Disk erişim kaynağı oluşturur

## INDEKI

```
New-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Yeni-Azdıskaccess** cmdlet 'ı disk erişim kaynağı oluşturur

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> New-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" -Location "NorthCentralUS"
```

Bu komut, verilen özelliklere sahip bir disk erişimi oluşturur.
