# Remove-AzDiskAccess

## SYNOPSIS
Disk erişimi kaynağını kaldırır.

## INDEKI

### DefaultParameterSet (varsayılan)
```
Remove-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceıdparameterset
```
Remove-AzDiskAccess [-ResourceId] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### Inputobjectparameterset
```
Remove-AzDiskAccess [-InputObject] <PSDiskAccess> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzDiskAccess** cmdlet 'i, disk erişimi kaynağını kaldırır.

## ÖRNEKLERDEN

### Örnek 1: varsayılan parametre kümesini kullanarak disk erişimini kaldırma
```
PS C:\> Remove-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01"
```

Bu komut, "ResourceGroup01" kaynak grubundaki "DiskAccess01" adlı disk erişimini kaldırır

### Örnek 2: kaynak KIMLIĞINI kullanarak disk erişimini kaldırma
```
PS C:\> $myDiskAccess = Get-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01"
PS C:\> Remove-AzDiskAccess -ResourceId $myDiskAccess.id
```

Bu komut, kaynak KIMLIĞIYLE disk erişimini kaldırır

### Örnek 3: giriş nesnesini kullanarak disk erişimini kaldırma
```
PS C:\> $myDiskAccess = Get-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01"
PS C:\> Remove-AzDiskAccess -InputObject $myDiskAccess
```

Bu komut, InputObject 'e göre disk erişimini kaldırır

### Örnek 4: giriş nesnesine göre disk erişimini kaldırma
```
PS C:\> Get-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" | Remove-AzDiskAccess 
```

Bu komut, InputObject 'i yönelterek disk erişimini kaldırır
