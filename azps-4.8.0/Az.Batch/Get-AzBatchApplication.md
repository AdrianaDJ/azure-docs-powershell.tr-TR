# Get-AzBatchApplication

## SYNOPSIS
Belirtilen uygulama hakkında bilgi alır.

## INDEKI

```
Get-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [[-ApplicationName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzBatchApplication** cmdlet 'ı bir Azure toplu hesabındaki bir uygulamayla ilgili bilgileri alır.

## ÖRNEKLERDEN

### Örnek 1: uygulamaları bir toplu Iş hesabında görüntüleme
```
PS C:\>Get-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup"
ApplicationName AllowUpdates DisplayName

------------- ------------ ----------------------------

litware       False        Litware Advanced Reticulator
```

Bu komut, ContosoBatch hesabındaki tüm uygulamaları görüntüler.
