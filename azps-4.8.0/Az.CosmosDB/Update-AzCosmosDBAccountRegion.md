# Update-AzCosmosDBAccountRegion

## SYNOPSIS
CosmosDB hesabının bölgelerini güncelleyin.

## INDEKI

### ByNameParameterSet (varsayılan)
```
Update-AzCosmosDBAccountRegion -ResourceGroupName <String> -Name <String> [-Location <String[]>]
 [-LocationObject <PSLocation[]>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Byresourceıdparameterset
```
Update-AzCosmosDBAccountRegion [-Location <String[]>] [-LocationObject <PSLocation[]>] -ResourceId <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObjectParameterSet
```
Update-AzCosmosDBAccountRegion [-Location <String[]>] [-LocationObject <PSLocation[]>]
 -InputObject <PSDatabaseAccountGetResults> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
CosmosDB hesabının bölgelerini güncelleyin. Konum, PSLocation türünde bir nesne veya yük devretme önceliği tarafından sıralanan konum adı dizeleri olarak sağlanabilir.
LocationObject parametresi, yeni konumlara karşılık gelen yeni LocationObjects 'in eklendiği geçerli konumların (yük devretme öncelik listeleri) listesini bekler.
Location parametresi geçerli konumun (yük devretme önceliğine göre sıralanmış) listesini ve yeni konumları bekler. Lütfen lütfen bölge eklenmesini destekliyoruz. Lütfen konum veya LocationObject sağlayın.
