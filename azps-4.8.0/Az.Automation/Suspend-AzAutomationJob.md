# Suspend-AzAutomationJob

## SYNOPSIS
Otomasyon işini askıya alır.

## INDEKI

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Askıya al-AzAutomationJob** cmdlet 'ı bir Azure Otomasyonu işini askıya alır.
Çalışan bir Otomasyon işi belirtin.
Askıya alınmış bir işi sürdürmek için Resume-AzAutomationJob cmdlet 'ini kullanın.

## ÖRNEKLERDEN

### Örnek 1: işi askıya alma
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Bu komut belirtilen KIMLIĞE sahip işi askıya alır.
