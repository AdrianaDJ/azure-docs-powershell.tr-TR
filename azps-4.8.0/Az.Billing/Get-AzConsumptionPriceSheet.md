# Get-AzConsumptionPriceSheet

## SYNOPSIS
Aboneliğin fiyat sayfalarını edinin.

## INDEKI

```
Get-AzConsumptionPriceSheet [-BillingPeriodName <String>] [-ExpandMeterDetail] [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzConsumptionPriceSheet** cmdlet 'inin, aboneliğin fiyat sayfalarını alır.
