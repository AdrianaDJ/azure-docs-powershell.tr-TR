# Az. Advisor modülü
## Tanım
Azure Danışmanı ile etkileşimde bulunmak için komutlar.

## Az. Advisor cmdlet 'Leri
### [Get-Aztavsiyeleri](Get-AzAdvisorRecommendation.md)
Azure Danışmanı önerilerinin listesini alır.

### [Disable-Azdanışmanlar Veöneri](Disable-AzAdvisorRecommendation.md)
Azure Danışmanı önerisini devre dışı bırakın.

### [Enable-Azdanışmanlar ve öneri](Enable-AzAdvisorRecommendation.md)
Azure Advisor tavsiyeleri etkinleştirilir.

### [Get-Azdanışmanlar Orconfiguration](Get-AzAdvisorConfiguration.md)
Verilen abonelik veya kaynak grubu için Azure Advisor yapılandırmalarını edinin.

### [Set-Azdanışmanlar Orconfiguration](Set-AzAdvisorConfiguration.md)
Azure Danışmanı yapılandırmasını güncelleştirir veya oluşturur.
