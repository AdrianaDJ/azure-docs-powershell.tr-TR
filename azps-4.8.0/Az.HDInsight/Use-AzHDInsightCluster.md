# Use-AzHDInsightCluster

## SYNOPSIS
Invoke-RmAzureHDInsightHiveJob cmdlet 'inde kullanılacak kümeyi seçer.

## INDEKI

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Use-AzHDInsightCluster** cmdlet 'i, Invoke-AzHDInsightHiveJob cmdlet 'Inin, kovan işlerini göndermek Için kullanacağı Azure HDInsight kümesini seçer.

## ÖRNEKLERDEN

### Örnek 1: yığın sorgu gönderimi için küme seçme
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Bu komut, kovan sorgu gönderimi için bir küme seçer.
