# Remove-AzHDInsightPersistedScriptAction

## SYNOPSIS
Bir HDInsight kümesinden kalıcı betik eylemini kaldırır.

## INDEKI

```
Remove-AzHDInsightPersistedScriptAction [-ClusterName] <String> [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Remove-AzHDInsightPersistedScriptAction** cmdlet 'i, belirtilen Azure HDInsight kümesinin kalıcı betik eylemleri listesinden kalıcı bir betik eylemini kaldırır.
Kaldırılan betik, küme ölçeklendiğinde artık yürütülmez.

## ÖRNEKLERDEN

### Örnek 1: bir kümedeki kalıcı betik eylemleri listesinden bir betik eylemini kaldırma
```
PS C:\>Remove-AzHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -Name "Scriptaction"
```

Bu komut ScriptAction adlı betik eylemini belirtilen kümedeki kalıcı betik eylemleri listesinden kaldırır.
