# Get-AzHpcCacheSku

## SYNOPSIS
Abonelikteki tüm STB 'leri alır.

## INDEKI

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzHpcCacheSku** cmdlet 'i, abonelikte bulunan SKU 'ların listesini döndürür.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
