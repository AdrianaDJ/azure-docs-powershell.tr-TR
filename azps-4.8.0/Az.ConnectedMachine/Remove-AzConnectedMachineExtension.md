# Remove-AzConnectedMachineExtension

## SYNOPSIS
Uzantıyı silme işlemi.

## INDEKI

### Sil (varsayılan)
```
Remove-AzConnectedMachineExtension -MachineName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### Deleteviaıdentity
```
Remove-AzConnectedMachineExtension -InputObject <IConnectedMachineIdentity> [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Tanım
Uzantıyı silme işlemi.

## ÖRNEKLERDEN

### Örnek 1: bir makine uzantısını kaldırın.
```powershell
PS C:\> Remove-AzConnectedMachineExtension -MachineName myMachine -ResourceGroupName myRG -Name custom

```

Makinede uzantıyı siler.

### Örnek 2: ardışık düzen aracılığıyla uzantıyı kaldırın
```powershell
PS C:\> Get-AzConnectedMachineExtension -ResourceGroupName contoso-connected-machines -MachineName myMachine | Remove-AzConnectedMachineExtension

```

Belirtilen makinedeki tüm uzantıları kaldırır.
