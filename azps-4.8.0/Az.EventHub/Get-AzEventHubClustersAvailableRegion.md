# Get-AzEventHubClustersAvailableRegion

## SYNOPSIS
Tek bir Eventhub kümesinin ayrıntılarını veya verilen kaynak grubundaki kümelerin listesini alır

## INDEKI

```
Get-AzEventHubClustersAvailableRegion [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Adanmış olarak atanmış olan bölgelerin Get-AzEventHubClustersAvailableRegion cmdlet listesi.
