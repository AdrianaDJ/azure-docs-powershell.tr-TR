## Tanım
**Set-AzSynapseIntegrationRuntime** cmdlet 'i belirli parametrelerle tümleştirme çalışma zamanını güncelleştirir.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' -Description 'New description'
```

Cmdlet, ' test-Selfhost-IR ' adlı tümleştirme çalışma zamanının açıklamasını güncelleştirir.

### Örnek 2
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' `
                                        -SharedIntegrationRuntimeResourceId '/subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir' -Type "SelfHosted"
```

Cmdlet, paylaşılan tümleştirme çalışma zamanını kullanmak için çalışma alanını ekler. Parametre kullanılırken `-SharedIntegrationRuntimeResourceId` de eklenmelidir `-Type` . Çalışma alanına cmdlet 'i çalıştırmadan önce tümleştirme çalışma zamanını kullanma izni verilmesi gerektiğini unutmayın.
