# Export-AzSynapseNotebook

## SYNOPSIS
Notbooks 'yi dışarı aktarır.

## INDEKI

### ExportByName (varsayılan)
```
Export-AzSynapseNotebook -WorkspaceName <String> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByObject
```
Export-AzSynapseNotebook -WorkspaceObject <PSSynapseWorkspace> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByInputObject
```
Export-AzSynapseNotebook -InputObject <PSNotebookResource> -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Export-AzSynapseNotebook** cmdlet 'ı bir Azure SYNAPSE Not defteri (. ipynb) dosyasına aktarır. Not defterinin adı, verilen dosyanın adıdır. Not defterinin adını belirtirseniz, cmdlet bu not defterini dışarı aktarır. Bir ad belirtmezseniz cmdlet, çalışma alanındaki tüm not defterlerini dışarı aktarır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -OutputFolder "C:\Notebook"
```

Çalışma alanı olan tüm not defterlerini çalışma alanı

### Örnek 2
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Çalışma alanı "C:\Not defteri" klasörüne ContosoNotebook adlı tek bir not defterini dışarı aktarır.

### Örnek 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Export-AzSynapseNotebook -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Çalışma alanı 'nda ContosoNotebook adlı tek bir not defterini ardışık düzen aracılığıyla "C:\Not defteri" klasörüne aktarır.

### Örnek 4
```powershell
PS C:\> $notebook = Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook
PS C:\> $notebook | Export-AzSynapseNotebook -OutputFolder "C:\Notebook"
```

Çalışma alanı 'nda ContosoNotebook adlı tek bir not defterini ardışık düzen aracılığıyla "C:\Not defteri" klasörüne aktarır.
