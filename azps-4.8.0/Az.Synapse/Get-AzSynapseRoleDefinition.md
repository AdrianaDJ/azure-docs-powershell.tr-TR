# Get-AzSynapseRoleDefinition

## SYNOPSIS
SYNAPSE Analytics rol tanımını alır.

## INDEKI

### GetByWorkspaceNameAndIdParameterSet (varsayılan)
```
Get-AzSynapseRoleDefinition -WorkspaceName <String> [-Id <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByWorkspaceNameAndNameParameterSet
```
Get-AzSynapseRoleDefinition -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Getbyıo Nesnetandidparameterset
```
Get-AzSynapseRoleDefinition -WorkspaceObject <PSSynapseWorkspace> -Id <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Getbyıo Nesnetandnameparameterset
```
Get-AzSynapseRoleDefinition -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzSynapseRoleDefinition** cmdlet 'ı bir Azure SYNAPSE Analytics rol tanımını alır.
Rol adı veya rol kimliği belirtmezseniz, bu cmdlet tüm rol tanımlarını alır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzSynapseRoleDefinition -WorkspaceName ContosoWorkspace
```

Bu komut, çalışma alanı altındaki tüm rol tanımlarını alır.

### Örnek 2
```powershell
PS C:\> Get-AzSynapseRoleDefinition -WorkspaceName ContosoWorkspace -Name ContosoRole
```

Bu komut, ad alanı olan ContosoWorkspace 'in altındaki rol tanımını ContosoRole.

### Örnek 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseRoleDefinition
```

Bu komut, ardışık düzen aracılığıyla tüm rol tanımlarını alır.
