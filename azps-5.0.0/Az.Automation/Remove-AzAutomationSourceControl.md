# Remove-AzAutomationSourceControl

## SYNOPSIS
Azure Otomasyonu kaynak denetimini kaldırır.

## INDEKI

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
Remove-AzAutomationSourceControl cmdlet 'i Azure Otomasyonu 'ndan kaynak denetimini kaldırır.

## ÖRNEKLERDEN

### Örnek 1
Bu komut, devAccount adlı hesapta VSTSNative adlı Otomasyon kaynak denetimini kaldırır.
Bu komut *Force* parametresini belirtir. Bu nedenle sizden onay istemez.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
