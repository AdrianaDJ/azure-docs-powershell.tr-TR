# Set-AzOperationalInsightsIntelligencePack

## SYNOPSIS
Belirtilen yönetim bilgileri paketini etkinleştirilir veya devre dışı bırakır.

## INDEKI

```
Set-AzOperationalInsightsIntelligencePack [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-IntelligencePackName] <String> [-Enabled] <Boolean> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Set-AzOperationalInsightsIntelligencePack** cmdlet 'ı $true *etkinleştirilmişse* , belirtilen yönetim bilgileri paketi etkinleştirilir ve $false *etkinleştirilirse* bu özelliği devre dışı bırakır.

## ÖRNEKLERDEN

### Örnek 1: yönetim bilgileri paketlerini ayarlama
```
PS C:\>Set-AzOperationalInsightsIntelligencePack -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace" -IntelligencePackName "ContosoWorkspace" -Enabled $True
```

Bu komut belirtilen zekası paketini etkinleştirilir.
