# New-AzNetAppFilesAccount

## SYNOPSIS
Yeni bir Azure NetApp dosyaları (ANF) oluşturur.

## INDEKI

```
New-AzNetAppFilesAccount -ResourceGroupName <String> -Location <String> -Name <String>
 [-ActiveDirectory <PSNetAppFilesActiveDirectory[]>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**New-AzNetAppFilesAccount** cmdlet 'ı BIR ANF hesabı oluşturur.

## ÖRNEKLERDEN

### Örnek 1: ANF hesabı oluşturma
```
PS C:\>New-AzNetAppFilesAccount -ResourceGroupName "MyRG" -Name "MyAnfAccount" -l "westus2"

Output:

Location          : westus2
Id                : /subscriptions/mySubs/resourceGroups/MyRG/providers/Microsoft.NetApp/netAppAccounts/MyAnfAccount
Name              : MyAnfAccount
Type              : Microsoft.NetApp/netAppAccounts
Tags              :
ProvisioningState : Succeeded
```

Bu komut, "MyAnfAccount" yeni ANF hesabını oluşturur.
