# Az. Stackhcı modülü
## Tanım
Microsoft Azure PowerShell: Stackhcı cmdlet 'leri

## Az. StackHCI cmdlet 'Leri
### [Yazmaç-AzStackHCI](Register-AzStackHCI.md)
Register-AzStackHCI, şirket içi kümeyi temsil eden bir Microsoft. AzureStackHCI bulut kaynağı oluşturur ve şirket içi kümeyi Azure ile kaydeder.

### [Unregister-Azstackhcı](Unregister-AzStackHCI.md)
Unregister-AzStackHCI, şirket içi kümeyi temsil eden Microsoft. AzureStackHCI bulut kaynağını silip şirket içi kümenin Azure ile kaydını siler.
