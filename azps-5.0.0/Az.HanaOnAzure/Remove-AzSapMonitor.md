# Remove-AzSapMonitor

## SYNOPSIS
Belirtilen abonelikle, kaynak grubuyla ve monitör adıyla bir SAP izleyicisini siler.

## INDEKI

### Sil (varsayılan)
```
Remove-AzSapMonitor -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Deleteviaıdentity
```
Remove-AzSapMonitor -InputObject <IHanaOnAzureIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Tanım
Belirtilen abonelikle, kaynak grubuyla ve monitör adıyla bir SAP izleyicisini siler.

## ÖRNEKLERDEN

### Örnek 1: ada göre SAP izleyicisini kaldırma
```powershell
PS C:\> Remove-AzSapMonitor -ResourceGroupName nancyc-hn1 -Name ps-sapmonitor-t02

```

Bu komut SAP monitörünü ada göre kaldırır.

### Örnek 2: nesne bazında SAP monitörünü kaldırma
```powershell
PS C:\> $sap = Get-AzSapMonitor -ResourceGroupName nancyc-hn1 -Name ps-sapmonitor-t01
PS C:\> Remove-AzSapMonitor -InputObject $sap

```

Bu komut, SAP monitörünü nesneye göre kaldırır.
