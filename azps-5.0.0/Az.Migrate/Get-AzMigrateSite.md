# Get-AzMigrateSite

## SYNOPSIS
Site alma yöntemi.

## INDEKI

```
Get-AzMigrateSite -Name <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## Tanım
Site alma yöntemi.
