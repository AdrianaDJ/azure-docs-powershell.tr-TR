# Save-AzResourceGroupDeploymentTemplate

## SYNOPSIS
Bir dosyaya kaynak grubu dağıtım şablonunu kaydeder.

## INDEKI

```
Save-AzResourceGroupDeploymentTemplate -ResourceGroupName <String> -DeploymentName <String> [-Path <String>]
 [-Force] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Save-AzResourceGroupDeploymentTemplate** cmdlet 'ı bir JSON dosyasına kaynak grubu dağıtım şablonunu kaydeder.

## ÖRNEKLERDEN

### Örnek 1: Dağıtım şablonunu kaydetme
```powershell
PS C:\>Save-AzResourceGroupDeploymentTemplate -DeploymentName "TestDeployment" -ResourceGroupName "TestGroup"
```

Bu komut TestDeployment 'den Dağıtım şablonunu alır ve geçerli dizine JSON dosyası olarak kaydeder.

### Örnek 2

Bir dosyaya kaynak grubu dağıtım şablonunu kaydeder. oluşturulmuş
