# Export-AzTemplateSpec

## SYNOPSIS
Yerel FileSystem 'a şablon belirtimi dışarı aktarır

## INDEKI

### ExportByNameParameterSet (varsayılan)
```
Export-AzTemplateSpec [-ResourceGroupName] <String> [-Name] <String> -Version <String> -OutputFolder <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Exportbyıdparameterset
```
Export-AzTemplateSpec [-ResourceId] <String> -Version <String> -OutputFolder <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Belirli bir şablon spec sürümünü yerel FileSystem 'a aktarır.

## ÖRNEKLERDEN

### Örnek 1: ada göre dışarı aktarma
```powershell
PS C:\> Export-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'MyTemplateSpec' -Version 'v1.0' -OutputFolder 'v1'
```

' MyTemplateSpec ' adlı şablon belirtiminin ' v 1.0 ' sürümünü, ' myRG ' kaynak grubunun içinde "v1" yerel çıkış klasörüne dışarı aktarır.

### Örnek 2: kaynak kimliğiyle dışarı aktarma
```powershell
PS C:\> Export-AzTemplateSpec -ResourceId '/subscriptions/{subId}/resourceGroups/myRG/providers/Microsoft.Resources/templateSpecs/MyTemplateSpec' -Version 'v1.0' -OutputFolder 'v1'
```

Abonelik alt kimliğindeki ' MyTemplateSpec ' adlı şablon belirtiminin ' v 1.0 ' sürümünü \{ \} "v1" yerel çıkış klasörüne dışarı aktarır.
