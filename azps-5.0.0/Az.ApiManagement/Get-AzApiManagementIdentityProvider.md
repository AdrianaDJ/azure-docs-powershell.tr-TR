# Get-AzApiManagementIdentityProvider

## SYNOPSIS
Kimlik sağlayıcısı yapılandırma ayrıntılarını edinin.

## INDEKI

### Aldentityproviders (varsayılan)
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdentityProvider
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Kimlik sağlayıcısı yapılandırma ayrıntılarını edinin.
ClientSecret, sonuç ayrıntılarına eklenmeyecek. İstemci gizliliğini almak için **Get-Azapsananagementidentityproviderclientsecret** seçeneğini kullanın.

## ÖRNEKLERDEN

### Örnek 1: tüm kimlik sağlayıcılarını alma

```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementIdentityProvider -Context $apimContext
```

Hizmette tüm kimlik sağlayıcısı yapılandırmasını edinin.

### Örnek 2: AAD türü kimlik sağlayıcısını edinme
