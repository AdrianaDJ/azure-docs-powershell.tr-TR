# Get-AzApiManagementApiRevision

## SYNOPSIS
API 'nin tüm API düzeltmelerinin ayrıntılarını alır

## INDEKI

```
Get-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azapsananagementapirevision** cmdlet 'inin tüm düzeltmelerinin ayrıntılarını alır

## ÖRNEKLERDEN

### Örnek 1: API 'nin tüm API düzeltmelerini alma
```powershell
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementApiRevision -Context $ApiMgmtContext -ApiId "5adf6fbf0faadf3ad8558065"
```
