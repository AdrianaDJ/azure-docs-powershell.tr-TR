# Get-AzApiManagementTenantAccess

## SYNOPSIS
Kiracının erişim yapılandırmasını alır.

## INDEKI

```
Get-AzApiManagementTenantAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Get-AzApiManagementTenantAccess** cmdlet 'i kiracının kiracı erişim yapılandırmasını alır.
Anahtarlar sonuç ayrıntılarına dahil olmaz. İstemci gizliliğini almak için **Get-Azapsananagementtenantaccesssecret** seçeneğini kullanın.

## ÖRNEKLERDEN

### Örnek 1: kiracı erişimi yapılandırmasını alma
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccess -Context $apimContext
```

Bu komut, belirtilen bağlam için kiracı erişim yapılandırmasını alır.
