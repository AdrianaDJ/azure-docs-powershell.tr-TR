# New-AzApiManagementResourceLocationObject

## SYNOPSIS
Yeni bir kaynak konumu sözleşmesi oluşturun (ağ geçitlerinde kullanılır).

## INDEKI

```
New-AzApiManagementResourceLocationObject -Name <String> [-City <String>] [-District <String>]
 [-CountryOrRegion <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Yeni-Azapsananagementresourseocationobject** cmdlet 'i, yeni bir kaynak konumu Sözleşmesi (ağ geçitlerinde kullanılır) oluşturur.

## ÖRNEKLERDEN

### Örnek 1: kaynak konumu sözleşmesi oluşturma
```powershell
PS C:\>$location = New-AzApiManagementResourceLocationObject -Name "n1" -City "c1" -District "d1" -CountryOrRegion "r1"
```

Bu komut bir kaynak konumu oluşturur.
