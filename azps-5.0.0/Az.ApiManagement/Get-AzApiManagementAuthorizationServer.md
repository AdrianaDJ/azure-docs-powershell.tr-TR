# Get-AzApiManagementAuthorizationServer

## SYNOPSIS
Bir API yönetim yetkilendirme sunucusu alır.

## INDEKI

### ContextParameterSet (varsayılan)
```
Get-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Resourceıdparameterset
```
Get-AzApiManagementAuthorizationServer [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azapsananagementauthorizationserver** cmdlet 'ı tüm Azure API yönetim yetkilendirme sunucularını veya belirtilen yetkilendirme sunucusunu alır.
ClientSecret, sonuç ayrıntılarına eklenmeyecek. İstemci gizliliğini almak için **Get-Azapsananagementauthorizationserverclientsecret** kullanın.

## ÖRNEKLERDEN

### Örnek 1: tüm yetkilendirme sunucularını alma
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServer -Context $ApiMgmtContext
```

Bu komut tüm API yönetim yetkilendirme sunucularını alır.

### Örnek 2: belirtilen yetkilendirme sunucusunu alma
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "0123456789"
```

Bu komut belirtilen yetkilendirme sunucusunu alır.
