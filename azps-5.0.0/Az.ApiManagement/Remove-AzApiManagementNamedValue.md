# Remove-AzApiManagementNamedValue

## SYNOPSIS
Bir API Yönetimi adlandırılmış değeri kaldırır.

## INDEKI

```
Remove-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azapsananagementnamedvalue** cmdlet 'i, BIR Azure API Yönetimi **adlandırılmış değerini** kaldırır.

## ÖRNEKLERDEN

### Örnek 1: adlandırılmış değeri kaldırma
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -PassThru
```

Bu komut, Property11 KIMLIĞINE sahip olan adlandırılmış değeri kaldırır.
