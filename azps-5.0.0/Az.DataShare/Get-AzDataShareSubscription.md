# Get-AzDataShareSubscription

## SYNOPSIS
Veri paylaşımı hesabında paylaşım aboneliği hakkında bilgi alır.

## INDEKI

### ByFieldsParameterSet (varsayılan)
```
Get-AzDataShareSubscription -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Byresourceıdparameterset
```
Get-AzDataShareSubscription -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Get-AzDataShareSubscription** cmdlet 'i, veri paylaşımı hesabında paylaşma aboneliği hakkında bilgi sağlar. Paylaşım aboneliği adı sağlanmışsa cmdlet, belirli bir paylaşım aboneliği hakkında bilgi verir. Aksi halde cmdlet, veri paylaşımı hesabında bir abonelik paylaşımı listesi sağlar.

```
PS C:\> Get-AzDataShareSubscription -ResourceGroupName "ADS" -AccountName "WikiAds" -Name "AdsShareSubscription"
```

Bu komut, veri paylaşımı hesap WikiAds 'de abonelik
