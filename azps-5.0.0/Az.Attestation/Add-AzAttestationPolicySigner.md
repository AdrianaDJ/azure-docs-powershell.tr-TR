# Add-AzAttestationPolicySigner

## SYNOPSIS
Azure kanıt içinde kiracı için güvenilen bir ilke imzalayanı ekler.

## INDEKI

### NameParameterSet
```
Add-AzAttestationPolicySigner [-Name] <String> [-ResourceGroupName] <String> -Signer <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceıdparameterset
```
Add-AzAttestationPolicySigner [-ResourceId] <String> -Signer <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Add-AzAttestationPolicySigner cmdlet 'i Azure kanıtlama 'ndaki bir kiracı için güvenilen bir ilke imzalayıcısı ekler.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $trustedSigner = Get-Content -Path .\trusted.signer.txt
PS C:\> Add-AzAttestationPolicySigner -Name pshtest -ResourceGroupName psh-test-rg -Signer $trustedSigner
```

*Pshtest* adlı Atteestation sağlayıcısı için güvenilen bir imzalayan ekleyin.
