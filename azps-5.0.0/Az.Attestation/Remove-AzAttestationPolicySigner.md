# Remove-AzAttestationPolicySigner

## SYNOPSIS
Azure kanıtlama 'ndaki bir kiracı için güvenilen ilke imzacısını kaldırır.

## INDEKI

### NameParameterSet
```
Remove-AzAttestationPolicySigner [-Name] <String> [-ResourceGroupName] <String> -Signer <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceıdparameterset
```
Remove-AzAttestationPolicySigner [-ResourceId] <String> -Signer <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Remove-AzAttestationPolicySigner cmdlet 'i Azure kanıtlama 'ndaki bir kiracı için güvenilen ilke imzacısını kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $trustedSigner = Get-Content -Path .\trusted.signer.txt
PS C:\> Remove-AzAttestationPolicySigner -Name pshtest -ResourceGroupName psh-test-rg -Signer $trustedSigner
```

*Pshtest* adındaki kanıtlama sağlayıcısı için güvenilen İmzalayanın kaldırma.
