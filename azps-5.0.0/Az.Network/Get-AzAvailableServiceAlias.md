# Get-AzAvailableServiceAlias

## SYNOPSIS
Bölgede kullanılabilir hizmet diğer adlarını alma.

## INDEKI

```
Get-AzAvailableServiceAlias -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzAvailableServiceAlias** cmdlet 'i sağlanan konumdaki bir alt ağın tüm kullanılabilir hizmet diğer adlarını almanıza olanak tanır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzAvailableServiceAlias -Location "westus"
```
