# New-AzApplicationGatewayRewriteRuleUrlConfiguration

## SYNOPSIS
Uygulama ağ geçidi için yeniden yazma kuralı URL yapılandırması oluşturur.

## INDEKI

```
New-AzApplicationGatewayRewriteRuleUrlConfiguration [-ModifiedPath <String>] [-ModifiedQueryString <String>] [-Reroute]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**AzApplicationGatewayRewriteRuleUrlConfiguration** cmdlet 'ı bir Azure uygulama ağ geçidi için yeniden yazma kuralı URL yapılandırması oluşturur.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $urlConfiguration = New-AzApplicationGatewayRewriteRuleUrlConfiguration -ModifiedPath "/abc" -ModifiedQueryString "x=y&a=b"
```

Bu komut yeniden yazma kuralı URL yapılandırması oluşturur ve sonucu $urlConfiguration adlı değişkende depolar.

Var olan herhangi bir URL yapılandırmasını güncelleştirmek istiyorsanız, bunu yeni bir Urlyapılandırması oluşturarak ve yeni Urlyapılandırmasını yeniden yazma kuralı eylem kümesinin UrlConfiguration özelliğine atayarak yapabilirsiniz.
