# Remove-AzExpressRouteCircuit

## SYNOPSIS
ExpressRoute devresini kaldırır.

## INDEKI

```
Remove-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azexpressroutedevresi** cmdlet 'ı bir ExpressRoute devresini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: ExpressRoute devresini silme
```
Remove-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### Örnek 2: ardışık düzeni kullanarak ExpressRoute devresini silme
```
Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzExpressRouteCircuit
```
