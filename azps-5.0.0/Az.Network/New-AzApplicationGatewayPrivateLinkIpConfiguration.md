# New-AzApplicationGatewayPrivateLinkIpConfiguration

## SYNOPSIS
PrivateLink yapılandırması ile ilişkilendirilecek bir IP yapılandırması oluşturur

## INDEKI

```
New-AzApplicationGatewayPrivateLinkIpConfiguration -Name <String> -Subnet <PSSubnet>
 [-PrivateIpAddressVersion <String>] [-PrivateIpAddress <String>] [-Primary]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Yeni-Azapplicationgatewayprivatelinkıp** 'si cmdlet 'i, bir Azure uygulama ağ geçidi Için Privatelink yapılandırmasıyla Ilişkilendirilecek bir IP yapılandırması oluşturur.

## ÖRNEKLERDEN

### Örnek 1: PrivateLink IP yapılandırması
```powershell
PS C:\> $PrivateLinkIpConfiguration = New-AzApplicationGatewayPrivateLinkIpConfiguration -Name "ipConfig01" -Subnet $subnet -Primary
```

Bu komut, ' ipConfig01 ' adlı bir PrivateLink IP yapılandırması oluşturur sonucu $PrivateLinkIpConfiguration adlı değişkende depolar.
