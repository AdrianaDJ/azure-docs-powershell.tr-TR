# Remove-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Bir uygulama ağ geçidinden privateLink yapılandırmasını kaldırır.

## INDEKI

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Remove-AzApplicationGatewayPrivateLinkConfiguration** cmdlet 'ı bir Azure uygulama ağ geçidinden Privatelink yapılandırmasını kaldırır.

## ÖRNEKLERDEN

### Örnek 1: bir uygulama ağ geçidi PrivateLink yapılandırmasını kaldırma
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

İlk komut bir uygulama ağ geçidi alır ve $AppGw değişkeninde depolar.
İkinci komut, $AppGw depolanan uygulama ağ geçidinden privateLinkConfig01 adlı privateLink yapılandırmasını kaldırır.
