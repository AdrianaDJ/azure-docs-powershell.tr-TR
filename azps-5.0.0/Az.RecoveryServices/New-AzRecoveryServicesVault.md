# New-AzRecoveryServicesVault

## SYNOPSIS
Yeni bir kurtarma hizmetleri Kasası oluşturur.

## INDEKI

```
New-AzRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Yeni-Azrecoveryserviceskasa** cmdlet 'i yeni bir kurtarma hizmetleri Kasası oluşturur.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> New-AzRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Kaynak grubunda ve verilen konumda kurtarma hizmeti Kasası oluşturun.
