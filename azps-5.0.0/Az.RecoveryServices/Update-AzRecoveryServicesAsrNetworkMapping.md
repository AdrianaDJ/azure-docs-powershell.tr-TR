# Update-AzRecoveryServicesAsrNetworkMapping

## SYNOPSIS
Belirtilen Azure Site Recovery Ağ eşlemesini güncelleştirir.

## INDEKI

### ByNetworkObject (varsayılan)
```
Update-AzRecoveryServicesAsrNetworkMapping -InputObject <ASRNetworkMapping> -RecoveryNetwork <ASRNetwork>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Byıd
```
Update-AzRecoveryServicesAsrNetworkMapping -InputObject <ASRNetworkMapping> -RecoveryAzureNetworkId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Update-AzRecoveryServicesAsrNetworkMapping** cmdlet 'i, belirtilen Azure Site Recovery Ağ eşlemesini güncelleştirir.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrNetworkMapping -Mapping $NetworkMapping -RecoveryNetwork $RecoveryNetwork
```

Belirtilen parametreleri kullanarak Ağ eşlemesini Güncelleştir işlemini başlatır ve işlemi izlemek için kullanılan ASR işini döndürür.
