# Remove-AzMediaService

## SYNOPSIS
Medya hizmetini kaldırır.

## INDEKI

```
Remove-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzMediaService** cmdlet 'i bir medya hizmetini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: medya hizmetini kaldırma
```
PS C:\>Remove-AzMediaService -ResourceGroupName "ResourceGroup001" -AccountName "MediaService0011"
```

Bu komut, ResourceGroup001 adındaki kaynak grubundaki MediaService0011 adındaki medya hizmetini kaldırır.
