# Get-AzHpcCacheUsageModel

## SYNOPSIS
NFS için tüm Usagemodel depolama hedefini alır.

## INDEKI

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzHpcCacheUsageModel** CMDLET 'i NFS için kullanım modellerinin bir listesini döndürür.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```
