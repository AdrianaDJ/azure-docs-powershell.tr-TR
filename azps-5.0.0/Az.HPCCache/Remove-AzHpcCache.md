# Remove-AzHpcCache

## SYNOPSIS
HPC önbelleğini kaldırır.

## INDEKI

```
Remove-AzHpcCache -ResourceGroupName <String> -Name <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzHpcCache** cmdlet 'ı BIR Azure HPC önbelleğini kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Remove-AzHpcCache -ResourceGroupName testRG -CacheName testCache
```
