# Resume-AzAnalysisServicesServer

## SYNOPSIS
Analysis Services sunucusu örneğini sürdürür

## INDEKI

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Resume-AzAnalysisServicesServer cmdlet 'i Analysis Services sunucusunun bir örneğini sürdürür

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Bu komut, resourcegroup testgroup 'da TestServer adlı duraklatılmış bir sunucuyu sürdürecek
