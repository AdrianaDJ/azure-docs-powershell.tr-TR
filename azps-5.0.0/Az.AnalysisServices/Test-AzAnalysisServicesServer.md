# Test-AzAnalysisServicesServer

## SYNOPSIS
Analysis Services sunucusu örneğinin varlığını sınar

## INDEKI

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Test-AzAnalysisServicesServer cmdlet 'i Analysis Services sunucusu örneğinin varlığını sınar

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Resourcegroup testgroup 'ta TestServer adlı bir sunucu varsa bu komut test edecektir
