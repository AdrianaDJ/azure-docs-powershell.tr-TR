## Tanım
Genel/Azure kayıt defterinden bir Azure Kapsayıcı kayıt defterine resim aktarma.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Import-AzContainerRegistryImage -SourceImage library/busybox:latest -ResourceGroupName $resourceGroupName -RegistryName $RegistryName -SourceRegistryUri docker.io -TargetTag busybox:latest
```

BUSYBOX 'ı ACR 'e aktar. Notu 
* "kitaplık/" kaynağının kaynak resimden önce eklenmesi gerekir. "BusyBox: latest" => "kitaplık/BUSYBOX: latest"
* Kaynak kayıt defteri genel kullanıma sunulursa gerekli kimlik bilgileri

### Örnek 2
```powershell
PS C:\> Import-AzContainerRegistryImage -SourceImage $SourceRegistry.azurecr.io/busybox:latest -ResourceGroupName $resourceGroupName -RegistryName $RegistryName -SourceRegistryResourceId $SourceACRID -TargetTag busybox:latest
```

Kaynak ACR 'den hedef ACR 'ye BUSYBOX alın. Notu 
* Kaynak kayıt defteri için Yönetici kullanıcısı devre dışı bırakılmışsa kimlik bilgileri gerekir.
