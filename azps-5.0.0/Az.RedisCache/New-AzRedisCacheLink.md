# New-AzRedisCacheLink

## SYNOPSIS
İki Redis önbellekler arasında coğrafi çoğaltma bağlantısı oluşturun.

## INDEKI

```
New-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
İki Redis önbellekler arasında coğrafi çoğaltma bağlantısı oluşturun.

## ÖRNEKLERDEN

### Örnek 1: iki önbellek arasında bağlantı oluşturma
```
PS C:\>New-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"

        PrimaryServerName   : mycache1
        SecondaryServerName : mycache2
        ProvisioningState   : Creating
```

Bu komut, Redis Cache myCache1 ve mycache2 arasında coğrafi çoğaltma bağlantısı oluşturur.
