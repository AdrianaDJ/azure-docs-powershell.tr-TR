# Remove-AzStreamAnalyticsJob

## SYNOPSIS
Akış Analizi işini kaldırır.

## INDEKI

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzStreamAnalyticsJob** cmdlet 'i Azure 'da belirli bir Stream Analytics işini eşzamansız olarak siler.

## ÖRNEKLERDEN

### Örnek 1: işi kaldırma
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Bu komut, StreamingJob öğesini kaldırır.
