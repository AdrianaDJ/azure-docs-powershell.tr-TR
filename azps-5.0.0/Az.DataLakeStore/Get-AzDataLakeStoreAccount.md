# Get-AzDataLakeStoreAccount

## SYNOPSIS
Data Lake Store hesabının ayrıntılarını alır.

## INDEKI

### Getallinaboneliği (varsayılan)
```
Get-AzDataLakeStoreAccount [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceGroup
```
Get-AzDataLakeStoreAccount [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetBySpecificAccount
```
Get-AzDataLakeStoreAccount [[-ResourceGroupName] <String>] [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzDataLakeStoreAccount** cmdlet 'ı Data Lake Store hesabının ayrıntılarını alır.

## ÖRNEKLERDEN

### Örnek 1: Data Lake Store hesabını edinme
```
PS C:\>Get-AzDataLakeStoreAccount -Name "ContosoADL"
```

Bu komut, ContosoADL adlı hesabı alır.
