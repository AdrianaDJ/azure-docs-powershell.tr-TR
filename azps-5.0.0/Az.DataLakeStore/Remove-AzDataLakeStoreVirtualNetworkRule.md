# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Belirtilen veri Lake Store 'da belirtilen sanal ağ kuralını kaldırır.

## INDEKI

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzDataLakeStoreVirtualNetworkRule** cmdlet 'ı belirtilen Data Lake Store 'da belirtilen sanal ağ kuralını kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

"MyVNET" sanal ağ kuralını "DLS" hesabından kaldırır
