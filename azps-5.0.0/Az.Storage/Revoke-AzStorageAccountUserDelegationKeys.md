# Revoke-AzStorageAccountUserDelegationKeys

## SYNOPSIS
Depolama hesabının tüm kullanıcı temsilci anahtarlarını iptal edin.

## INDEKI

### AccountName (varsayılan)
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountObject
```
Revoke-AzStorageAccountUserDelegationKeys -InputObject <PSStorageAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Accountresourceıd
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Revoke-AzStorageAccountUserDelegationKeys** cmdlet 'i, depolama hesabındaki tüm kimlik SAS belirteci de iptal edilecek.

## ÖRNEKLERDEN

### Örnek 1: depolama hesabının tüm kullanıcı temsilci anahtarlarını Iptal etme
```powershell
PS C:\>Revoke-AzStorageAccountUserDelegationKeys -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount"
```

Bu örnekte, bir depolama hesabının tüm kullanıcı temsilci anahtarının iptal edildiği, kullanıcı temsilcisi anahtarlarından oluşturulan tüm kimlik SAS belirteçleri de iptal edilecek.
