# Disable-AzStorageStaticWebsite

## SYNOPSIS
Azure depolama hesabı için statik Web sitesini devre dışı bırakın.

## INDEKI

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Disable-AzStorageStaticWebsite** cmdlet 'ı, Azure depolama hesabı için statik Web sitesini devre dışı bırakır.

## ÖRNEKLERDEN

### Örnek 1: Azure depolama hesabı için statik Web sitesini devre dışı bırakma
```
C:\PS>Disable-AzStorageStaticWebsite
```

Bu komut, Azure depolama hesabı için statik Web sitesini devre dışı bırakır.
