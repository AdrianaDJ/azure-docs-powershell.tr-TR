# Get-AzSynapseTriggerSubscriptionStatus

## SYNOPSIS
Olay tetikleyicisi için aboneliğin durumunu belirtilen dış hizmet olaylarına alın.

## INDEKI

### GetByName (varsayılan)
```
Get-AzSynapseTriggerSubscriptionStatus -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseTriggerSubscriptionStatus -WorkspaceObject <PSSynapseWorkspace> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByInputObject
```
Get-AzSynapseTriggerSubscriptionStatus -InputObject <PSTriggerResource>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzSynapseTriggerSubscriptionStatus** cmdlet 'i, olay tetikleyicisi için belirtilen dış hizmet olaylarına aboneliğin durumunu alır. Tetik, iade durumu "Enabled" olana kadar başlatılamıyor.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzSynapseTriggerSubscriptionStatus -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Bu komut, ContosoTrigger adındaki subscribtion tetikleyicisi 'nin dış hizmet olaylarına durumunu alır.

### Örnek 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseTriggerSubscriptionStatus -Name ContosoTrigger
```

Bu komut, "subscribtion" tetikleyicisi için, Contosotetikleyicisi dış hizmet olaylarına ardışık düzen aracılığıyla.

### Örnek 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Get-AzSynapseTriggerSubscriptionStatus
```

Bu komut, "subscribtion" tetikleyicisi için, Contosotetikleyicisi dış hizmet olaylarına ardışık düzen aracılığıyla.
