# Set-AzDataLakeAnalyticsDataSource

## SYNOPSIS
Veri Lake Analytics hesabının veri kaynağının ayrıntılarını değiştirir.

## INDEKI

```
Set-AzDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-AccessKey] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Set-Azdatalakeanaliz Ticsdatasource** cmdlet 'i, bir Azure Data Lake Analytics hesabının veri kaynağının ayrıntılarını değiştirir.

## ÖRNEKLERDEN

### Örnek 1: veri kaynağının erişim anahtarını değiştirme
```
PS C:\>Set-AzDataLakeAnalyticsDataSource -Account "ContosoAdlAccount" -Blob "contosowasb" -AccessKey "...newaccesskey..."
```

Bu komut, Azure Blob depolama veri kaynağı için depolanan erişim tuşunu değiştirir.
