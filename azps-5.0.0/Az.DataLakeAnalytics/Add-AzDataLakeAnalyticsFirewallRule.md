# Add-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Bir veri Lake Analytics hesabına güvenlik duvarı kuralı ekler.

## INDEKI

```
Add-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Add-Azdatalakeanalizleri Ticsfirewallrule** cmdlet 'ı bir Azure Data Lake Analytics hesabına güvenlik duvarı kuralı ekler.

## ÖRNEKLERDEN

### Örnek 1: güvenlik duvarı kuralı ekleme
```
PS C:\>Add-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

Bu komut "ContosoAdlAcct" hesabından "My Firewall Rule" adlı güvenlik duvarı kuralını IP aralığıyla ekler: 127.0.0.1-127.0.0.10
