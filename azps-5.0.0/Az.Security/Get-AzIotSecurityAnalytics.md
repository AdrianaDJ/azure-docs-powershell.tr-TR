# Get-AzIotSecurityAnalytics

## SYNOPSIS
IoT güvenlik çözümlemelerini alma

## INDEKI

```
Get-AzIotSecurityAnalytics -ResourceGroupName <String> -SolutionName <String> [-Default]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Get-AzIotSecurityAnalytics cmdlet 'i, belirli bir IoT güvenlik çözümünün güvenlik çözümlemelerini döndürür
