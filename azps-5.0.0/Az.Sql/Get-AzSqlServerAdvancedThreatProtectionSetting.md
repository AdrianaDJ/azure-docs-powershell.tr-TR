# Get-AzSqlServerAdvancedThreatProtectionSetting

## SYNOPSIS
Sunucunun Gelişmiş tehdit koruması ayarlarını alır.

## INDEKI

```
Get-AzSqlServerAdvancedThreatProtectionSetting -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Get-AzSqlServerAdvancedThreatProtectionSetting** cmdlet 'i, BIR Azure SQL Server 'ın Gelişmiş tehdit koruması ayarlarını alır.
Bu cmdlet 'i kullanmak için, bu cmdlet 'in ayarları aldığı sunucuyu tanımlayacak *Resourcegroupname* ve *ServerName* parametrelerini belirtin.

## ÖRNEKLERDEN

### Örnek 1: sunucunun Gelişmiş tehdit koruması ayarlarını alma
```
PS C:\>Get-AzSqlServerAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Bu komut, server01 adlı bir sunucunun Gelişmiş tehdit koruması ayarlarını alır.
Sunucu, ResourceGroup11 kaynak grubuna atanır.
