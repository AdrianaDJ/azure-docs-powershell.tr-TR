# Repair-AzVmssServiceFabricUpdateDomain

## SYNOPSIS
El ile platform güncelleştirme etki alanı hizmet yapısı sanal makine ölçek kümesindeki sanal makineleri güncelleştirmeye yönelik ilerleme.

## INDEKI

### DefaultParameter (varsayılan)
```
Repair-AzVmssServiceFabricUpdateDomain [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-PlatformUpdateDomain] <Int32> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Resourceıdparameter
```
Repair-AzVmssServiceFabricUpdateDomain [-PlatformUpdateDomain] <Int32> [-ResourceId] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectParameter
```
Repair-AzVmssServiceFabricUpdateDomain [-PlatformUpdateDomain] <Int32>
 [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
El ile platform güncelleştirme etki alanını zorla hizmet yapısı sanal makine ölçek kümesindeki sanal makineleri güncelleştirmeye ilerleme.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Repair-AzVmssServiceFabricUpdateDomain -ResourceGroupName $rgname -VMScaleSetName $vmssName -PlatformUpdateDomain 0
```

Bu komut, kaynak grubu adı ve ölçek kümesi adıyla belirtilen sanal makine ölçek kümesi

### Örnek 2
```
PS C:\> $vmss = Get-AzVmss -ResourceGroupName $rgname -VMScaleSetName $vmssName
PS C:\> Repair-AzVmssServiceFabricUpdateDomain -VirtualMachineScaleSet $vmss -PlatformUpdateDomain 1
```

Bu komut, VM Ölçek kümesi nesnesinin belirttiği sanal makine ölçeği kümesi için UD 1 ' de hizmet yapısı güncelleştirmesini zorlar.

### Örnek 3
```
PS C:\> Repair-AzVmssServiceFabricUpdateDomain -ResourceId $resourceId  -PlatformUpdateDomain 2;
```

Bu komut, kaynak kimliği tarafından belirtilen sanal makine ölçeği kümesi için UD 2 ' ye hizmet yapısı güncelleştirmesini zorlar.
