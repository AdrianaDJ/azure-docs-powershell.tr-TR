# Get-AzSnapshot

## SYNOPSIS
Anlık görüntünün özelliklerini alır

## INDEKI

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-azsnapshot** cmdlet 'inin özelliklerini alır.
