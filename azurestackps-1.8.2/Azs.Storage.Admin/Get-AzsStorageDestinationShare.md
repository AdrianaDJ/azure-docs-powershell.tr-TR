# Get-AzsStorageDestinationShare

## SYNOPSIS
Sistemin geçiş için en iyi aday olarak belirlediği hedef paylaşımların listesini döndürür.

## INDEKI

```
Get-AzsStorageDestinationShare [-SourceShareName] <String> [-FarmName] <String> [[-ResourceGroupName] <String>]
 [<CommonParameters>]
```

## Tanım
Sistemin geçiş için en iyi aday olarak belirlediği hedef paylaşımların listesini döndürür.

## ÖRNEKLERDEN

### ÖRNEK 1
```
Get-AzsStorageDestinationShare -FarmName f9b8e2e2-e4b4-44e0-9d92-6a848b1a5376 -SourceShareName "||SU1FileServer.azurestack.local|SU1_ObjStore"
```

Sistemin geçiş için en iyi aday olarak belirlediği hedef paylaşımların listesini edinin.
