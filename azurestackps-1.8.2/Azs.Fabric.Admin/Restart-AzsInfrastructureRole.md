# Restart-AzsInfrastructureRole

## SYNOPSIS
Requestd altyapısı rolünü yeniden başlatır.

## INDEKI

### Yeniden Başlat (varsayılan)
```
Restart-AzsInfrastructureRole -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-AsJob]
 [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Içermiyor
```
Restart-AzsInfrastructureRole -ResourceId <String> [-AsJob] [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Requestd altyapısı rolünü yeniden başlatır.

## ÖRNEKLERDEN

### ÖRNEK 1
```
Restart-AzsInfrastructureRole -Name "Active Directory Federation Services"
```

Kilitlendi bir altyapı rolünü yeniden başlatın.
