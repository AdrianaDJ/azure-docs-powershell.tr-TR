# New-AzApplicationGatewayFirewallPolicy

## SYNOPSIS
Uygulama ağ geçidi güvenlik duvarı ilkesi oluşturur.

## INDEKI

```
New-AzApplicationGatewayFirewallPolicy -Name <String> -ResourceGroupName <String> -Location <String>
 [-CustomRule <PSApplicationGatewayFirewallCustomRule[]>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Yeni-AzApplicationGatewayFirewallPolicy** cmdlet 'i uygulama ağ geçidi güvenlik ilkesi oluşturur.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $firewallPolicy = New-AzApplicationGatewayFirewallPolicy -Name wafResource1 -ResourceGroupName "rg1"  -Location  "westus" -CustomRules $customRule
```

Bu komut, $customRule değişkeninde tanımlanan özel kurallarla "westus" kaynak grubunda "RG1" kaynak grubunda "" adlı yeni bir Azure uygulama ağ geçidi güvenlik ilkesi oluşturur
