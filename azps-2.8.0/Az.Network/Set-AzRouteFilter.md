# Set-AzRouteFilter

## SYNOPSIS
Bir yol filtresi güncelleştirir.

## INDEKI

```
Set-AzRouteFilter -RouteFilter <PSRouteFilter> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-AzApplicationGateway** cmdlet 'i bir yol filtresini güncelleştirir

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Set-AzRouteFilter -RouteFilter $rf
```

Bu komut, $rf değişkenindeki ayarlar ile yol filtresini güncelleştirir.
