# Remove-AzPrivateEndpointConnection

## SYNOPSIS
Özel uç nokta bağlantısını kaldırır.

## INDEKI

### Byresourceıd (varsayılan)
```
Remove-AzPrivateEndpointConnection [-Force] [-AsJob] [-PassThru] -ResourceId <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResource
```
Remove-AzPrivateEndpointConnection [-Force] [-AsJob] [-PassThru] -Name <String> -ServiceName <String>
 -ResourceGroupName <String> [-Description <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzPrivateEndpointConnection** cmdlet 'i özel uç nokta bağlantısını kaldırır. 

## ÖRNEKLERDEN

### Örnek 1
```
Remove-AzPrivateEndpointConnection -Name MyPrivateEndpointConnection1 -ResourceGroupName TestResourceGroup -ServiceName MyPrivateLinkServiceName
```

Bu örnekte MyPrivateEndpointConnection1 adlı özel bir uç nokta bağlantısı kaldırılsın
