# Set-AzRouteTable

## SYNOPSIS
Yol tablosunu güncelleştirir.

## INDEKI

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-AzRouteTable** cmdlet 'i bir yol tablosunu güncelleştirir.

### Örnek 2: yol tablosunu değiştirme

```
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
False
PS C:\> $rt.DisableBgpRoutePropagation = $true
PS C:\> Set-AzRouteTable -RouteTable $rt
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
True
```

İlk komut rtName adındaki yol tablosunu alır ve $rt değişkeninde depolar.
İkinci komut, Disablebgprouteyayma değerini görüntüler.
Disablebgprouteyaymanın üçüncü komut güncelleştirme değeri.
Dördüncü komut sunucudaki yol tablosunu güncelleştirir.
Beşinci komut, güncelleştirilmiş yol tablosunu alır ve $rt değişkeninde depolar.
Altıncı komut, Disablebgprouteyayma değerini görüntüler.
