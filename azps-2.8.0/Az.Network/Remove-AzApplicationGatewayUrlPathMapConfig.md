# Remove-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
URL yol eşlemelerini arka uç sunucu havuzuna kaldırır.

## INDEKI

```
Remove-AzApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Remove-AzApplicationGatewayUrlPathMapConfig** cmdlet 'i arka uç sunucu havuzuna yönelik URL yol eşlemelerini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: uygulama ağ geçidinden URL yolu eşlemesini kaldırma
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway $appgw -Name "map01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

İlk komut appGwName adlı uygulama ağ geçidini alır ve sonucu $appgw değişkenine depolar.
İkinci komut map01 adındaki URL yol eşlemesini uygulama ağ geçidinden kaldırır.
Üçüncü komut, uygulama ağ geçidini güncelleştirir.
