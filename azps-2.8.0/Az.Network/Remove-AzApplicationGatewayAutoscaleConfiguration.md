# Remove-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Otomatik ölçeklendirme yapılandırmasını uygulama ağ geçidinden kaldırır.

## INDEKI

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzApplicationGatewayAutoscaleConfiguration** cmdlet 'i var olan uygulama ağ geçidinden otomatik ölçeklendirme yapılandırmasını kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

İlk komut uygulama ağ geçidini alır ve $gw değişkeninde depolar.
İkinci komut, uygulama ağ geçidinden otomatik ölçeklendirme yapılandırmasını kaldırır.
Üçüncü komut, Azure 'daki uygulama ağ geçidini güncelleştirir.
