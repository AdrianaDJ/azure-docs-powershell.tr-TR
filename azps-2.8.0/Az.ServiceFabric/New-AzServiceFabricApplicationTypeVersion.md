# New-AzServiceFabricApplicationTypeVersion

## SYNOPSIS
Belirtilen kaynak grubu ve küme altında yeni uygulama türü sürümü oluşturun.

## INDEKI

```
New-AzServiceFabricApplicationTypeVersion [-ResourceGroupName] <String> [-ClusterName] <String>
 [-Name] <String> [-Version] <String> -PackageUrl <String> [-DefaultParameter <Hashtable>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Bu cmdlet, Package 'de belirtilen paketi kullanarak yeni bir uygulama türü sürümü oluşturur; Bu, dağıtım sırasında Azure Resource Manager için bir REST uç noktası aracılığıyla erişilebilir ve uygulamanın paketlenmesi Bu komut zaten yoksa uygulama türü oluşturur.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $resourceGroupName = "testRG"
PS C:\> $clusterName = "testCluster"
PS C:\> $appTypeName = "testAppType"
PS C:\> $version = "v1"
PS C:\> $packageUrl = "https://sftestapp.blob.core.windows.net/sftestapp/testApp_1.0.sfpkg"
PS C:\> New-AzServiceFabricApplicationTypeVersion -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Version $version -PackageUrl $packageUrl -Verbose
```

Bu örnek, "testAppType" türünde bir uygulama türü sürümü "v1" oluşturur. Pakette bulunan uygulama bildirimindeki sürüm, sürüm ile belirtilen sürümle aynı sürüme sahip olmalıdır.
