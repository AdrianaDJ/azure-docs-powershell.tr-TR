# Suspend-AzKustoCluster

## SYNOPSIS
Var olan kusto kümesini askıya alma.

## INDEKI

### Binaik Vseçresourcegroup (varsayılan)
```
Suspend-AzKustoCluster [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Byresourceıd
```
Suspend-AzKustoCluster [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Suspend-AzKustoCluster [-InputObject] <PSKustoCluster> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Var olan kusto kümesini askıya alma.

## ÖRNEKLERDEN

### Örnek 1-varolan bir kusto kümesini adla askıya alma

```
PS C:\> Suspend-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster
```

Yukarıdaki komut, "testrg" kaynak grubunda bulunan "mykustocluster" adlı kusto kümesini askıya alır.

### Örnek 2-varolan bir kusto kümesini boruları kullanarak askıya alma

```
PS C:\> Get-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster | Suspend-AzKustoCluster
```

Yukarıdaki komut, cmdlet 'i kullanarak "mykustocluster" kaynak grubunda bulunan "" adlı kusto kümesini alır `Get-AzKustoCluster` ve sonra da sonucu askıya almak için yol gösterir `Suspend-AzKustoCluster` .
