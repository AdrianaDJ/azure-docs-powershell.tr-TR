# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Data Lake Store hesabını kalıcı olarak siler.

## INDEKI

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzDataLakeStoreAccount** cmdlet 'ı bir Data Lake Store hesabını kalıcı olarak siler.

## ÖRNEKLERDEN

### Örnek 1: veri Lake Store hesabını kaldırma
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Bu komut, Data Lake Store 'dan ContosoADL adlı hesabı kaldırır.
