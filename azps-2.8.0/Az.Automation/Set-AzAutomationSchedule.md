# Set-AzAutomationSchedule

## SYNOPSIS
Otomasyon zamanlamasını değiştirir.

## INDEKI

```
Set-AzAutomationSchedule [-Name] <String> [-IsEnabled <Boolean>] [-Description <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Set-Azautomationzamanlama** cmdlet 'ı, Azure Otomasyonu 'nda bir zamanlamayı değiştirir.

## ÖRNEKLERDEN

### Örnek 1: zamanlamayı değiştirme
```
PS C:\>Set-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -Description "Automation Schedule" -ResourceGroupName "ResourceGroup01"
```

Bu komut, Schedule01 adlı zamanlamanın açıklamasını değiştirir.
