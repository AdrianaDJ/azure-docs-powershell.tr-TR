# Import-AzAutomationRunbook

## SYNOPSIS
Otomasyon Runbook 'unu alır.

## INDEKI

```
Import-AzAutomationRunbook [-Path] <String> [-Description <String>] [-Name <String>] [-Tags <IDictionary>]
 -Type <String> [-LogProgress <Boolean>] [-LogVerbose <Boolean>] [-Published] [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Import-AzAutomationRunbook** cmdlet 'ı bir Azure Otomasyonu runbook 'unu alır. Wps_2 ve wps_2 için Iş akışı runbook 'ları, (. graphrunbook) dosya için wps_2 (. ps1) dosya yolu belirtin Iş akışı runbook 'ları wps_2 için, betikte dosyanın adıyla eşleşen tek bir wps_2 Iş akışı tanımı bulunmalıdır.

## ÖRNEKLERDEN

### Örnek 1: dosyadan runbook Içeri aktarma
```
PS C:\> $Tags = @{"tag01"="value01"; "tag02"="value02"}
PS C:\> Import-AzAutomationRunbook -Path .\GraphicalRunbook06.graphrunbook -Tags $Tags -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -Type GraphicalPowershell
```

İlk komut $Tags değişkenine iki anahtar/değer çifti atar.
İkinci komut, GraphicalRunbook06 adlı bir grafik runbook 'u AutomationAccount01 adlı Otomasyon hesabına aktarır.
Komut ayrıca $Tags uygulamasında depolanan etiketleri de atar.
