# Get-AzAutomationSoftwareUpdateConfiguration

## SYNOPSIS
Azure Otomasyonu yazılım güncelleştirme yapılandırmalarının listesini alır.

## INDEKI

### Tümü (varsayılan)
```
Get-AzAutomationSoftwareUpdateConfiguration [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationSoftwareUpdateConfiguration -Name <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Byvmıd
```
Get-AzAutomationSoftwareUpdateConfiguration -AzureVMResourceId <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Get-AzAutomationSoftwareUpdateConfiguration yazılım güncelleştirmesi yapılandırmalarının listesini döndürür. Belirli bir yazılım güncelleştirmesi yapılandırması edinmek için, ad parametresini belirtin. Bu sanal makine için Azure Resource ID belirterek belirli Azure sanal makinesini hedefleyen yazılım güncelleştirme yapılandırmalarını da ekleyebilirsiniz.

## ÖRNEKLERDEN

Azure Otomasyonu yazılım güncelleştirme yapılandırmasını ada göre edinin.

