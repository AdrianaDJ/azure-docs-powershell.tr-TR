# Set-AzCognitiveServicesAccount

## SYNOPSIS
Bir hesabı değiştirir.

## INDEKI

```
Set-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-SkuName <String>]
 [-Tag <Hashtable[]>] [-CustomSubdomainName <String>] [-NetworkRuleSet <PSNetworkRuleSet>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-Azsitiveservicesaccount** cmdlet 'i, belirtilen öğretici Sien HIZMETLER hesabının SKU veya etiketlerini değiştirir.
