# New-AzApiManagementHttpMessageDiagnostic

## SYNOPSIS
**PsApiManagementHttpMessageDiagnostic** 'In bir HTTP ileti tanı ayarı olan bir örneğini oluşturur

## INDEKI

```
New-AzApiManagementHttpMessageDiagnostic [-HeadersToLog <String[]>] [-BodyBytesToLog <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**New-AzApiManagementHttpMessageDiagnostic** cmdlet 'ı http ileti tanı ayarını oluşturur.

## ÖRNEKLERDEN

### Örnek 1: temel bir http Ileti tanılama ayarı oluşturma
```powershell
PS C:\>  New-AzApiManagementHttpMessageDiagnostic -Headers 'Content-Type', 'UserAgent' -BodyBytes 100

Headers                   Body
-------                   ----
{Content-Type, UserAgent} Microsoft.Azure.Commands.ApiManagement.ServiceManagement.Models.PsApiManagementBodyDiagnosticSetting
```

`Content-Type`100 baytı ile birlikte günlüğe kaydetmek ve üstbilgiler için bir HTTP iletisi tanılama ayarı oluşturma `User-Agent``body`
