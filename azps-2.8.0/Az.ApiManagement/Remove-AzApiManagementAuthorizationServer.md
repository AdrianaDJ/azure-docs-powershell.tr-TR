# Remove-AzApiManagementAuthorizationServer

## SYNOPSIS
Yetkilendirme sunucusunu kaldırır.

## INDEKI

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azapsananagementauthorizationserver** cmdlet 'ı BIR Azure API yönetim yetkilendirme sunucusunu kaldırır.

## ÖRNEKLERDEN

### Örnek 1: yetkilendirme sunucusunu kaldırma
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Bu komut belirtilen API yönetim yetkilendirme sunucusunu kaldırır.
*Force* parametresi belirtildiğinden, hiçbir onay gerekmez.
