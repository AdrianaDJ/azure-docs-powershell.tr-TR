# Remove-AzAvailabilitySet

## SYNOPSIS
Azure 'dan bir kullanılabilirlik kümesini kaldırır.

## INDEKI

```
Remove-AzAvailabilitySet [-ResourceGroupName] <String> [[-Name] <String>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzAvailabilitySet** cmdlet 'i Azure 'dan bir kullanılabilirlik kümesini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: kullanılabilirlik kümesini kaldırma
```
PS C:\> Remove-AzAvailabilitySet -Name "AvailabilitySet03" -ResourceGroupName "ResourceGroup11"
```

Bu komut, ResourceGroup11 adındaki kaynak grubundaki AvailabilitySet03 adındaki bir kullanılabilirlik kümesini kaldırır.
