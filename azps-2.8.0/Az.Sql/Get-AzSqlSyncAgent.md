# Get-AzSqlSyncAgent

## SYNOPSIS
Azure SQL eşitleme aracıları hakkında bilgi verir.

## INDEKI

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzSqlSyncAgent** cmdlet 'i bir veya birden çok Azure SQL eşitleme Aracısı hakkında bilgi döndürür.
Yalnızca bu eşitleme aracısının bilgilerini görmek için eşitleme aracısının adını belirtin.
