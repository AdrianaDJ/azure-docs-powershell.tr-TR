# Get-AzSqlDatabaseImportExportStatus

## SYNOPSIS
Azure SQL veritabanı alma veya verme işleminin ayrıntılarını alır.

## INDEKI

```
Get-AzSqlDatabaseImportExportStatus [-OperationStatusLink] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Get-Azsqldatabaseımportexportstatus** cmdlet 'i, depolama hesabındaki bir bacpac dosyasının ayrıntılarını, bir depolama hesabındaki BIR Azure SQL veritabanına aktarma
Bu cmdlet, Azure 'da SQL Server genişletme veritabanı hizmeti tarafından da desteklenir.

## ÖRNEKLERDEN

### Örnek 1: SQL veritabanının içeri ve dışarı aktarma durumunu alma
```
PS C:\>Get-AzSqlDatabaseImportExportStatus -OperationStatusLink "https://management.contoso.com/subscriptions/00000000-0000-0000-0000-000000000000/resourceGroups/resource01/providers/Microsoft.Sql/servers/server01/databases/database01/importExportOperationResults/00000000-000-0000-0000-000000000000?api-version=2014-04-01"
OperationStatusLink : 
ErrorMessage        : 
LastModifiedTime    : 4/15/2016 10:16:14 PM
QueuedTime          : 4/15/2016 10:16:13 PM
StatusMessage       : Running, Progress = 5.00 %
Status              : InProgress
```

Bu komut, belirtilen URL 'de bir veritabanı için içeri veya dışarı aktarma isteğinin durumunu alır.
