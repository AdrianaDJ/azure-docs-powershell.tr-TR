# Set-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Bir veritabanı coğrafi yedekleme ilkesi ayarlar.

## INDEKI

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Set-AzSqlDatabaseGeoBackupPolicy** cmdlet 'i veritabanına kaydedilen coğrafi yedekleme ilkesini ayarlar.
Bu, yedek depolama ilkesini tanımlamak için kullanılan bir Azure yedekleme kaynağı.
