# Get-AzSqlElasticPool

## SYNOPSIS
Bir Azure SQL veritabanında esnek havuzlar ve özellik değerlerini alır.

## INDEKI

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azsqlelaunpool** cmdlet 'i esnek havuzlar ve bunların özellik değerlerini alır.
Yalnızca bu havuzun özellik değerlerini görmek için varolan esnek havuzun adını belirtin.
