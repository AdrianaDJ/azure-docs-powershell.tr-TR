# Clear-AzSqlServerAdvancedThreatProtectionSettings

## SYNOPSIS
Gelişmiş tehdit koruması ayarlarını sunucudan kaldırır.

## INDEKI

```
Clear-AzSqlServerAdvancedThreatProtectionSettings [-PassThru] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
Clear-AzSqlServerAdvancedThreatProtectionSettings cmdlet 'i Azure SQL Server 'dan Gelişmiş tehdit koruması ayarlarını kaldırır.
Bu cmdlet 'i kullanmak için, bu cmdlet 'in ayarları kaldırdığı sunucuyu belirlemek için ResourceGroupName ve ServerName parametrelerini belirtin.

## ÖRNEKLERDEN

### Örnek 1: veritabanı için Gelişmiş tehdit koruması ayarlarını kaldırma
```
PS C:\> Clear-AzSqlServerAdvancedThreatProtectionSettings -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Bu komut, server01 adındaki Gelişmiş tehdit koruması ayarlarını kaldırır.
