# Remove-AzRelayNamespace

## SYNOPSIS
Belirtilen kaynak grubundan ad alanını kaldırır. 

## INDEKI

```
Remove-AzRelayNamespace [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzRelayNamespace** cmdlet 'i belirtilen kaynak grubundaki ad alanını kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Remove-AzRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -Name TestNameSpace-Relay1
```

Belirtilen kaynak grubundan geçiş ad alanını kaldırır `TestNameSpace-Relay1` `Default-ServiceBus-WestUS` .
