# Remove-AzStreamAnalyticsInput

## SYNOPSIS
Akış Analizi işinden bir girişi siler.

## INDEKI

```
Remove-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzStreamAnalyticsInput** cmdlet 'i Azure 'Daki bir Stream Analytics işinden gelen bir girişi zaman uyumsuz olarak siler.

## ÖRNEKLERDEN

### Örnek 1: bir işten giriş akışını kaldırma
```
PS C:\>Remove-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Bu komut, StreamingJob 'dan giriş Olayakışını kaldırır.
