# Revoke-AzHDInsightRdpServicesAccess

## SYNOPSIS
Windows kümesine RDP erişimini devre dışı bırakır.

## INDEKI

```
Revoke-AzHDInsightRdpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Revoke-AzHDInsightRdpServicesAccess** cmdlet 'i Windows tabanlı bir Azure HDInsight kümesine Uzak Masaüstü Protokolü (RDP) erişimini devre dışı bırakır.

## ÖRNEKLERDEN

### Örnek 1: belirtilen kümeye RDP erişimini devre dışı bırakma
```
PS C:\>Revoke-AzHDInsightRdpServicesAccess -ClusterName "your-hadoop-001"
```

Bu komut,-Hadoop-001 adlı kümeye RDP erişimini iptal eder.
