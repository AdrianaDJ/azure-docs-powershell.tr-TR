# Remove-AzStorageBlob

## SYNOPSIS
Belirtilen depolama blobunu kaldırır.

## INDEKI

### Ad boru hattı (varsayılan)
```
Remove-AzStorageBlob [-Blob] <String> [-Container] <String> [-DeleteSnapshot] [-Force] [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Blobboru hattı
```
Remove-AzStorageBlob -CloudBlob <CloudBlob> [-DeleteSnapshot] [-Force] [-PassThru] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Kapsayıcı boru hattı
```
Remove-AzStorageBlob -CloudBlobContainer <CloudBlobContainer> [-Blob] <String> [-DeleteSnapshot] [-Force]
 [-PassThru] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Remove-AzStorageBlob** cmdlet 'i, belirtilen blob 'u Azure 'daki bir depolama hesabından kaldırır.

## ÖRNEKLERDEN

### Örnek 1: depolama blob 'unu ada göre kaldırma
```
PS C:\>Remove-AzStorageBlob -Container "ContainerName" -Blob "BlobName"
```

Bu komut adıyla tanımlanan bir blob 'u kaldırır.

### Örnek 2: ardışık düzeni kullanarak depolama blobunu kaldırma
```
PS C:\>Get-AzStorageBlob -Container "ContainerName" -Blob "BlobName" | Remove-AzStorageBlob
```

Bu komut ardışık düzeni kullanır.

### Örnek 3: ardışık düzeni kullanarak depolama bloblarını kaldırma
```
PS C:\>Get-AzStorageContainer -Container container* | Remove-AzStorageBlob -Blob "BlobName"
```

Bu komut, blob veya blob 'ları almak için yıldız (*) joker karakterini ve ardışık düzeni kullanır ve sonra bunları kaldırır.
