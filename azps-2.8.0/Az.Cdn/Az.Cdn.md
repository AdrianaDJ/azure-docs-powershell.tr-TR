# Az. CDN modülü
## Tanım
Bu bölümdeki konular, Azure Resource Manager (ARM) çerçevesindeki Azure Içerik teslim ağı (CDN) için Azure PowerShell cmdlet 'lerini belgeleyin. Cmdlet 'ler Microsoft. Azure. Commands. CDN ad alanında bulunur.
