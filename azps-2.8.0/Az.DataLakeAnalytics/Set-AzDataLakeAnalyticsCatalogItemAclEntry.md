```
 -Permissions <PermissionType> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Set-Azdatalakeanalizleri Tic, Alogitemaclentry** cmdlet 'ı, Data Lake Analytics 'teki bir kataloğun veya katalog öğesinin erişim denetim lıstesıne (ACL) GIRDI (ACE) ekler veya değiştirir.

## ÖRNEKLERDEN

### Örnek 1: Katalog için Kullanıcı izinlerini değiştirme
```powershell
PS C:\> Set-AzDataLakeAnalyticsCatalogItemAclEntry -Account "contosoadla" -User -ObjectId (Get-AzADUser -Mail "PattiFuller@contoso.com").Id -Permissions Read

Type  Id                                   Permissions
----  --                                   -----------
User  90a6f74b-fd73-490e-900a-c4f0f9694d02        Read
Group 902b155a-5601-4ca8-8178-ad3289211f88   ReadWrite
Other 00000000-0000-0000-0000-000000000000        None
User  bd0b55bb-3a57-442a-b2f6-78c95c10ef86        Read
```

Bu komut, Patti Tamlaştırıcı 'ın Katalog ACE 'sini okuma izinlerine sahip olacak şekilde değiştirir.

### Örnek 2: veritabanı için Kullanıcı Izinlerini değiştirme
```powershell
PS C:\> Set-AzDataLakeAnalyticsCatalogItemAclEntry -Account "contosoadla" -User -ObjectId (Get-AzADUser -Mail "PattiFuller@contoso.com").Id -ItemType Database -Path "databaseName" -Permissions Read

Type  Id                                   Permissions
----  --                                   -----------
User  90a6f74b-fd73-490e-900a-c4f0f9694d02        Read
Group 902b155a-5601-4ca8-8178-ad3289211f88   ReadWrite
Other 00000000-0000-0000-0000-000000000000        None
User  bd0b55bb-3a57-442a-b2f6-78c95c10ef86        Read
```

Bu komut, Patti Tamleri'nin veritabanı ACE 'sini okuma izinlerine sahip olacak şekilde değiştirir.

### Örnek 3: kataloğun diğer izinlerini değiştirme
```powershell
PS C:\> Set-AzDataLakeAnalyticsCatalogItemAclEntry -Account "contosoadla" -Other -Permissions Read

----  --                                   -----------
User  90a6f74b-fd73-490e-900a-c4f0f9694d02        Read

Other 00000000-0000-0000-0000-000000000000        Read
User  bd0b55bb-3a57-442a-b2f6-78c95c10ef86        Read
```

Bu komut, diğer için Katalog ACE 'sini okuma izinlerine sahip olacak şekilde değiştirir.

### Örnek 4: veritabanı için diğer Izinleri değiştirme
```powershell

User  bd0b55bb-3a57-442a-b2f6-78c95c10ef86        Read
```

### Örnek 5: kataloğun Kullanıcı sahibi izinlerini değiştirme

Bu komut, hesabın sahip iznini okuma izni olarak ayarlar.

### Örnek 6: veritabanı için Kullanıcı sahibi Izinlerini değiştirme
```powershell

```
