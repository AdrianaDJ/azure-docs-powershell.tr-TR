# AzureRM. Locationbasevseçservices modülü
## Tanım
Bu konuda, Azure konum tabanlı hizmetler cmdlet 'Lerinin yardımı görüntülenir.

## AzureRM. Locationbasevseçservices cmdlet 'Leri
### [Get-Azurermlocationbasevseçservicesaccount](Get-AzureRmLocationBasedServicesAccount.md)
Hesabı alır.

### [Get-Azurermlocationbasevseçservicesaccountkey](Get-AzureRmLocationBasedServicesAccountKey.md)
Bir hesabın API anahtarlarını alır. Bu tuşlar, Azure konum tabanlı hizmetler için sonraki çağrılarda kullanılan kimlik doğrulama mekanizmasıdır.

### [Yeni-Azurermlocationbasevseçservicesaccount](New-AzureRmLocationBasedServicesAccount.md)

### [New-Azurermlocationbasevseçservicesaccountkey](New-AzureRmLocationBasedServicesAccountKey.md)
Hesap anahtarını yeniden oluşturur.

### [Remove-Azurermlocationbasevseçservicesaccount](Remove-AzureRmLocationBasedServicesAccount.md)
Konum tabanlı hizmetler hesabını siler.
