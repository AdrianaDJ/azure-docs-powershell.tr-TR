# Set-AzureRmDtlAllowedVMSizesPolicy

## SYNOPSIS
DevTest laboratuvarlarında bir laboratuvarın izin verilen sanal makine boyutları ilkesini ayarlar.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### Etkinleştir (varsayılan)
```
Set-AzureRmDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Bırakılacağı
```
Set-AzureRmDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Set-AzureRmDtlAllowedVMSizesPolicy** cmdlet 'i, bir laboratuvarda izin verilen sanal makine boyutlarının listesini belirten izin verilen sanal makine boyutları ilkesini ayarlar.
Cmdlet, ilkeyi ayarlamak için belirtilen kaynak grubunu ve laboratuarın adını kullanır.
