# Stop-AzureRmSqlDatabaseActivity

## SYNOPSIS
Veritabanındaki zaman uyumsuz güncelleştirmeler işlemini iptal eder.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Stop-AzureRmSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Stop-AzureRmSqlDatabaseActivity** cmdlet 'i veritabanındaki zaman uyumsuz güncelleştirmeler işlemini iptal eder.

## ÖRNEKLERDEN

### Örnek 1: veritabanındaki zaman uyumsuz güncelleştirmeler işlemini Iptal etme
```

```

Bu komut, veritabanındaki zaman uyumsuz güncelleştirmeler işlemini iptal eder.
