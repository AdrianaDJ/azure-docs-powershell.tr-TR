# Set-AzureRmSqlServerVirtualNetworkRule

## SYNOPSIS
Azure SQL Server sanal ağ kuralının yapılandırmasını değiştirir.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Set-AzureRmSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Bu komut, Azure SQL Server sanal ağ kuralının yapılandırmasını değiştirir.


Sunucudaki sanal ağ kuralları kümesini denetlemek için bunun yerine ' Add-AzureRmSqlServerVirtualNetworkRule ' ve ' Remove-AzureRmSqlServerVirtualNetworkRule ' seçeneğini kullanın.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> $virtualNetworkRule = Set-AzureRmSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Yeni sanal ağ hakkında bilgi içeren yeni sanal ağ alt ağı kimliği ile mevcut bir sanal ağ kuralını değiştirir
