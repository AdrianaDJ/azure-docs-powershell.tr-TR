# Get-AzureRmSqlServerCommunicationLink

## SYNOPSIS
Veritabanı sunucuları arasındaki elastik veritabanı işlemleri için iletişim bağlantılarını alır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Get-AzureRmSqlServerCommunicationLink [[-LinkName] <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Get-AzureRmSqlServerCommunicationLink** cmdlet 'ı, Azure SQL veritabanındaki elastik veritabanı işlemleri için sunucudan sunucuya iletişim bağlantılarını alır.
Bağlantının özelliklerini görmek için sunucu iletişim bağlantısının adını belirtin.

## ÖRNEKLERDEN

### Örnek 1: sunucunun tüm iletişim bağlantılarını alma
```
PS C:\>Get-AzureRmSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17"
```

Bu komut, ContosoServer17 adlı sunucudaki elastik veritabanı işlemleri için tüm sunucudan sunucuya iletişim bağlantılarını alır.

### Örnek 2: sunucu için belirli bir iletişim bağlantısı alma
```
PS C:\>Get-AzureRmSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Bu komut, Link01 adlı sunucudan sunucuya iletişim bağlantısını alır.
