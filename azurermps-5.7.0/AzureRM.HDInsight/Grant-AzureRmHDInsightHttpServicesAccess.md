# Grant-AzureRmHDInsightHttpServicesAccess

## SYNOPSIS
Kümeye HTTP erişimi verir.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Grant-AzureRmHDInsightHttpServicesAccess [-ClusterName] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Grant-AzureRmHDInsightHttpServicesAccess** cmdlet 'ı, ODBC, ambarı, Oozie ve Web Hizmetleri kullanarak bir Azure HDıNSIGHT kümesine http erişimi verir.

## ÖRNEKLERDEN

### Örnek 1: Azure HDInsight kümesine HTTP erişimi verme
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzureRmHDInsightHttpServicesAccess `
            -ClusterName $clusterName `
            -HttpCredential $newClusterCreds
```

Bu komut,-Hadoop-001 adlı kümeye HTTP erişimi verir.
