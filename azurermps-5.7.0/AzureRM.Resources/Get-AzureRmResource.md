## Tanım
**Get-AzureRmResource** cmdlet 'i Azure kaynaklarını alır.

## ÖRNEKLERDEN

### Örnek 1: belirli türdeki tüm kaynakları alma
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType microsoft.web/sites
```

Bu komut, ResourceGroup11 altında Microsoft. Web/Sites türünde bir kaynak alır.

### Örnek 2: ada göre kaynak alma
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceName ContosoWebsite
```

Bu komut, ResourceGroup11 altında Contosoweb sitesi adlı bir kaynak alır.

### Örnek 3: kaynak grubundaki depolama hesaplarının tüm durumunu gösterme
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType Microsoft.ClassicStorage/storageAccounts -ExpandProperties |
   Select * -Expand Properties |
   Sort Name |
   Format-Table Name,Status*
```
