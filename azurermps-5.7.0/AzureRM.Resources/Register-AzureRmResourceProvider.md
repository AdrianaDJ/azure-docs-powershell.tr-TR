# Register-AzureRmResourceProvider

## SYNOPSIS
Kaynak sağlayıcısını kaydeder.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Register-AzureRmResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Register-AzureRmResourceProvider** cmdlet 'ı bir Azure Kaynak sağlayıcısı kaydeder.

## ÖRNEKLERDEN

### Örnek 1: sağlayıcıyı kaydettirme
```
PS C:\>Register-AzureRmResourceProvider -ProviderNamespace Microsoft.Network
```

Bu, hesabınızın Microsoft. Network sağlayıcısını kaydeder.
