# Remove-AzureBatchPool

## SYNOPSIS
Belirtilen toplu havuzu siler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
Remove-AzureBatchPool [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzureBatchPool** cmdlet 'ı belirtilen Azure toplu iş havuzunu siler.
*Force* parametresini kullanmazsanız onaylamanız istenir.

## ÖRNEKLERDEN

### Örnek 1: havuz KIMLIĞINE göre bir toplu Iş havuzu silme
```
PS C:\>Remove-AzureBatchPool -Id "MyPool" -BatchContext $Context
```

Bu komut MyPool KIMLIKLI havuzu siler.
Silme işlemi gerçekleşmeden önce kullanıcıdan onay istenir.

### Örnek 2: zorunlu olarak tüm toplu Iş havuzlarını silme
```
PS C:\>Get-AzureBatchPool -BatchContext $Context | Remove-AzureBatchPool -Force -BatchContext $Context
```

Bu komut tüm toplu Iş havuzlarını siler.
*Force* parametresi bulunduğundan, onay istemi bastırılır.

## PARAMETRELERINE

### -BatchContext
Bu cmdlet 'in toplu Iş hizmetiyle etkileşimli çalışmak için kullandığı **Batchaccountcontext** örneğini belirtir.
BatchAccountContext 'i almak için Get-AzureRmBatchAccount cmdlet 'ini kullanıyorsanız, toplu Iş hizmetiyle etkileşim kurarken Azure Active Directory kimlik doğrulaması kullanılır.
