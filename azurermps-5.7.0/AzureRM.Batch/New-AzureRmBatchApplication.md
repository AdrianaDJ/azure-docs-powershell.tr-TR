# New-AzureRmBatchApplication

## SYNOPSIS
Belirtilen toplu hesaba uygulama ekler.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

```
New-AzureRmBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationId] <String>
 [[-AllowUpdates] <Boolean>] [[-DisplayName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**New-AzureRmBatchApplication** cmdlet 'ı belirtilen Azure toplu hesabına bir uygulama ekler.

## ÖRNEKLERDEN

### Örnek 1: bir toplu Iş hesabına boş uygulama ekleme
```
PS C:\>New-AzureRmBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "Litware" -AllowUpdates $True -DisplayName "Litware Advanced Reticulator"
```

Bu komut, ContosoBatch hesabında Litwin uygulamasını oluşturur.
Uygulama başlangıçta paket yok.
