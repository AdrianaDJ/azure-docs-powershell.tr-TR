# Start-AzureRmSiteRecoveryCommitFailoverJob

## SYNOPSIS
Site kurtarma nesnesi için yük devretmeyi Yürüt eylemini başlatır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## INDEKI

### ByRPIObject (varsayılan)
```
Start-AzureRmSiteRecoveryCommitFailoverJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByRPObject
```
Start-AzureRmSiteRecoveryCommitFailoverJob -RecoveryPlan <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Btypeınfo
```
Start-AzureRmSiteRecoveryCommitFailoverJob -ProtectionEntity <ASRProtectionEntity>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Start-AzureRmSiteRecoveryCommitFailoverJob** cmdlet 'ı bir Azure Site Recovery nesnesi için yük devretme işlemi tamamlandıktan sonra işlemi tamamlama işlemini başlatır.
