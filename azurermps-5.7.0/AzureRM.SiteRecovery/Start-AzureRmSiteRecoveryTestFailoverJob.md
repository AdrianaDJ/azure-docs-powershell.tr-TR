# Start-AzureRmSiteRecoveryTestFailoverJob

## SYNOPSIS
Site kurtarma koruma varlığı için test yük devretmesi başlatır.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Tanım
**Start-AzureRmSiteRecoveryTestFailoverJob** cmdlet 'ı bir Azure Site Recovery koruma varlığının veya kurtarma planının yük devretmeyi başlatır.
Get-AzureRmSiteRecoveryJob cmdlet 'ini kullanarak işin başarılı olup olmadığını denetleyebilirsiniz.
