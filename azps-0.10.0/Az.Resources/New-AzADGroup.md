# New-AzADGroup

## SYNOPSIS
Yeni bir Active Directory grubu oluşturur.

## INDEKI

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Yeni bir Active Directory grubu oluşturur.

## ÖRNEKLERDEN

### Örnek 1-yeni bir AD grubu oluşturma

```
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

"MyGroupDisplayName" ve posta takma adı "" adlı yeni bir AD grubu oluşturur myemail@domain.com .
