# Remove-AzManagedApplication

## SYNOPSIS
Yönetilen uygulamayı kaldırır

## INDEKI

### Removebynaik Vseçresourcegroup (varsayılan)
```
Remove-AzManagedApplication -Name <String> -ResourceGroupName <String> [-Force] [-ApiVersion <String>]
 [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Removebyıd
```
Remove-AzManagedApplication -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzManagedApplication** cmdlet 'i yönetilen uygulamayı kaldırır

## ÖRNEKLERDEN

### Örnek 1: yönetilen uygulamayı kaynak KIMLIĞIYLE kaldırma
```
PS C:\>$Application = Get-AzManagedApplication -Name "myApp" -ResourceGroupName "myRG"
PS C:\>Remove-AzManagedApplication -Id $Application.ResourceId -Force
```

İlk komut, Uygulamam adlı bir yönetilen uygulamayı Get-AzManagedApplication cmdlet 'i kullanarak alır.
Komut, $Application değişkeninde depolar.
İkinci komut $Application 'in **RESOURCEID** özelliğiyle tanımlanan yönetilen uygulamayı kaldırır.
