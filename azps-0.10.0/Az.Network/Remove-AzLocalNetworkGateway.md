# Remove-AzLocalNetworkGateway

## SYNOPSIS
Yerel ağ geçidi siler

## INDEKI

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Yerel ağ geçidi, VPN cihazınızı şirket Içinde temsil eden bir nesnedir.

**Remove-AzLocalNetworkGateway** cmdlet 'i, şirket içi ağ geçidinizi temsil eden nesneyi ad ve kaynak grubu adına göre siler.

## ÖRNEKLERDEN

### 1: yerel ağ ağ geçidini silme
```
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Yerel ağ geçidinin "myLocalGW" adlı nesnesini, "myRG" kaynak grubunda siler

Not: **Remove-AzVirtualNetworkGatewayConnection** cmdlet 'Ini kullanarak yerel ağ ağ geçitindeki tüm bağlantıları silmelisiniz.
