# Remove-AzNetworkInterface

## SYNOPSIS
Ağ arabirimini kaldırır.

## INDEKI

```
Remove-AzNetworkInterface -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzNetworkInterface** cmdlet 'ı bir Azure ağ arabirimini kaldırır.

## ÖRNEKLERDEN

### Örnek 1: ağ arabirimini kaldırma
```
PS C:\>Remove-AzNetworkInterface -Name "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

Bu komut, kaynak grubundaki ResourceGroup1 NetworkInterface1 ağ arabirimini kaldırır.
*Force* parametresi kullanılmadığından, kullanıcıdan bu eylemi onaylaması istenir.

### Örnek 2: ağ arabirimini kaldırma
```
PS C:\>Get-AzNetworkInterface -ResourceGroupName "ResourceGroup1" | Remove-AzNetworkInterface -Force
```

Bu komut, kaynak grubu ResourceGroup1 tüm ağ arabirimlerini kaldırır.
*Force* parametresi kullanıldığından, kullanıcıdan onay istenmez.
