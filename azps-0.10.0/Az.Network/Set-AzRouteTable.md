# Set-AzRouteTable

## SYNOPSIS
Yol tablosunun hedef durumunu ayarlar.

## INDEKI

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-AzRouteTable** cmdlet 'ı bir Azure yol tablosunun hedef durumunu ayarlar.
