# Remove-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Uygulama ağ geçidinden kimlik doğrulama sertifikasını kaldırır.

## INDEKI

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String>
 -ApplicationGateway <PSApplicationGateway> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Remove-AzApplicationGatewayAuthenticationCertificate** cmdlet 'i, bir Azure uygulama ağ geçidinden kimlik doğrulama sertifikasını kaldırır.
