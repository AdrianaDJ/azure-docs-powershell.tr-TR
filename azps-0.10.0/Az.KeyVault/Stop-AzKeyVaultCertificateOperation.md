# Stop-AzKeyVaultCertificateOperation

## SYNOPSIS
Anahtar Kasası 'nda sertifika işlemini iptal eder.

## INDEKI

```
Stop-AzKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Dur-Azanahtarvaultcertificateoperation** cmdlet 'ı Azure Anahtar Kasası hizmetinde sertifika işlemini iptal eder.
