# Remove-AzKeyVaultCertificateOperation

## SYNOPSIS
Anahtar kasasından sertifika işlemini siler.

## INDEKI

```
Remove-AzKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzKeyVaultCertificateOperation** cmdlet 'i anahtar kasasından sertifika işlemini siler.

## ÖRNEKLERDEN

### Örnek 1: sertifika işlemini kaldırma
```
PS C:\>Remove-AzKeyVaultCertificateOperation -VaultName "ContosoKV01" -Name "TestCert01" -Force
```

Bu komut, TestCert01 adındaki sertifika işlemini, onay istemeden ContosoKV01 anahtar kasasından kaldırır.
