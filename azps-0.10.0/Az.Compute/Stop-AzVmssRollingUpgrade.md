# Stop-AzVmssRollingUpgrade

## SYNOPSIS
Geçerli sanal makine ölçek ayarı çalışırken yükseltmeyi iptal eder.

## INDEKI

### DefaultParameter (varsayılan)
```
Stop-AzVmssRollingUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Kolay Yöntem
```
Stop-AzVmssRollingUpgrade [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
Geçerli sanal makine ölçek ayarı çalışırken yükseltmeyi iptal eder.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Stop-AzVmssRollingUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Bu komut, "Group001" kaynak grubundaki "VMSS001" VM Ölçek kümesi için çalışırken yükseltmeyi iptal eder.
