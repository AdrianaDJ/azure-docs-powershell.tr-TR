# Get-AzVMExtensionImage

## SYNOPSIS
Bir Azure uzantısı için tüm sürümleri alır.

## INDEKI

```
Get-AzVMExtensionImage -Location <String> -PublisherName <String> -Type <String>
 [-FilterExpression <String>] [-Version <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
**Get-Azvmextensionımage** cmdlet 'ı bir Azure uzantısı için tüm sürümleri alır.

## ÖRNEKLERDEN

### Örnek 1: uzantı resminin sürümlerini alma
```
PS C:\> Get-AzVMExtensionImage -Location "Central US" -PublisherName "Fabrikam" -Type "FabrikamEndpointProtection"
```

Bu komut, belirtilen konum, yayıncı ve tür için uzantı resminin tüm sürümlerini alır.
