# Disable-AzVmssDiskEncryption

## SYNOPSIS
VM ölçeklendirme kümesinde disk şifrelemesini devre dışı bırakır.

## INDEKI

```
Disable-AzVmssDiskEncryption [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [[-ExtensionName] <String>] [-VolumeType <String>] [-ForceUpdate] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
VM ölçeklendirme kümesinde disk şifrelemesini devre dışı bırakır.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Disable-AzVmssDiskEncryption -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Group001 adındaki kaynak grubuna ait olan VMSS001 adlı VM Ölçek kümesindeki disk şifrelemesini devre dışı bırakır.
