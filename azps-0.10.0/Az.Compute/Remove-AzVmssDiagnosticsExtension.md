# Remove-AzVmssDiagnosticsExtension

## SYNOPSIS
Bir tanılama uzantısını VMSS 'den kaldırır.

## INDEKI

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <VirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-Azvmssdiagnosticsextenma** cmdlet 'ı, sanal makine ölçek kümesinden (VMSS) bir tanılama uzantısını kaldırır.

## ÖRNEKLERDEN

### Örnek 1: VMSS 'den tanılama uzantısı kaldırma
```
PS C:\> Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

Bu komut, VMSS 'den tanılama uzantısını kaldırır.
