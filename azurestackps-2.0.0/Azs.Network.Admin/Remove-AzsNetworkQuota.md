# Remove-AzsNetworkQuota

## SYNOPSIS
Bir kotayı ada göre silme.

## INDEKI

### Sil (varsayılan)
```
Remove-AzsNetworkQuota -Name <String> [-Location <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Deleteviaıdentity
```
Remove-AzsNetworkQuota -InputObject <INetworkAdminIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Tanım
Bir kotayı ada göre silme.

## ÖRNEKLERDEN

### --------------------------ÖRNEK 1--------------------------
```
Remove-AzsNetworkQuota -Name NetworkQuota1
```

Ağ kotasını ada göre kaldırma.

### --------------------------ÖRNEK 2--------------------------
```
Get-AzsNetworkQuota -Name NetworkQuota1 | Remove-AzsNetworkQuota
```

Kanal kullanarak ağ kotasını kaldırma.

### --------------------------ÖRNEK 3--------------------------
```
Remove-AzsNetworkQuota -Name NetworkQuota1
```

Ağ kotasını kaldırma.
