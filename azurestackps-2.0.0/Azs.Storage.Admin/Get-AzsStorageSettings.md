# Get-AzsStorageSettings

## SYNOPSIS
Depolama kaynak sağlayıcısı ayarlarını döndürür.

## INDEKI

```
Get-AzsStorageSettings [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## Tanım
Depolama kaynak sağlayıcısı ayarlarını döndürür.

## ÖRNEKLERDEN

### Örnek 1:
```powershell
PS C:\> Get-AzsStorageSettings
```

Depolama ayarlarını edinin.
