# Set-AzDtlAutoShutdownPolicy

## SYNOPSIS
Laboratuvar dev-test laboratuarının otomatik kapatma ilkesini ayarlar.

## INDEKI

### Etkinleştir (varsayılan)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Bırakılacağı
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-AzDtlAutoShutdownPolicy** cmdlet 'i, belirli bir zamanda laboratuarda tüm sanal makineleri otomatik olarak kapatan bir laboratuvarın otomatik kapatma ilkesini ayarlar.
Cmdlet, ilkeyi ayarlamak için belirtilen kaynak grubunu ve laboratuarın adını kullanır.
