# Remove-AzServiceFabricClusterCertificate

## SYNOPSIS
Küme güvenliği için kullanılan küme sertifikasını kaldırma.

## INDEKI

```
Remove-AzServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Kümede zaten kullanımda olan başka bir geçerli sertifika olduğu sürece, kümeden küme sertifikasını kaldırmak için **Remove-AzServiceFabricClusterCertificate** kullanın.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Remove-AzServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Bu komut, küme güvenliği için parmak izi 5F3660C715EBBDA31DB1FFDCF508302348DE8E7A kullanılmasını kaldırır.
