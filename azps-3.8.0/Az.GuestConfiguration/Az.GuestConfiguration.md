# Az. GuestConfiguration modülü
## Tanım
Kullanıcıların Azure 'daki sanal makinelerin uyumluluk durumunu anlamasına olanak sağlayan cmdlet 'ler içeren modül.

## Az. GuestConfiguration cmdlet 'Leri
### [Get-AzVMGuestPolicyStatus](Get-AzVMGuestPolicyStatus.md)
VM 'e atanan "Konuk yapılandırması" türünde bir girişimin Konuk yapılandırma ilkesi durumlarını (ayrıntılı) alır.
Initiative, "Initiative" tanım türü ilkedir.

### [Get-Azvmguestpolicydurumgeçmişi](Get-AzVMGuestPolicyStatusHistory.md)
VM 'e atanan "Konuk yapılandırması" türündeki bir girişimin Konuk yapılandırma ilkesi uyumluluk durumu geçmişini alır.
Initiative, "Initiative" tanım türü ilkedir.
