# Get-AzDataFactoryV2DataFlow

## SYNOPSIS
Veri Fabrikası içindeki veri akışları hakkında bilgi alır.

## INDEKI

### ByFactoryName (varsayılan)
```
Get-AzDataFactoryV2DataFlow [[-Name] <String>] [-ResourceGroupName] <String> [-DataFactoryName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryV2DataFlow [[-Name] <String>] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Byresourceıd
```
Get-AzDataFactoryV2DataFlow [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
Get-AzDataFactoryV2DataFlow cmdlet 'i Azure Veri Fabrikası 'ndaki veri akışları hakkında bilgi alır.
Veri akışının adını belirtirseniz, bu cmdlet bu veri akışı hakkında bilgi alır.
Bir ad belirtmezseniz, bu cmdlet Veri Fabrikası içindeki tüm veri akışları hakkında bilgi alır.

## ÖRNEKLERDEN
### Örnek 1: tüm veri akışları hakkında bilgi alın
```powershell
PS C:\> Get-AzDataFactoryV2DataFlow -ResourceGroupName "ADF" -DataFactoryName "WikiADF"

DataFlowName           DataFactoryName ResourceGroupName                                                    Properties
------------           --------------- -----------------                                                    ----------
TaxiDemo1                      WikiADF               adf Microsoft.Azure.Management.DataFactory.Models.MappingDataFlow
dataflow1                      WikiADF               adf Microsoft.Azure.Management.DataFactory.Models.MappingDataFlow
dataflow3                      WikiADF               adf Microsoft.Azure.Management.DataFactory.Models.MappingDataFlow
```

Bu komut, WikiADF adlı veri fabrikası 'ndaki tüm veri akışları hakkında bilgi alır.

### Örnek 2: belirli bir veri akışı hakkında bilgi alma
```powershell
PS C:\> Get-AzDataFactoryV2DataFlow -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -Name "dataflow1"

DataFlowName           DataFactoryName ResourceGroupName                                                    Properties
------------           --------------- -----------------                                                    ----------
TaxiDemo1                      WikiADF               adf Microsoft.Azure.Management.DataFactory.Models.MappingDataFlow
```

Bu komut WikiADF adındaki dataflow1 adlı veri akışı hakkında bilgi alır.
