# Get-AzCosmosDBSqlDatabaseThroughput

## SYNOPSIS
CosmosDB SQL veritabanına karşılık gelen üretilen iş ayarlarını alır.

## INDEKI

### ByNameParameterSet (varsayılan)
```
Get-AzCosmosDBSqlDatabaseThroughput -ResourceGroupName <String> -AccountName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBSqlDatabaseThroughput -Name <String> -InputObject <PSDatabaseAccount>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azcosmosdbsqldatabaseüretilen** iş cmdlet 'ı Cosmosdb SQL veritabanına karşılık gelen üretilen iş ayarlarını alır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Get-AzCosmosDBSqlDatabaseThroughput -AccountName {accountName} -ResourceGroupName {resourceGroupName} -Name {databaseName}

Name                : {throughputResourceName}
Id                  : {throughputId}
Throughput          : 0
MinimumThroughput   :
OfferReplacePending :
```
