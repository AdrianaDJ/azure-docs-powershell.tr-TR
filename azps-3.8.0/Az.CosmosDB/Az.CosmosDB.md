# Az. CosmosDB modülü
## Tanım
Bu bölümdeki konular, Azure Resource Manager (ARM) çerçevesindeki Azure CosmosDB için Azure PowerShell cmdlet 'lerini belgeleyin. Microsoft. Azure. Commands. CosmosDB ad alanında cmdlet 'ler vardır.
