# Remove-AzBlueprintAssignment

## SYNOPSIS
Bir abonelikten bir şeması kaldırın.

## INDEKI

### DeleteBlueprintAssignmentByName (varsayılan)
```
Remove-AzBlueprintAssignment [[-SubscriptionId] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteBlueprintAssignmentByObject
```
Remove-AzBlueprintAssignment [[-SubscriptionId] <String>] [-InputObject] <PSBlueprintAssignment> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Aboneliğe atanmış bir şeması öğesini kaldırın.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Remove-AzBlueprintAssignment -Name "myAssignment" -Subscription "00000000-1111-0000-1111-000000000000" -Confirm
```

Belirtilen abonelikteki ad ile belirtilen şeması atamasını kaldırın. Cmdlet, komutu yürütmeden önce onay isteyecek.
