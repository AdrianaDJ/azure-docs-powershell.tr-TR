# Get-AzBlueprintArtifact

## SYNOPSIS
Bir şeması tanımından yapıları alın.

## INDEKI

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Bir şeması tanımından yapıları alın. Şeması tanım sürümü belirtilmemişse taslak sürümü alınır. Taslak sürümü olmadığı durumlarda, yayımlanan en son şeması tanımı döndürülür.
