# Get-AzStorageBlobContent

## SYNOPSIS
Bir Depolama Blobu yükler.

## INDEKI

### ReceiveManual (varsayılan)
```
Get-AzStorageBlobContent [-Blob] <String> [-Container] <String> [-Destination <String>] [-CheckMd5] [-Force]
 [-AsJob] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Blobboru hattı
```
Get-AzStorageBlobContent -CloudBlob <CloudBlob> [-Destination <String>] [-CheckMd5] [-Force] [-AsJob]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Kapsayıcı boru hattı
```
Get-AzStorageBlobContent -CloudBlobContainer <CloudBlobContainer> [-Blob] <String> [-Destination <String>]
 [-CheckMd5] [-Force] [-AsJob] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>]
 [-ClientTimeoutPerRequest <Int32>] [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Get-AzStorageBlobContent** cmdlet 'i belirtilen depolama blobundan indirir.
Blob adı yerel bilgisayar için geçerli değilse, bu cmdlet mümkünse otomatik olarak çözer.

## ÖRNEKLERDEN

### Örnek 1: blob içeriğini ada göre Indirme
```
PS C:\>Get-AzStorageBlobContent -Container "ContainerName" -Blob "Blob" -Destination "C:\test\"
```

Bu komut, bir blob 'u adla indirir.

### Örnek 2: ardışık düzeni kullanarak blob içeriğini Indirme
```
PS C:\>Get-AzStorageBlob -Container containername -Blob blobname | Get-AzStorageBlobContent
```

Bu komut, blob içeriğini bulmak ve indirmek için ardışık düzeni kullanır.

### Örnek 3: ardışık düzen ve joker karakter kullanarak blob içeriği Indirin
```
PS C:\>Get-AzStorageContainer container* | Get-AzStorageBlobContent -Blob "cbox.exe" -Destination "C:\test"
```

Bu örnekte, blob içeriğini bulmak ve indirmek için joker karakter ve ardışık düzen kullanılır.

### Örnek 4: blob nesnesi alma ve bir değişkene kaydetme, ardından blob nesnesini blob nesnesiyle indirme
```
PS C:\>$blob = Get-AzStorageBlob -Container containername -Blob blobname 
PS C:\>Get-AzStorageBlobContent -CloudBlob $blob.ICloudBlob -Destination "C:\test"
```

Bu örnekte, önce bir blob nesnesi alın ve bir değişkene kaydedin, ardından blob nesnesini blob nesnesiyle indirin.
