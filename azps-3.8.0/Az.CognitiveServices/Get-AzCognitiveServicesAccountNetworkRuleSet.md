# Get-AzCognitiveServicesAccountNetworkRuleSet

## SYNOPSIS
Bir öğretici hizmet hesabının NetworkRule özelliğini alma

## INDEKI

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azlıveiveservicesaccountnetworkruleset** cmdlet 'i, bir öğretici hizmet hesabının networkrule özelliğini alır

## ÖRNEKLERDEN

### Örnek 1: belirtilen bir öğretici hizmet hesabının NetworkRule özelliğini al
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

Bu komut belirtilen bir öğretici hizmet hesabının NetworkRule özelliğini alır
