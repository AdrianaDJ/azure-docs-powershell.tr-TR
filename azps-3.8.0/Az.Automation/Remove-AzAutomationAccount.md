# Remove-AzAutomationAccount

## SYNOPSIS
Otomasyon hesabını kaldırır.

## INDEKI

```
Remove-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzAutomationAccount** cmdlet 'i, bir kaynak grubundan Azure Otomasyonu hesabını kaldırır.
Otomasyon hesapları hakkında daha fazla bilgi için New-AzAutomationAccount cmdlet bölümüne bakın.

## ÖRNEKLERDEN

### Örnek 1: Otomasyon hesabını kaldırma
```
PS C:\>Remove-AzAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

Bu komut, kullanıcı doğrulaması istemeden ContosoAutomationAccount adlı bir Otomasyon hesabını kaldırır.
