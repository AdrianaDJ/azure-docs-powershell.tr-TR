# Remove-AzRouteFilterRuleConfig

## SYNOPSIS
Yol filtresi 'nden bir yol filtresi kuralını kaldırır.

## INDEKI

```
Remove-AzRouteFilterRuleConfig -Name <String> -RouteFilter <PSRouteFilter> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzRouteFilterRuleConfig** cmdlet 'i, yol filtresi 'nden bir yol filtresi kuralını kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
```

İlk komut, ResourceGroup01 adındaki kaynak grubuna ait RouteFilter01 adlı bir yol filtresi alır ve $rf değişkeninde depolar.
İkinci komut, $rf depolanan yol filtresindeki Rule01 adındaki yol filtresi kuralını kaldırır.
