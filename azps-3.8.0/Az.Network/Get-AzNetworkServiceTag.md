# Get-AzNetworkServiceTag

## SYNOPSIS
Hizmet etiketi bilgi kaynakları listesini alır.

## INDEKI

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzNetworkServiceTag** cmdlet 'i, hizmet etiketi bilgi kaynaklarının listesini alır.

Belirttiğiniz Azure bölgesi bilgilerinin sürüm için bir başvuru olarak kullanılacağını (konuma göre filtre olarak değil) unutmayın. Örneğin, belirtseniz bile, `-Location eastus2` tüm bölgelerde önek ayrıntılarını içeren, ancak aboneliğinizin ait olduğu bulut ile sınırlı olan hizmet etiketlerinin listesini alacaksınız (örneğin, genel, ABD hükümeti, Çin veya Almanya).
