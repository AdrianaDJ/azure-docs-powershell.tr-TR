# Set-AzVirtualNetworkTap

## SYNOPSIS
Sanal bir ağı güncelleştirir.

## INDEKI

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-AzVirtualNetworkTap** cmdlet 'i sanal bir ağ öğesine dokunun.

## ÖRNEKLERDEN

### Örnek 1: sanal bir ağ yapılandırma
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

Komut hedef IP yapılandırması 'nı güncelleştirir ve sanal ağı dokunun.
Başvuruda bulunan herhangi bir dokun yapılandırması varsa, tüm kaynak trafiği, yeni hedef IP yapılandırması sonrası güncellemesine başlatılmaz.
