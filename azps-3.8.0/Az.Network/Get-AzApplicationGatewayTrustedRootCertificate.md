# Get-AzApplicationGatewayTrustedRootCertificate

## SYNOPSIS
Uygulama ağ geçidinden belirli bir adla güvenilen kök sertifikayı alır.

## INDEKI

```
Get-AzApplicationGatewayTrustedRootCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzApplicationGatewayTrustedRootCertificate** cmdlet 'ı uygulama ağ geçidinden belirli bir adla güvenilen kök sertifikayı alır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedRootCert = Get-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\rootCA.cer"
```

İlk komut uygulama ağ geçidini alır ve $gw değişkeninde depolar.
İkinci komut, uygulama ağ geçidinden belirtilen adla güvenilen kök sertifikayı alır.
