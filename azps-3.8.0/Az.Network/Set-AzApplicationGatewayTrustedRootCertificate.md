# Set-AzApplicationGatewayTrustedRootCertificate

## SYNOPSIS
Uygulama ağ geçidi 'nin güvenilen kök sertifikasını güncelleştirir.

## INDEKI

```
Set-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Set-AzApplicationGatewayTrustedRootCertificate** cmdlet 'i, bir uygulama ağ geçidinin varolan güvenilen kök sertifikasını değiştirir.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\rootCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Yukarıdaki örnek senaryolar, kök sertifika alındığında varolan bir güvenilen kök sertifikanın nasıl güncelleştirileceğini gösterir.
İlk komut bir uygulama ağ geçidi alır ve $gw değişkeninde depolar.
İkinci komut, varolan güvenilen kök sertifikayı yeni bir kök sertifikayla değiştirir.
Üçüncü komut, Azure 'daki uygulama ağ geçidini güncelleştirir.
