# Get-AzNetworkUsage

## SYNOPSIS
Abonelik için ağ kullanımlarını listeler

## INDEKI

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
Get-AzNetworkUsage cmdlet 'i ağ kaynaklarının sınırlarını ve geçerli kullanımını alır.
