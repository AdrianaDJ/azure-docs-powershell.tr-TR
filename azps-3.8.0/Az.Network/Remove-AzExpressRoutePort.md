# Remove-AzExpressRoutePort

## SYNOPSIS
ExpressRoutePort öğesini kaldırır.

## INDEKI

### ResourceNameParameterSet (varsayılan)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobjectparameterset
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceıdparameterset
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzExpressRoutePort** cmdlet 'ı bir expressrouteport öğesini kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Aboneliğinizdeki $rg kaynak grubunda ExpressRoutePort kaynağını $PortName kaldırır.

### Örnek 2
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

InputObject içindeki ExpressRoutePort kaynağını kaldırır.

### Örnek 3
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

RESOURCEID $id ile ExpressRoutePort kaynağını kaldırır.
