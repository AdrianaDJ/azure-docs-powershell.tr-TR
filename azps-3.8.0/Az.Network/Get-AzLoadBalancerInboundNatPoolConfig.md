# Get-AzLoadBalancerInboundNatPoolConfig

## SYNOPSIS
Bir yük dengeleyiciden bir veya birden çok gelen NAT havuz yapılandırmasını alır.

## INDEKI

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azloadbalancerınboundnatpoolconfig** cmdlet 'i, bir yük dengeleyiciden bir veya birden çok gelen NAT havuz yapılandırması alır.

## ÖRNEKLERDEN

### 1: Get
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
