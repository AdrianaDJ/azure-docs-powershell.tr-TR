# New-AzP2sVpnGateway

## SYNOPSIS

## Tanım
**New-AzP2sVpnGateway** cmdlet 'i, site Istemcilerinden Azure virtualwan 'a noktadan site bağlanabilirliğinin üzerine gelmek Için virtualhub 'ın altında yeni bir P2SVpnGateway oluşturmanıza imkan verir.
