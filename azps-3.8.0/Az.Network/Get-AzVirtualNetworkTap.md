# Get-AzVirtualNetworkTap

## SYNOPSIS
Sanal ağ 'a dokunun

## INDEKI

### ListParameterSet (varsayılan)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Getbyresourceıdparameterset
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
**Get-AzVirtualNetworkTap** cmdlet 'ı bir Azure sanal ağına dokunun veya kaynak grubundaki Azure sanal ağ dokunuş listesine dokunun.

## ÖRNEKLERDEN

### Örnek 1: sanal ağ alma
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

Bu komut, "ResourceGroup1" içinde verilen "VirtualTap1" için VirtualNetwork dokunma başvurusu alır.

### Örnek 2: filtreleme kullanarak tüm sanal ağ dokunuş
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

Bu komut, "VirtualTap" ile başlayan tüm VirtualNetwork dokunma başvurularını alır.
