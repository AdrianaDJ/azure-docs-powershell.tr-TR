# New-AzNetworkWatcherNetworkConfigurationDiagnosticProfile

## SYNOPSIS
Yeni bir ağ yapılandırması tanılama profili nesnesi oluşturur. Bu nesne, belirtilen ölçütleri kullanarak Tanılama oturumu sırasında ağ yapılandırmasını kısıtlamak için kullanılır.

## INDEKI

```
New-AzNetworkWatcherNetworkConfigurationDiagnosticProfile -Direction <String> -Protocol <String>
 -Source <String> -Destination <String> -DestinationPort <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Tanım
New-AzNetworkWatcherNetworkConfigurationDiagnosticProfile cmdlet 'i yeni bir tanılama profili nesnesi oluşturur. Bu nesne, ağ yapılandırması Tanılama oturumu sırasında belirtilen ölçütleri kullanarak ağ yapılandırmasını kısıtlamak için kullanılır.
