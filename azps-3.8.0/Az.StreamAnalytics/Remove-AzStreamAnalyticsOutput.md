# Remove-AzStreamAnalyticsOutput

## SYNOPSIS
Bir Stream Analytics işinden çıktı siler.

## INDEKI

```
Remove-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzStreamAnalyticsOutput** cmdlet 'i Azure 'Daki bir Stream Analytics işinin çıktısını zaman uyumsuz olarak siler.

## ÖRNEKLERDEN

### Örnek 1: bir işten çıktı kaldırma
```
PS C:\>Remove-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Bu komut, iş StreamingJob 'daki çıkış çıkışını kaldırır.
