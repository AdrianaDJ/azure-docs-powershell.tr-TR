# Remove-AzWebAppAccessRestrictionRule

## SYNOPSIS
Bir Azure Web uygulamasından erişim kısıtlama kuralını kaldırır.

## INDEKI

```
Remove-AzWebAppAccessRestrictionRule [-ResourceGroupName] <String> [-WebAppName] <String> [-Name <String>]
 [-Action <String>] [-TargetScmSite] [-SlotName <String>] [-IpAddress <String>] [-SubnetName <String>]
 [-VirtualNetworkName <String>] [-SubnetId <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzWebAppAccessRestrictionRule** cmdlet 'ı bir Azure Web uygulamasından erişim kısıtlama kuralını kaldırır

## ÖRNEKLERDEN

### Örnek 1: Web uygulaması erişim kısıtlaması kuralını kaldırma
```
PS C:\>Remove-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" -Name IpRule
```

Bu komut, varsayılan-Web-WestUS adlı kaynak grubuna ait olan ContosoSite adlı Azure Web App 'ten ıprule erişim kısıtlama kuralını kaldırır.
