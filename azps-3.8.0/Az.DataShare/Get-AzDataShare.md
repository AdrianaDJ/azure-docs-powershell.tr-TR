# Get-AzDataShare

## SYNOPSIS
Veri paylaşımları hakkında bilgi edinin.

## INDEKI

### ByFieldsParameterSet (varsayılan)
```
Get-AzDataShare -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Byresourceıdparameterset
```
Get-AzDataShare -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-AzDataShare** cmdlet 'i, bir Azure veri paylaşımında veri paylaşımı hakkında bilgi alır.
Veri paylaşımının adını belirtirseniz, bu cmdlet bu veri paylaşımı hakkında bilgi alır.
Bir ad belirtmezseniz, bu cmdlet bir Azure veri paylaşımı hesabındaki tüm veri paylaşımları hakkında bilgi alır.

## ÖRNEKLERDEN

### Örnek 1: belirli bir veri paylaşımı alma
```
PS C:\>Get-AzDataShare -ResourceGroupName "ADS" -AccountName "WikiAdsAccount" -Name "AdsShare"
Name                : AdsShare
Id                  : /subscriptions/f3ead1ff-d0ab-4cf4-9a5a-86f1661d4685/resourceGroups/ADS/providers/Microsoft.DataShare/accounts/WikiAdsAccount/shares/AdsShare
Type                : Microsoft.DataShare/shares
CreatedAt           : 6/11/2019 12:00:00 AM
CreatedBy           : Contoso ADS
ShareKind           : CopyBased
Description         : Example of description  
ProvisioningState   : Succeeded
Terms               : This should not be shared
```

Bu komut, Azure veri paylaşımı hesabındaki veri paylaşımı AdsShare ile ilgili bilgileri görüntüler.
