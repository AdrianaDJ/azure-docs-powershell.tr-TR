# Get-AzDataShareInvitation

## SYNOPSIS
Veri paylaşımının bilgi davetini alır.

## INDEKI

### ByFieldsParameterSet (varsayılan)
```
Get-AzDataShareInvitation -ResourceGroupName <String> -AccountName <String> -ShareName <String>
 [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Byresourceıdparameterset
```
Get-AzDataShareInvitation -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Tanım
**Get-Azdatasharedavetcmdlet** 'i veri paylaşımında bulunan davetlerde bilgi alır. Davetin adını belirtirseniz, bu cmdlet davet hakkında bilgi alır. Bir ad belirtmezseniz, bu cmdlet paylaşımdaki tüm davetler hakkında bilgi alır.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Get-AzDataShareInvitation -ResourceGroupName "ADS" -AccountName "WikiAds" -ShareName "AdsShare" -Name "AdsInvitation"

InvitationId     : 167e06ff-567f-4bc7-be0c-645a6de710f3
InvitationStatus : Pending
Sender           : adsprovider@microsoft.com
SentAt           : 7/8/2019 10:47:10 PM
TargetEmail      : adstest@microsoft.com
TargetObjectId   :
TargetTenantId   :
Id               : /subscriptions/1834da9b-787a-44f6-ae81-60707ab8c957/resourceGroups/ADS/providers/Microsoft.DataShare/accounts/WikiAds/shares/AdsShare/invitations/AdsInvitation
Name             : AdsInvitation
Type             : Microsoft.DataShare/Invitations
```

Bu komut, veri paylaşımı AdsShare ile ilgili davet etme hakkında bilgi sağlar.
