# Az. DevSpaces modülü
## Tanım
Azure DevSpaces denetleyicisiyle etkileşimde bulunma komutları.

## Az. DevSpaces cmdlet 'Leri
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Azure DevSpaces denetleyicisini alın veya listeleyin.

### [Yeni-AzDevSpacesController](New-AzDevSpacesController.md)
Yeni bir Azure DevSpaces denetleyicisi oluşturun.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
DevSpaces denetleyicisini silme.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Etiket eklemek için DevSpaces denetleyicisini güncelleyin.
