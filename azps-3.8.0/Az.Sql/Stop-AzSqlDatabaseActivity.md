# Stop-AzSqlDatabaseActivity

## SYNOPSIS
Veritabanındaki zaman uyumsuz güncelleştirmeler işlemini iptal eder.

## INDEKI

```
Stop-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Tanım
**Stop-AzSqlDatabaseActivity** cmdlet 'i veritabanındaki zaman uyumsuz güncelleştirmeler işlemini iptal eder.

## ÖRNEKLERDEN

### Örnek 1: veritabanındaki zaman uyumsuz güncelleştirmeler işlemini Iptal etme
