# Remove-AzKustoDatabase

## SYNOPSIS
Var olan bir kusto veritabanını siler.

## INDEKI

### Binaik Vseçresourcegroup (varsayılan)
```
Remove-AzKustoDatabase [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Byresourceıd
```
Remove-AzKustoDatabase [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKustoDatabase [-InputObject] <PSKustoDatabase> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
Var olan bir kusto veritabanını siler.

## ÖRNEKLERDEN

### Örnek 1-varolan bir kusto veritabanını adla silme

```
PS C:\> Remove-AzKustoDatabase -ResourceGroupName testrg -ClusterName mykustocluster -Name mykustodatabase
```

Yukarıdaki komut, "mykustodatabase" adlı kusto veritabanını "testrg" kaynak grubunda bulunan "mykustocluster" kümesinde siler.

### Örnek 2-yöneltme ile mevcut bir kusto veritabanını silme

```
PS C:\> Get-AzKustoDatabase -ResourceGroupName testrg -ClusterName mykustocluster -Name mykustodatabase | Remove-AzKustoDatabase
```

Yukarıdaki komut, cmdlet 'i kullanarak "mykustodatabase" kaynak grubunda bulunan "mykustocluster" kümesindeki "" adlı kusto veritabanını alır `Get-AzKustoDatabase` ve sonra `Remove-AzKustoDatabase` da bunu silmek için sonucu yöneltin.
