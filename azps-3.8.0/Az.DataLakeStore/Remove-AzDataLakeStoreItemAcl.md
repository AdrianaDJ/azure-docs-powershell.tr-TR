# Remove-AzDataLakeStoreItemAcl

## SYNOPSIS
Data Lake Store 'da bir dosya veya klasörün ACL 'sini temizler.

## INDEKI

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Remove-AzDataLakeStoreItemAcl** cmdlet 'ı, Data Lake Store 'daki bir dosyanın veya klasörün erişim denetim LISTESINI (ACL) temizler.

## ÖRNEKLERDEN

### Örnek 1: ACL 'yi klasörden kaldırma
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Bu komut, ContosoADL hesabına ait kök dizinin ACL 'sini kaldırır.
