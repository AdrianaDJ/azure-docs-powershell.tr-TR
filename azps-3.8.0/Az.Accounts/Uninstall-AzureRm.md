# Uninstall-AzureRm

## SYNOPSIS
Bir makineden tüm AzureRm modüllerini kaldırır.

## INDEKI

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Tanım
Bir makineden tüm AzureRm modüllerini kaldırır.

## ÖRNEKLERDEN

### Örnek 1
```
PS C:\> Uninstall-AzureRm
```

Bu komut çalıştırıldığında, cmdlet 'in çalıştırıldığı PowerShell sürümü için makineden tüm AzureRm modülleri kaldırılır.
