# Save-AzDeploymentScriptLog

## SYNOPSIS
Dağıtım betiği yürütme günlüğünü diske kaydeder.

## INDEKI

### SaveDeploymentScriptLogByName (varsayılan)
```
Save-AzDeploymentScriptLog [-ResourceGroupName] <String> [-Name] <String> [-OutputPath] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Savedeploymentscriptlogbyresourceıd
```
Save-AzDeploymentScriptLog [-DeploymentScriptResourceId] <String> [-OutputPath] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SaveDeploymentScriptLogByInputObject
```
Save-AzDeploymentScriptLog [-DeploymentScriptInputObject] <PsDeploymentScript> [-OutputPath] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Tanım
**Save-AzDeploymentScriptLog** , dağıtım betiği yürütme günlüğünü diske kaydeder.

## ÖRNEKLERDEN

### Örnek 1
```powershell
PS C:\> Save-AzDeploymentScriptLog -Name MyDeploymentScript -ResourceGroupName DS-TestRg -OutputPath C:\Workspace
```

Bir dağıtım komut dosyasının günlüğünü verilen ad ve kaynak grubuyla kaydeder.
