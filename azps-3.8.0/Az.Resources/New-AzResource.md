## Tanım
**New-AzResource** cmdlet 'i, bir kaynak grubundaki Web sitesi, Azure SQL veritabanı sunucusu veya Azure SQL veritabanı gibi bir Azure kaynağı oluşturur.

## ÖRNEKLERDEN

### Örnek 1: kaynak oluşturma
```
PS> New-AzResource -Location "West US" -Properties @{test="test"} -ResourceName TestSite06 -ResourceType microsoft.web/sites -ResourceGroupName ResourceGroup11 -Force
```

Bu komut, ResourceGroup11 'da Web sitesi olan bir kaynak oluşturur.

### Örnek 2: splatın kullanarak kaynak oluşturma
```
PS> $prop = @{
    Location          = "West US"
    Properties        = @{test = "test"}
    ResourceName      = "TestSite06"
    ResourceType      = "microsoft.web/sites"
    ResourceGroupName = "ResourceGroup11"
    Force             = $true
}

PS> New-AzResource @prop
```

Bu komut, ResourceGroup11 'da Web sitesi olan bir kaynak oluşturur.
