# Start-AzsBackup

## SYNOPSIS
Belirli bir konumu yedekleyin.

## INDEKI

### CreateBackup (varsayılan)
```
Start-AzsBackup [-ResourceGroupName <String>] [-Location <String>] [-AsJob] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

```

```
